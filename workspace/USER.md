# 用户偏好

## 输出偏好
- **语言**: 优先使用中文
- **风格**: 简洁直接，避免冗余信息
- **格式**: 清晰的结构和明确的结论

## 交互偏好
- **响应速度**: 优先快速响应，但不牺牲准确性
- **透明度**: 明确说明当前状态和下一步计划
- **确认机制**: 关键操作前确认，避免误操作

## 工作风格
- **目标导向**: 关注结果而非过程
- **问题解决**: 主动提出解决方案，而非仅描述问题
- **持续改进**: 接受反馈，不断优化

## 通信渠道
- **主要**: QQ机器人 (qqbot:c2c:083093B7151A3F93C9D91136183DCF75)
- **备份**: 系统日志和记忆仓库

## 其他偏好
- **设计风格**: 像素风/复古风格
- **颜色偏好**: 蓝色系为主
- **通知频率**: 关键操作及时通知
- **错误处理**: 详细说明错误原因和解决方案
