# 第12章 概率统计

## 模块1：统计

### 1.1 抽样方法

**核心方法：**

- **简单随机抽样**：从总体$N$个单位中逐个抽取$n$个，每个个体被抽到的概率相等
  - 方法：抽签法、随机数法
  - 特点：逐个抽取、不放回、等概率

- **系统抽样（等距抽样）**：将总体分成$n$组，每组$k = \dfrac{N}{n}$个，在第1组随机抽1个，以后每隔$k$个抽1个
  - 步骤：编号$\to$计算间隔$k$$\to$在$[1,k]$中随机取$r$$\to$抽取$r, r+k, r+2k, \ldots$

- **分层抽样**：将总体按特征分成若干层，按比例从各层中抽取
  - 各层抽取数：$n_i = n \cdot \dfrac{N_i}{N}$（按比例分配）
  - 特点：层内差异小，层间差异大

**解题思路：**

1. 根据总体特征选择抽样方法
2. 计算各部分抽样数
3. 注意系统抽样中分组与编号的关系

**典型题型：**

- 选择合适的抽样方法
- 计算各层抽取的样本数
- 系统抽样的编号与间隔

**二级结论：**

- 三种抽样方法都是等概率抽样
- 分层抽样中，每层内部可用简单随机抽样或系统抽样
- 系统抽样要求$N$能被$n$整除（否则需先剔除$N \mod n$个个体）

### 1.2 频率分布直方图

**核心方法：**

- **频率分布直方图**：
  - 纵轴表示$\dfrac{\text{频率}}{\text{组距}}$，横轴表示数据
  - 每个小矩形面积$=$该组频率
  - 所有矩形面积之和$= 1$
  - 频率$=$矩形面积$=$矩形高度$\times$组距

- **关键量计算**：
  - 频数$=$频率$\times$样本容量$n$
  - 频率$=\dfrac{\text{频数}}{n}$
  - 累积频率$=$小于某值的频率之和

- **由直方图估计总体特征**：
  - 众数：最高矩形的中点横坐标
  - 中位数：面积累积到0.5处的横坐标
  - 平均数：$\bar{x} = \sum x_i \cdot f_i$（$x_i$为各组中值，$f_i$为各组频率）

**解题思路：**

1. 读懂直方图（注意纵轴是频率/组距）
2. 利用面积$=$频率的关系求解
3. 估计总体特征值

**典型题型：**

- 补全频率分布直方图
- 由直方图求频率、频数
- 由直方图估计众数、中位数、平均数

**二级结论：**

- 频率分布直方图中，矩形高度$=$频率/组距，面积$=$频率
- 众数$\neq$最高矩形的中心值，而是该矩形的横坐标中点
- 中位数：左右两侧面积各为0.5的分界点（通常需要线性插值）

### 1.3 茎叶图

**核心方法：**

- **茎叶图**：将数据按位数分为"茎"和"叶"
  - 茎：高位数字
  - 叶：低位数字
  - 优点：保留了原始数据信息

- **由茎叶图计算统计量**：
  - 中位数：排序后取中间值
  - 众数：出现最多的数
  - 平均数：所有数据之和/数据个数

**解题思路：**

1. 读取茎叶图中的所有数据
2. 排序后计算统计量
3. 比较两组数据的分布特征

**典型题型：**

- 由茎叶图求统计量
- 比较两组数据的集中趋势和离散程度

**二级结论：**

- 茎叶图中数据已按大小排列，便于找中位数
- 茎叶图的优势：保留原始数据，便于计算精确统计量
- 缺点：数据量大时不方便

### 1.4 方差与标准差

**核心方法：**

- **方差**：
  $$s^2 = \dfrac{1}{n}\sum_{i=1}^{n}(x_i - \bar{x})^2 = \dfrac{1}{n}\sum_{i=1}^{n}x_i^2 - \bar{x}^2$$
- **标准差**：$s = \sqrt{s^2}$
- **简化公式**：$s^2 = \dfrac{1}{n}\sum x_i^2 - \bar{x}^2$（常用！）

**解题思路：**

1. 计算平均数$\bar{x}$
2. 用方差公式计算
3. 比较不同数据组的方差（方差越小越稳定）

**典型题型：**

- 求方差和标准差
- 比较两组数据的稳定性

**二级结论：**

- 方差越大，数据越分散；方差越小，数据越集中
- 若$x_i' = ax_i + b$，则$s'^2 = a^2 s^2$（方差只与$a$有关，与$b$无关）
- 标准差的单位与原数据相同
- $\sum(x_i - \bar{x})^2 = \sum x_i^2 - n\bar{x}^2$

### 1.5 回归分析

**核心方法：**

- **回归直线方程**：$\hat{y} = \hat{b}x + \hat{a}$
  - $\hat{b} = \dfrac{\sum_{i=1}^n x_i y_i - n\bar{x}\bar{y}}{\sum_{i=1}^n x_i^2 - n\bar{x}^2}$
  - $\hat{a} = \bar{y} - \hat{b}\bar{x}$
  - 回归直线过定点$(\bar{x}, \bar{y})$

- **相关系数**：
  $$r = \dfrac{\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^n(x_i-\bar{x})^2 \cdot \sum_{i=1}^n(y_i-\bar{y})^2}}$$
  - $|r| \leq 1$，$|r|$越接近1，线性相关性越强
  - $r > 0$正相关，$r < 0$负相关

- **决定系数**$R^2$：$R^2$越接近1，回归效果越好

**解题思路：**

1. 计算$\bar{x}, \bar{y}$
2. 计算$\hat{b}$和$\hat{a}$
3. 写出回归方程
4. 利用回归方程进行预测

**典型题型：**

- 求回归直线方程
- 利用回归方程进行预测
- 求相关系数并判断相关性

**二级结论：**

- 回归直线必过点$(\bar{x}, \bar{y})$
- $\hat{b}$的符号与$r$的符号相同
- $r \in [-1, 1]$，$|r| > 0.75$一般认为有较强线性关系
- 样本点越靠近回归直线，$R^2$越接近1

### 1.6 独立性检验

**核心方法：**

- **独立性检验（$\chi^2$检验）**：
  $$\chi^2 = \dfrac{n(ad - bc)^2}{(a+b)(c+d)(a+c)(b+d)}$$
  其中$n = a + b + c + d$为总频数

- **步骤**：
  1. 提出假设$H_0$：两个分类变量独立
  2. 计算观测值$\chi^2$
  3. 与临界值比较
  4. 做出判断

- **临界值**：
  - $\chi^2 > 3.841$：有95%把握认为两者有关
  - $\chi^2 > 6.635$：有99%把握认为两者有关

**解题思路：**

1. 列出$2 \times 2$列联表
2. 计算$\chi^2$值
3. 与临界值比较并判断

**典型题型：**

- 计算卡方值并判断是否相关
- 补全列联表

**二级结论：**

- $\chi^2$越大，两个变量相关的把握越大
- 独立性检验的结论是"有/没有足够的把握"而非"一定相关/独立"
- 常用临界值：$P(\chi^2 \geq 3.841) = 0.05$，$P(\chi^2 \geq 6.635) = 0.01$

## 模块2：随机事件的概率与独立性

### 2.1 古典概型

**核心方法：**

- **古典概型条件**：
  1. 有限性：样本空间中基本事件数有限
  2. 等可能性：每个基本事件发生的概率相等

- **概率公式**：
  $$P(A) = \dfrac{\text{事件}A\text{包含的基本事件数}}{\text{基本事件总数}} = \dfrac{m}{n}$$

**解题思路：**

1. 确定基本事件总数$n$（用排列组合计算）
2. 确定事件$A$包含的基本事件数$m$
3. $P(A) = \dfrac{m}{n}$

**典型题型：**

- 掷骰子/摸球/抽牌的概率
- 有放回/无放回抽样概率

**二级结论：**

- 古典概型核心：数数（排列组合）
- 注意"有放回"vs"无放回"的区别
- 有放回时基本事件总数$= n^k$（$n$为可选数，$k$为抽取次数）
- 无放回时基本事件总数$= A_n^k$或$C_n^k$（看是否有序）

### 2.2 几何概型

**核心方法：**

- **几何概型条件**：
  1. 无限性：样本空间中基本事件数无限
  2. 等可能性：每个基本事件发生的概率相等

- **概率公式**：
  $$P(A) = \dfrac{\text{事件}A\text{对应的区域度量}}{\text{样本空间对应的区域度量}}$$
  - 度量可以是长度、面积、体积等

**解题思路：**

1. 确定样本空间对应的几何区域
2. 确定事件$A$对应的几何区域
3. 计算度量之比

**典型题型：**

- 线段上的随机点概率
- 区域内的随机点概率
- 时间上的随机概率

**二级结论：**

- 几何概型核心：求"长度/面积/体积"之比
- 常见类型：$|x-a| + |y-b| < c$型区域、两线段交点型
- 注意区分：随机取点在线段上vs在正方形内

### 2.3 互斥与对立

**核心方法：**

- **互斥事件**：$A \cap B = \emptyset$（不可能同时发生）
  - 加法公式：$P(A \cup B) = P(A) + P(B)$

- **对立事件**：$A \cup B = \Omega$且$A \cap B = \emptyset$
  - $P(\bar{A}) = 1 - P(A)$
  - 对立一定互斥，互斥不一定对立

- **一般加法公式**：$P(A \cup B) = P(A) + P(B) - P(A \cap B)$

**解题思路：**

1. 判断事件关系（互斥、对立、独立）
2. 选择合适的概率公式
3. 利用对立事件简化计算（"正难则反"）

**典型题型：**

- 利用互斥加法公式求概率
- 利用对立事件求概率

**二级结论：**

- "正难则反"策略：求$P(A)$困难时，求$P(\bar{A})$再用$1 - P(\bar{A})$
- $n$个互斥事件的概率：$P(A_1 \cup A_2 \cup \cdots \cup A_n) = P(A_1) + P(A_2) + \cdots + P(A_n)$
- 至少/至多问题常用对立事件

### 2.4 条件概率

**核心方法：**

- **条件概率**：
  $$P(B|A) = \dfrac{P(AB)}{P(A)}$$
  （在$A$发生的条件下$B$发生的概率）

- **乘法公式**：
  $$P(AB) = P(A) \cdot P(B|A) = P(B) \cdot P(A|B)$$

**解题思路：**

1. 缩小样本空间法：在$A$发生后的新样本空间中计算$B$的概率
2. 公式法：用$P(B|A) = \dfrac{P(AB)}{P(A)}$计算

**典型题型：**

- 求条件概率
- 利用乘法公式求积事件的概率

**二级结论：**

- $P(B|A) \neq P(A|B)$，一般不相等
- $P(B|A) + P(\bar{B}|A) = 1$
- 条件概率的本质：缩小样本空间

### 2.5 全概率公式

**核心方法：**

- **全概率公式**：若$A_1, A_2, \ldots, A_n$是样本空间的一个划分（两两互斥且并集为$\Omega$），则
  $$P(B) = \sum_{i=1}^n P(A_i) \cdot P(B|A_i)$$

- **贝叶斯公式**：
  $$P(A_k|B) = \dfrac{P(A_k) \cdot P(B|A_k)}{\sum_{i=1}^n P(A_i) \cdot P(B|A_i)}$$

**解题思路：**

1. 找到样本空间的一个划分（如：第一次取红球/白球）
2. 对每种情况求条件概率
3. 用全概率公式求和

**典型题型：**

- 两步及以上抽样问题
- "已知结果求原因"型（贝叶斯）

**二级结论：**

- 全概率公式适用场景：分阶段/分情况讨论的概率问题
- 贝叶斯公式适用场景：已知结果反推原因的概率
- "先验概率"$P(A_k) \to$ "后验概率"$P(A_k|B)$

## 模块3：离散型随机变量及其分布

### 3.1 条件概率与全概率

**核心方法（进阶应用）：**

- **全概率公式在随机变量中的应用**：
  - 将复杂事件分解为若干互斥情形
  - 每种情形下分别求条件概率
  - 加权求和

- **树状图法**：用树状图列出所有可能情况，计算各路径概率

**解题思路：**

1. 画出树状图（或列举所有情况）
2. 每条路径的概率$=$各步概率之积
3. 所求事件的概率$=$相关路径概率之和

**典型题型：**

- 多步抽样后求概率
- 有放回/无放回逐次抽样

**二级结论：**

- 树状图适用于分步试验
- 无放回抽样：每步概率取决于之前的结果
- 有放回抽样：各步独立

### 3.2 分布列与数字特征

**核心方法：**

- **离散型随机变量的分布列**：

  | $X$ | $x_1$ | $x_2$ | $\cdots$ | $x_n$ |
  |-----|-------|-------|---------|-------|
  | $P$ | $p_1$ | $p_2$ | $\cdots$ | $p_n$ |

  - 性质：$p_i \geq 0$，$\sum p_i = 1$

- **数学期望（均值）**：
  $$E(X) = \sum_{i=1}^n x_i p_i$$

- **方差**：
  $$D(X) = E(X - E(X))^2 = \sum_{i=1}^n (x_i - E(X))^2 p_i = E(X^2) - (E(X))^2$$

- **标准差**：$\sigma(X) = \sqrt{D(X)}$

**解题思路：**

1. 确定随机变量的所有可能取值
2. 计算每个取值的概率
3. 列出分布列
4. 计算期望和方差

**典型题型：**

- 求随机变量的分布列
- 求期望和方差
- 利用期望和方差做决策

**二级结论：**

- $E(aX + b) = aE(X) + b$
- $D(aX + b) = a^2 D(X)$
- $E(X+Y) = E(X) + E(Y)$（无论$X,Y$是否独立）
- $D(X) = E(X^2) - [E(X)]^2$（常用计算公式）
- 若$X, Y$独立，则$D(X+Y) = D(X) + D(Y)$
- 若$X, Y$独立，则$E(XY) = E(X) \cdot E(Y)$

### 3.3 二项分布与超几何分布

**核心方法：**

- **二项分布**$X \sim B(n, p)$：
  - 背景：$n$次独立重复试验，每次成功概率$p$
  - $P(X = k) = C_n^k p^k (1-p)^{n-k}$（$k = 0, 1, \ldots, n$）
  - $E(X) = np$，$D(X) = np(1-p)$

- **超几何分布**$X \sim H(N, M, n)$：
  - 背景：$N$件产品中有$M$件次品，取$n$件，次品数$X$
  - $P(X = k) = \dfrac{C_M^k C_{N-M}^{n-k}}{C_N^n}$（$\max(0, n-N+M) \leq k \leq \min(M, n)$）
  - $E(X) = \dfrac{nM}{N}$

- **二项分布vs超几何分布**：
  - 二项分布：有放回抽样（各次独立）
  - 超几何分布：无放回抽样（各次不独立）
  - 当$N$很大时，超几何分布近似为二项分布

**解题思路：**

1. 判断是"有放回"还是"无放回"
2. 有放回$\to$二项分布；无放回$\to$超几何分布
3. 套用公式计算

**典型题型：**

- 求二项分布的概率
- 求超几何分布的概率
- 求期望和方差

**二级结论：**

- 二项分布$B(n, p)$的期望$E = np$，方差$D = npq$（$q = 1-p$）
- 超几何分布的期望$E = \dfrac{nM}{N}$（直觉：按比例分配）
- 判断方法：看是否"放回"$\to$放回用二项，不放回用超几何
- 当$N \to \infty$，$\dfrac{M}{N} \to p$时，超几何$\to$二项

### 3.4 正态分布

**核心方法：**

- **正态分布**$X \sim N(\mu, \sigma^2)$：
  - $\mu$：均值（决定对称轴位置）
  - $\sigma^2$：方差（决定"胖瘦"程度）
  - $\sigma$越大，曲线越"矮胖"；$\sigma$越小，曲线越"高瘦"

- **正态曲线性质**：
  - 关于$x = \mu$对称
  - 在$x = \mu$处取得最大值
  - 曲线与$x$轴之间的面积为1

- **$3\sigma$原则**：
  - $P(\mu - \sigma \leq X \leq \mu + \sigma) \approx 0.6827$
  - $P(\mu - 2\sigma \leq X \leq \mu + 2\sigma) \approx 0.9545$
  - $P(\mu - 3\sigma \leq X \leq \mu + 3\sigma) \approx 0.9973$

- **标准化**：$Z = \dfrac{X - \mu}{\sigma} \sim N(0, 1)$

**解题思路：**

1. 确定$\mu$和$\sigma$
2. 利用对称性和$3\sigma$原则求概率
3. 标准化后查表（高考中一般给出需要的数据）

**典型题型：**

- 利用正态分布求概率
- 正态分布的参数估计
- 正态分布与$3\sigma$原则应用

**二级结论：**

- $P(X \leq \mu) = P(X \geq \mu) = 0.5$（对称性）
- $P(X \leq a) = P(X \geq 2\mu - a)$（对称性推广）
- $P(\mu - k\sigma \leq X \leq \mu + k\sigma)$只与$k$有关，与$\mu, \sigma$无关
- 正态分布的概率计算常利用对称性转化为标准正态分布

## 模块4：成对数据的统计分析

### 4.1 回归方程

**核心方法：**

- **一元线性回归模型**：$y = bx + a + e$（$e$为随机误差）
- **最小二乘法估计**：
  - $\hat{b} = \dfrac{\sum_{i=1}^n(x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^n(x_i - \bar{x})^2}$
  - $\hat{a} = \bar{y} - \hat{b}\bar{x}$
- **回归直线**：$\hat{y} = \hat{b}x + \hat{a}$，必过$(\bar{x}, \bar{y})$
- **残差**：$\hat{e}_i = y_i - \hat{y}_i$
- **残差平方和**：$\sum \hat{e}_i^2 = \sum(y_i - \hat{y}_i)^2$，越小拟合越好

**解题思路：**

1. 计算$\bar{x}, \bar{y}$和必要求和
2. 计算$\hat{b}, \hat{a}$
3. 写出回归方程
4. 利用回归方程进行预测（注意预测范围）

**典型题型：**

- 求线性回归方程
- 利用回归方程预测
- 比较不同回归模型的优劣

**二级结论：**

- 回归直线必过样本中心点$(\bar{x}, \bar{y})$
- $\hat{b}$与$r$同号：$r > 0$则$\hat{b} > 0$（正相关），$r < 0$则$\hat{b} < 0$（负相关）
- 预测时只能对自变量在样本范围内的值进行内插预测
- 非线性回归：通过变换（如取对数）转化为线性回归

### 4.2 独立性检验

**核心方法：**

- **$2 \times 2$列联表**：

  |  | $B$ | $\bar{B}$ | 合计 |
  |--|-----|----------|------|
  | $A$ | $a$ | $b$ | $a+b$ |
  | $\bar{A}$ | $c$ | $d$ | $c+d$ |
  | 合计 | $a+c$ | $b+d$ | $n$ |

- **卡方统计量**：
  $$\chi^2 = \dfrac{n(ad - bc)^2}{(a+b)(c+d)(a+c)(b+d)}$$

- **判断规则**：
  - $\chi^2 \geq 3.841$：有95%的把握认为$A$与$B$有关
  - $\chi^2 \geq 6.635$：有99%的把握认为$A$与$B$有关
  - $\chi^2 < 3.841$：没有充分证据认为$A$与$B$有关

**解题思路：**

1. 列出列联表
2. 计算$\chi^2$值
3. 与临界值比较
4. 给出统计结论（注意表述规范）

**典型题型：**

- 独立性检验的完整过程
- 由部分数据计算$\chi^2$

**二级结论：**

- 独立性检验的结论是概率性的，不是确定性的
- 正确表述："有$95\%$的把握认为$A$与$B$有关"
- 错误表述："$A$与$B$有关"或"$A$与$B$一定有关"
- $\chi^2$值越大，拒绝原假设的把握越大

### 4.3 相关系数

**核心方法：**

- **样本相关系数**：
  $$r = \dfrac{\sum_{i=1}^n(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^n(x_i - \bar{x})^2 \cdot \sum_{i=1}^n(y_i - \bar{y})^2}}$$

- **性质**：
  - $|r| \leq 1$
  - $r > 0$：正相关；$r < 0$：负相关
  - $|r|$越接近1：线性相关程度越强
  - $|r|$越接近0：线性相关程度越弱

- **与回归系数的关系**：
  - $\hat{b} = r \cdot \dfrac{s_y}{s_x}$（$s_x, s_y$为标准差）
  - $r^2 = R^2$（决定系数，衡量回归拟合优度）

**解题思路：**

1. 计算$r$值
2. 根据$|r|$大小判断线性相关程度
3. $r$的符号判断正相关/负相关

**典型题型：**

- 计算相关系数
- 判断线性相关程度
- 相关系数与回归分析综合

**二级结论：**

- $|r| > 0.75$：认为$x$与$y$有较强的线性相关关系
- $0.3 < |r| \leq 0.75$：中等相关
- $|r| \leq 0.3$：弱相关或基本不相关
- $r$只衡量"线性"相关程度，非线性关系$r$可能很小但关系很强
- $r^2 = R^2$（决定系数）：回归平方和/总平方和
