# 第10章 解析几何

## 模块1：直线与方程

### 1.1 直线方程五种形式

**核心方法：**

- **点斜式**：$y - y_0 = k(x - x_0)$（不能表示垂直于$x$轴的直线）
- **斜截式**：$y = kx + b$（不能表示垂直于$x$轴的直线，$b$为$y$轴截距）
- **两点式**：$\dfrac{y - y_1}{y_2 - y_1} = \dfrac{x - x_1}{x_2 - x_1}$（不能表示垂直于坐标轴的直线）
- **截距式**：$\dfrac{x}{a} + \dfrac{y}{b} = 1$（不能表示过原点和垂直于坐标轴的直线）
- **一般式**：$Ax + By + C = 0$（$A, B$不同时为0，可表示所有直线）

**解题思路：**

1. 根据已知条件选择合适的直线方程形式
2. 注意各种形式的限制条件
3. 需要讨论斜率不存在的情况

**典型题型：**

- 求过已知点/已知斜率的直线方程
- 根据截距条件求直线方程
- 直线过定点问题

**二级结论：**

- 过$(x_0, y_0)$的直线可设为$y - y_0 = k(x - x_0)$或$x = x_0$
- 过两点的直线斜率 $k = \dfrac{y_2 - y_1}{x_2 - x_1}$
- 直线的倾斜角$\alpha$与斜率$k$的关系：$k = \tan\alpha$（$\alpha \in [0, \pi)$）

### 1.2 点到直线距离

**核心方法：**

- **点到直线距离公式**：点$P(x_0, y_0)$到直线$Ax + By + C = 0$的距离
  $$d = \dfrac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$$

- **两平行直线距离**：$Ax + By + C_1 = 0$与$Ax + By + C_2 = 0$的距离
  $$d = \dfrac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$$

**解题思路：**

1. 直接套用距离公式
2. 注意直线方程需化为一般式
3. 平行线距离公式要求两直线$A, B$系数相同

**典型题型：**

- 求点到直线距离
- 求两平行线间距离
- 利用距离条件求直线方程

**二级结论：**

- 点到直线距离的几何意义：垂线段长度
- 两平行线间距离等于一条线上任一点到另一条线的距离

### 1.3 两直线位置关系

**核心方法：**

- **平行**：$l_1: y = k_1x + b_1$，$l_2: y = k_2x + b_2$，平行$\iff k_1 = k_2$且$b_1 \neq b_2$
  - 一般式：$\dfrac{A_1}{A_2} = \dfrac{B_1}{B_2} \neq \dfrac{C_1}{C_2}$
- **垂直**：$k_1 \cdot k_2 = -1$（斜率都存在时）
  - 一般式：$A_1A_2 + B_1B_2 = 0$
- **相交**：$k_1 \neq k_2$（斜率都存在时）
- **重合**：$k_1 = k_2$且$b_1 = b_2$
  - 一般式：$\dfrac{A_1}{A_2} = \dfrac{B_1}{B_2} = \dfrac{C_1}{C_2}$

**解题思路：**

1. 用斜率判断（注意斜率不存在的情况）
2. 用一般式系数之比判断
3. 求交点：联立方程组求解

**典型题型：**

- 判断两直线位置关系
- 求过一点与已知直线平行/垂直的直线
- 对称问题（点关于直线对称、直线关于直线对称）

**二级结论：**

- 点$(x_0, y_0)$关于直线$Ax+By+C=0$的对称点：利用中点在直线上+连线与直线垂直两个条件
- 点关于$x$轴对称$((x,y) \to (x,-y))$，关于$y$轴对称$((x,y) \to (-x,y))$，关于原点对称$((x,y) \to (-x,-y))$，关于$y=x$对称$((x,y) \to (y,x))$
- 直线关于直线对称：取两个特殊点分别求对称点，再确定直线

## 模块2：圆与方程

### 2.1 圆的方程

**核心方法：**

- **标准方程**：$(x-a)^2 + (y-b)^2 = r^2$，圆心$(a, b)$，半径$r$
- **一般方程**：$x^2 + y^2 + Dx + Ey + F = 0$
  - 圆心$\left(-\dfrac{D}{2}, -\dfrac{E}{2}\right)$，半径$r = \dfrac{\sqrt{D^2 + E^2 - 4F}}{2}$
  - 表示圆的条件：$D^2 + E^2 - 4F > 0$
- **参数方程**：$x = a + r\cos\theta$，$y = b + r\sin\theta$

**解题思路：**

1. 待定系数法求圆的方程（三个条件确定一个圆）
2. 已知圆心和半径用标准方程
3. 已知一般条件用一般方程

**典型题型：**

- 求圆的方程
- 圆过定点问题
- 圆的方程中参数的范围

**二级结论：**

- 圆心在直线$Ax + By + C = 0$上的圆可设为$(x-a)^2 + (y-b)^2 = r^2$，其中$Aa + Bb + C = 0$
- 过原点的圆的一般方程中$F = 0$

### 2.2 直线与圆的位置关系

**核心方法：**

- **代数法**：联立方程，判别式$\Delta > 0$相交，$\Delta = 0$相切，$\Delta < 0$相离
- **几何法**（推荐）：圆心$(a,b)$到直线$Ax+By+C=0$的距离$d$
  - $d < r$：相交；$d = r$：相切；$d > r$：相离
- **弦长公式**：$|AB| = 2\sqrt{r^2 - d^2}$（$d$为圆心到直线的距离）
- **切线长公式**：过圆外点$P$到圆的切线长 $l = \sqrt{|PC|^2 - r^2}$

**解题思路：**

1. 判断位置关系：用几何法（比较$d$与$r$）
2. 求弦长：先求$d$，再用弦长公式
3. 求切线方程：利用$d = r$条件

**典型题型：**

- 判断直线与圆的位置关系
- 求弦长、求切线方程
- 直线与圆相交的弦中点问题

**二级结论：**

- 过圆上一点$(x_0, y_0)$的切线方程：$(x_0-a)(x-a) + (y_0-b)(y-b) = r^2$
- 弦中点与圆心连线垂直于弦
- 直线被圆截得的弦的中点轨迹：以圆心和定点为直径的圆（在已知圆内的部分）

### 2.3 圆与圆的位置关系

**核心方法：**

- 设两圆半径分别为$r_1, r_2$（$r_1 \geq r_2$），圆心距为$d$
- 相离：$d > r_1 + r_2$；外切：$d = r_1 + r_2$；相交：$|r_1 - r_2| < d < r_1 + r_2$
- 内切：$d = |r_1 - r_2|$；内含：$d < |r_1 - r_2|$

**二级结论：**

- 两圆公共弦所在直线：两圆方程相减
- 两圆相外切时，切点在两圆心连线上

### 2.4 切线问题

**核心方法：**

- **过圆上一点的切线**：直接利用切线方程公式
- **过圆外一点的切线**：利用$d = r$（几何法）或$\Delta = 0$（代数法）
- **切线长**：$l = \sqrt{|PC|^2 - r^2}$

**二级结论：**

- 过圆外一点有两条切线
- 从圆外一点到圆的两切线长相等
- 两切点连线与圆心和该点连线垂直

## 模块3：椭圆与方程

### 3.1 定义、标准方程与几何性质

**核心方法：**

- **定义**：与两个定点$F_1, F_2$的距离之和等于常数$2a$（$2a > |F_1F_2|$）的点的轨迹
- **标准方程**：
  - 焦点在$x$轴：$\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1$（$a > b > 0$）
  - 焦点在$y$轴：$\dfrac{y^2}{a^2} + \dfrac{x^2}{b^2} = 1$（$a > b > 0$）
- **基本关系**：$a^2 = b^2 + c^2$，$c^2 = a^2 - b^2$
- **几何性质**：焦距$2c$，离心率$e = \dfrac{c}{a}$（$0 < e < 1$），长轴$2a$，短轴$2b$

**二级结论：**

- 椭圆上点到焦点距离范围：$[a-c, a+c]$
- 椭圆上点到原点距离范围：$[b, a]$
- $e$越接近0越接近圆，越接近1越扁

### 3.2 焦点三角形

**核心方法：**

- **面积公式**：$S_{\triangle PF_1F_2} = b^2 \tan\dfrac{\theta}{2}$（$\theta = \angle F_1PF_2$）

**二级结论：**

- 当$P$在短轴端点时，$\theta$最大，面积最大
- 焦半径：$|PF_1| = a + ex_0$，$|PF_2| = a - ex_0$（$P(x_0, y_0)$在椭圆上）
- 过焦点最短弦长$\dfrac{2b^2}{a}$

### 3.3 设点设线方法

**核心方法：**

- **设点法**：设椭圆上点为$(a\cos\theta, b\sin\theta)$或$(x_0, y_0)$满足方程
- **设线法**：设斜率为$k$：$y = kx + m$；斜率不存在时设$x = t$
- **韦达定理法**：联立直线与椭圆方程，利用韦达定理

**二级结论：**

- 中点弦问题：点差法（两点在椭圆上，相减得斜率关系）
- 点差法结论：$k_{OP} \cdot k_{AB} = -\dfrac{b^2}{a^2}$（$O$为中心，$AB$为弦，$P$为中点）

### 3.4 高考常用二级结论

- **焦半径公式**：$|PF_1| = a + ex_0$，$|PF_2| = a - ex_0$
- **点差法**：弦中点$(x_0, y_0)$，斜率$k$，则$k = -\dfrac{b^2 x_0}{a^2 y_0}$
- **切线方程**：点$(x_0,y_0)$处切线：$\dfrac{x_0 x}{a^2}+\dfrac{y_0 y}{b^2}=1$
- **切点弦方程**：圆外点$P(x_0,y_0)$引切线，切点弦：$\dfrac{x_0 x}{a^2}+\dfrac{y_0 y}{b^2}=1$
- **斜率积定值**：椭圆上两点$A, B$关于原点对称，任取椭圆上一点$P$，则$k_{PA} \cdot k_{PB} = -\dfrac{b^2}{a^2}$

## 模块4：双曲线与方程

### 4.1 定义、标准方程与几何性质

**核心方法：**

- **定义**：与两个定点$F_1, F_2$的距离之差的绝对值等于常数$2a$（$0 < 2a < |F_1F_2|$）的点的轨迹
- **标准方程**：焦点在$x$轴：$\dfrac{x^2}{a^2} - \dfrac{y^2}{b^2} = 1$；焦点在$y$轴：$\dfrac{y^2}{a^2} - \dfrac{x^2}{b^2} = 1$
- **基本关系**：$c^2 = a^2 + b^2$
- **渐近线**：$y = \pm\dfrac{b}{a}x$（焦点在$x$轴）
- **离心率**：$e = \dfrac{c}{a}$（$e > 1$）

**二级结论：**

- 双曲线上点到焦点距离范围：$[c-a, +\infty)$
- 等轴双曲线：$a = b$，渐近线$y = \pm x$，$e = \sqrt{2}$
- 共轭双曲线：$\dfrac{x^2}{a^2}-\dfrac{y^2}{b^2}=1$与$\dfrac{y^2}{b^2}-\dfrac{x^2}{a^2}=1$互为共轭双曲线
- 共渐近线双曲线系：$\dfrac{x^2}{a^2}-\dfrac{y^2}{b^2}=\lambda$（$\lambda \neq 0$）

### 4.2 焦点三角形

**核心方法：**

- **面积公式**：$S_{\triangle PF_1F_2} = b^2\cot\dfrac{\theta}{2}$（$\theta = \angle F_1PF_2$）

**二级结论：**

- 焦半径公式：设$P$在右支上，$|PF_1| = a + ex_0$，$|PF_2| = -a + ex_0$

### 4.3 渐近线问题

**核心方法：**

- 渐近线与离心率关系：$\dfrac{b}{a} = \sqrt{e^2 - 1}$，$e = \sqrt{1 + \left(\dfrac{b}{a}\right)^2}$
- 已知渐近线求双曲线：设$\dfrac{x^2}{a^2}-\dfrac{y^2}{b^2}=\lambda$（$\lambda \neq 0$）

**二级结论：**

- 双曲线焦点到渐近线的距离等于$b$
- 等轴双曲线离心率$e = \sqrt{2}$
- 共渐近线的双曲线方程：$\dfrac{x^2}{a^2}-\dfrac{y^2}{b^2} = t$（$t \neq 0$）

### 4.4 高考常用二级结论

- **焦半径公式**：$|PF_1| = |a + ex_0|$，$|PF_2| = |a - ex_0|$
- **切线方程**：双曲线上点$(x_0, y_0)$处切线：$\dfrac{x_0 x}{a^2}-\dfrac{y_0 y}{b^2}=1$
- **点差法**：弦中点$(x_0, y_0)$，斜率$k$，则$k = \dfrac{b^2 x_0}{a^2 y_0}$
- **双曲线焦点到渐近线距离**$= b$
- **双曲线通径**长$= \dfrac{2b^2}{a}$

## 模块5：抛物线与方程

### 5.1 定义、标准方程与几何性质

**核心方法：**

- **定义**：与定点$F$和定直线$l$（准线）距离相等的点的轨迹
- **标准方程**（四种形式，以$y^2 = 2px$为例，$p > 0$）：
  - 焦点$F\left(\dfrac{p}{2}, 0\right)$，准线$x = -\dfrac{p}{2}$
- **焦半径公式**：$|PF| = x_0 + \dfrac{p}{2}$
- **焦点弦长**：$|AB| = x_A + x_B + p$

**二级结论：**

- 抛物线焦点弦的中点到准线的距离$= \dfrac{|AB|}{2}$
- 通径长$= 2p$

### 5.2 定义与几何性质综合

**核心方法：**

- **焦准距法**：利用$|PF| = d$（$d$为$P$到准线距离）转化距离
- **最值问题**：利用焦半径公式和三角形不等式

**二级结论：**

- 焦点弦中，通径最短$= 2p$
- $|PF| \geq \dfrac{p}{2}$（焦半径最小值）

### 5.3 综合运算

**核心方法：**

- **联立方程法**：直线与抛物线联立，利用韦达定理
- **设点法**：设抛物线上点为$\left(\dfrac{y_0^2}{2p}, y_0\right)$

**典型题型：**

- 直线与抛物线交点问题
- 弦长、面积问题

### 5.4 高考常用二级结论

（以$y^2 = 2px$为例）

- **焦半径**：$|PF| = x_0 + \dfrac{p}{2}$
- **焦点弦公式**：$|AB| = \dfrac{2p}{\sin^2\alpha}$（$\alpha$为弦与$x$轴的夹角）
- **焦点弦性质**：
  - $x_A \cdot x_B = \dfrac{p^2}{4}$
  - $y_A \cdot y_B = -p^2$
  - 以$AB$为直径的圆与准线相切
  - $\dfrac{1}{|AF|} + \dfrac{1}{|BF|} = \dfrac{2}{p}$
- **切线方程**：点$(x_0, y_0)$处切线：$y_0 y = p(x + x_0)$
- **点差法**：弦中点$(x_0, y_0)$，斜率$k$，则$k = \dfrac{p}{y_0}$

## 模块6：解析几何大题

### 6.1 联立韦达定理

**核心方法：**

- **联立步骤**：设直线$\to$代入曲线$\to$整理一元二次方程$\to$韦达定理$\to$注意$\Delta > 0$
- **椭圆联立**：$y = kx + m$代入$\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}=1$
  $$(a^2k^2+b^2)x^2 + 2a^2kmx + a^2m^2 - a^2b^2 = 0$$
- **双曲线联立**：$y = kx + m$代入$\dfrac{x^2}{a^2}-\dfrac{y^2}{b^2}=1$
  $$(a^2k^2-b^2)x^2 + 2a^2kmx + a^2m^2 + a^2b^2 = 0$$
- **抛物线联立**：$y = kx + m$代入$y^2 = 2px$
  $$k^2x^2 + (2km - 2p)x + m^2 = 0$$

### 6.2 设而不求

**核心方法：**

- **核心思想**：设交点坐标$(x_1, y_1)$，$(x_2, y_2)$但不具体求出，利用韦达定理整体代换
- **常用技巧**：
  - $x_1^2 + x_2^2 = (x_1+x_2)^2 - 2x_1x_2$
  - $\dfrac{y_1 - y_2}{x_1 - x_2} = k$（$k$为直线斜率）
  - $y_1 + y_2 = k(x_1+x_2) + 2m$

**二级结论：**

- 弦长公式：$|AB| = \sqrt{1+k^2}\sqrt{(x_1+x_2)^2 - 4x_1x_2}$
- 三角形面积（以原点为第三顶点）：$S = \dfrac{1}{2}|x_1y_2 - x_2y_1|$

### 6.3 点乘双根法

**核心方法：**

- **适用场景**：求$\dfrac{1}{x_1 - a} + \dfrac{1}{x_2 - a}$类型表达式
- **方法**：设$f(t) = a(t-x_1)(t-x_2)$，则
  - $\dfrac{1}{t-x_1} + \dfrac{1}{t-x_2} = \dfrac{f'(t)}{f(t)}$，代入$t = a$
  - $(t-x_1)(t-x_2) = \dfrac{f(t)}{a}$，代入$t = a$

**二级结论：**

- 点乘双根法本质：$f(t) = a(t-x_1)(t-x_2)$，$\dfrac{f'(t)}{f(t)} = \dfrac{1}{t-x_1}+\dfrac{1}{t-x_2}$
- 适用于"某点分式"型条件的快速化简

### 6.4 弦长与面积

**核心方法：**

- **弦长公式**：$|AB| = \sqrt{1+k^2}|x_1-x_2| = \sqrt{1+k^2}\sqrt{(x_1+x_2)^2 - 4x_1x_2}$
- **三角形面积**（以原点为第三顶点）：$S = \dfrac{1}{2}|x_1y_2 - x_2y_1|$
- **一般三角形面积**：$S = \dfrac{1}{2}|(x_1-x_0)(y_2-y_0) - (x_2-x_0)(y_1-y_0)|$（第三顶点$(x_0,y_0)$）

**解题思路：**

1. 联立求韦达定理
2. 用弦长公式/面积公式整体代换
3. 化简求解

**典型题型：**

- 求弦长
- 求三角形面积
- 面积最值问题

### 6.5 定点定值

**核心方法：**

- **定值问题**：证明某个量与参数无关（消参后为常数）
- **定点问题**：证明直线/曲线恒过某定点
- **一般策略**：
  1. 设参数（如斜率$k$）
  2. 用韦达定理表示目标量
  3. 化简消参，证明与$k$无关
  4. 定点问题：将直线方程整理为$f(k) \cdot x + g(k) \cdot y + h = 0$，令$f(k)=0$且$g(k)=0$解出定点

**解题思路：**

1. **特殊化探路**：取特殊位置（如$k=0, k=1$等）猜测定值/定点
2. **一般化证明**：设一般参数，用韦达定理严格推导
3. 验证特殊情况是否成立

**典型题型：**

- 证明直线恒过定点
- 证明某量为定值
- 求定点的坐标或定值的大小

**二级结论：**

- 定点问题常用"参数分离法"：将直线方程写为$y - y_0 = k(x - x_0)$的形式
- 定值问题常用"特殊值法"验证：取$k=0$或垂直情况先求出定值，再一般证明

### 6.6 存在性问题

**核心方法：**

- **存在性问题**：假设存在，推导条件，判断是否满足所有约束
- **一般策略**：
  1. 假设存在，设参数
  2. 利用条件建立方程
  3. 求解参数值
  4. 验证是否满足$\Delta > 0$等约束条件

**解题思路：**

1. 假设存在，设所求量
2. 建立方程求解
3. 检验解的合理性（判别式、范围等）
4. 若有解则存在，无解则不存在

**典型题型：**

- 是否存在直线使某条件成立
- 是否存在点使某条件成立
- 是否存在参数使结论成立

**二级结论：**

- 存在性问题的答案：存在时给出具体值，不存在时说明理由
- 常见陷阱：求出的参数需要检验$\Delta > 0$、斜率存在等条件
- 解析几何大题的终极策略：**设线$\to$联立$\to$韦达$\to$整体代换$\to$消参化简**
