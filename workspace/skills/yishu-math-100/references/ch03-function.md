# 第3章 函数与导数

## 模块1：函数概念与性质

### 3.1 定义域值域求法

**核心方法**：
- 定义域：使函数有意义的一切实数x的集合
- 值域：函数值的集合
- 求定义域的方法：
  - 分母不为0
  - 偶次根号下非负
  - 对数真数大于0
  - 零指数幂底数不为0
- 求值域的方法：
  - 配方法
  - 换元法
  - 单调性法
  - 数形结合法

**解题技巧**：
- 先求定义域，再求值域
- 定义域是函数的"活动范围"
- 值域是函数的"输出范围"
- 注意定义域的限制对值域的影响

**二级结论**：
1. 定义域是函数存在的必要条件
2. 值域是函数的"输出集合"
3. 若函数在定义域上单调递增，则值域为[f_min, f_max]

**典型题型**：
1. 函数定义域的求法
2. 函数值域的求法
3. 定义域与值域的综合

### 3.2 单调性判断

**核心方法**：
- 单调递增：f(x₁) < f(x₂) ⇔ x₁ < x₂
- 单调递减：f(x₁) > f(x₂) ⇔ x₁ < x₂
- 定义法：任取x₁,x₂，比较f(x₁),f(x₂)
- 导数法：f'(x) > 0 ⇒ 单调递增，f'(x) < 0 ⇒ 单调递减
- 复合函数单调性：同增异减

**解题技巧**：
- 定义法：取x₁<x₂，证明f(x₁)<f(x₂)
- 导数法：求导，确定导数的符号
- 复合函数：外层函数和内层函数同增异减
- 数形结合：画图辅助判断

**二级结论**：
1. 若f(x)在区间I上单调递增，则f'(x) ≥ 0（等号仅在有限点成立）
2. 若f(x)在区间I上单调递减，则f'(x) ≤ 0（等号仅在有限点成立）
3. 复合函数f(g(x))的单调性与内层函数g(x)、外层函数f(u)的单调性同增异减

**典型题型**：
1. 函数单调性的定义法判断
2. 导数法判断单调性
3. 复合函数单调性
4. 含参函数单调性

### 3.3 奇偶性判断

**核心方法**：
- 奇函数：f(-x) = -f(x)
- 偶函数：f(-x) = f(x)
- 奇偶性判断步骤：
  1. 判断定义域是否关于原点对称
  2. 验证f(-x)与f(x)的关系
- 奇函数图像关于原点对称，偶函数图像关于y轴对称

**解题技巧**：
- 先看定义域是否关于原点对称
- 利用奇偶性的定义验证
- 奇函数在x=0处有定义时，f(0)=0
- 偶函数在x=0处有定义时，f(0)≥0

**二级结论**：
1. 奇函数与偶函数的定义域必须关于原点对称
2. 奇函数的图像关于原点对称，偶函数的图像关于y轴对称
3. 若f(x)既是奇函数又是偶函数，则f(x)恒为0
4. 奇函数在x=0处有定义时，f(0)=0

**典型题型**：
1. 奇偶性的判断
2. 奇偶函数的性质
3. 奇偶函数的综合应用

### 3.4 周期性

**核心方法**：
- 周期函数：f(x+T) = f(x)（T为非零常数）
- 周期：最小的正周期
- 周期函数的性质：
  - 若f(x)是周期为T的函数，则f(x+nT)=f(x)（n∈Z）
  - 若f(x)是周期为T的函数，则f'(x)也是周期为T的函数

**解题技巧**：
- 利用周期性求函数值
- 利用周期性求函数图像
- 利用周期性简化计算
- 注意最小正周期

**二级结论**：
1. 周期函数的周期具有倍数关系：若T是周期，则nT（n∈Z）也是周期
2. 周期函数的导数也是周期函数，周期不变
3. 若f(x)是周期为T的函数，则f(x)在[0,T)上的性质可以推广到整个定义域

**典型题型**：
1. 周期性的判断
2. 周期函数的性质
3. 周期函数的应用

### 3.5 对称性

**核心方法**：
- 中心对称：f(a+x) + f(a-x) = 2b，对称中心为(a,b)
- 轴对称：f(a+x) = f(a-x)，对称轴为x=a
- 对称性的判断：
  - 中心对称：取x₁, x₂，使得x₁+x₂=2a，比较f(x₁)+f(x₂)
  - 轴对称：取x₁, x₂，使得x₁-x₂=a，比较f(x₁),f(x₂)

**解题技巧**：
- 利用对称性求函数值
- 利用对称性画图
- 利用对称性简化计算
- 数形结合

**二级结论**：
1. 中心对称：f(a+x) + f(a-x) = 2b ⇔ 对称中心为(a,b)
2. 轴对称：f(a+x) = f(a-x) ⇔ 对称轴为x=a
3. 若f(x)是中心对称函数，则g(x)=f(x)+b是奇函数
4. 若f(x)是轴对称函数，则g(x)=f(x+a)是偶函数

**典型题型**：
1. 对称性的判断
2. 对称函数的性质
3. 对称函数的应用

## 模块2：函数三类基本题型

### 3.1 函数零点

**核心方法**：
- 零点：方程f(x)=0的根
- 零点存在定理：若f(x)在[a,b]上连续，且f(a)·f(b) < 0，则f(x)在(a,b)内有零点
- 零点个数：根据函数图像与x轴的交点个数判断

**解题技巧**：
- 利用零点存在定理证明零点存在
- 数形结合：画图辅助判断
- 利用单调性判断零点唯一性
- 分类讨论：含参问题

**二级结论**：
1. 若f(x)在[a,b]上单调且f(a)·f(b) < 0，则f(x)在(a,b)内有且仅有一个零点
2. 若f(x)在[a,b]上连续，且f(a)·f(b) = 0，则f(x)在[a,b]内有零点
3. 若f(x)在[a,b]上连续，且f(a)·f(b) > 0，则f(x)在(a,b)内可能有零点，也可能没有

**典型题型**：
1. 零点存在定理的应用
2. 零点个数的判断
3. 含参零点问题

### 3.2 函数值比大小

**核心方法**：
- 比较大小的方法：
  - 作差法：f(x₁)-f(x₂)
  - 作商法：f(x₁)/f(x₂)
  - 单调性法：利用函数的单调性
  - 数形结合法：画图辅助

**解题技巧**：
- 根据函数类型选择合适的方法
- 作差法：判断符号
- 作商法：判断比值
- 单调性法：利用函数的单调性
- 数形结合：画图辅助

**二级结论**：
1. 若f(x)在区间I上单调递增，则x₁<x₂ ⇒ f(x₁)<f(x₂)
2. 若f(x)在区间I上单调递减，则x₁<x₂ ⇒ f(x₁)>f(x₂)
3. 若f(x)是奇函数，则f(x₁)与f(x₂)的大小关系与x₁,x₂的奇偶性有关

**典型题型**：
1. 函数值比大小（作差法）
2. 函数值比大小（作商法）
3. 函数值比大小（单调性法）
4. 函数值比大小（数形结合）

### 3.3 函数图像识别

**核心方法**：
- 图像识别：根据函数的图像特征判断函数类型
- 图像特征：
  - 开口方向：a>0向上，a<0向下
  - 顶点位置：根据顶点式确定
  - 对称轴：x=-b/(2a)
  - 与坐标轴交点：与x轴交点为方程的根，与y轴交点为(0,c)
  - 单调性：根据导数或函数性质判断

**解题技巧**：
- 画图辅助分析
- 数形结合
- 利用对称性
- 分类讨论

**二级结论**：
1. 抛物线与x轴交点为方程ax²+bx+c=0的根
2. 抛物线与y轴交点为(0,c)
3. 抛物线的对称轴为x=-b/(2a)
4. 抛物线的顶点为(-b/(2a), (4ac-b²)/(4a))

**典型题型**：
1. 函数图像的识别
2. 函数图像的画法
3. 函数图像的变换
4. 函数图像的综合

## 模块3：导数常规题型

### 3.1 求导法则

**核心方法**：
- 常数函数：C' = 0
- 幂函数：xⁿ' = nxⁿ⁻¹
- 指数函数：aˣ' = aˣln a
- 对数函数：ln x' = 1/x
- 三角函数：sin x' = cos x，cos x' = -sin x
- 复合函数求导：f(g(x))' = f'(g(x))·g'(x)（链式法则）

**解题技巧**：
- 熟记基本导数公式
- 熟练运用复合函数求导法则
- 注意求导顺序：从外到内
- 分段函数分段求导

**二级结论**：
1. (f(x)±g(x))' = f'(x) ± g'(x)
2. (f(x)·g(x))' = f'(x)·g(x) + f(x)·g'(x)
3. (f(x)/g(x))' = (f'(x)·g(x) - f(x)·g'(x))/g²(x)
4. (f(g(x)))' = f'(g(x))·g'(x)

**典型题型**：
1. 基本导数公式
2. 复合函数求导
3. 四则运算求导
4. 分段函数求导

---

### 3.2 切线方程

**核心方法**：
- 切线方程：y - f(x₀) = f'(x₀)(x - x₀)
- 求切线方程的步骤：
  1. 求f'(x)
  2. 求f'(x₀)
  3. 代入切线方程公式
- 斜率不存在的情况：x = x₀

**解题技巧**：
- 先求导数，再求斜率
- 注意切点坐标
- 斜率不存在时，切线方程为x = x₀
- 求曲线上的切线时，要注意切点在曲线上

**二级结论**：
1. 切线方程：y = f'(x₀)(x - x₀) + f(x₀)
2. 斜率k = f'(x₀)
3. 若切线斜率为0，则切线方程为y = f(x₀)
4. 若切线斜率不存在，则切线方程为x = x₀

**典型题型**：
1. 曲线切线方程的求法
2. 过曲线上一点的切线
3. 过曲线外一点的切线
4. 斜率存在/不存在的切线

### 3.3 单调区间

**核心方法**：
- 单调递增：f'(x) > 0
- 单调递减：f'(x) < 0
- 单调区间：f'(x) > 0的区间为单调递增区间，f'(x) < 0的区间为单调递减区间

**解题技巧**：
- 求f'(x)
- 解f'(x) > 0，求递增区间
- 解f'(x) < 0，求递减区间
- 注意定义域的限制
- 分类讨论：含参问题

**二级结论**：
1. f'(x) = 0的点可能是单调区间的分界点
2. f'(x) = 0的点不一定是单调区间的分界点（可能是极值点或拐点）
3. f'(x) > 0的区间是单调递增区间，f'(x) < 0的区间是单调递减区间

**典型题型**：
1. 求函数的单调区间
2. 含参函数的单调区间
3. 分段函数的单调区间
4. 复合函数的单调区间

### 3.4 极值最值

**核心方法**：
- 极值：f(x₀)是f(x)的极值 ⇔ 存在δ>0，当x∈(x₀-δ, x₀+δ)时，f(x) < f(x₀)（极大值）或f(x) > f(x₀)（极小值）
- 极值的必要条件：f'(x₀) = 0
- 极值的充分条件：
  - 第一充分条件：在x₀两侧，f'(x)由正变负，则x₀为极大值点；由负变正，则x₀为极小值点
  - 第二充分条件：f''(x₀) < 0 ⇒ x₀为极大值点；f''(x₀) > 0 ⇒ x₀为极小值点
- 最值：在闭区间上，最值在端点或极值点处取得

**解题技巧**：
- 求f'(x)
- 求f'(x) = 0的根
- 判断极值的类型（第一充分条件或第二充分条件）
- 求出极值
- 求出最值（端点值 + 极值）

**二级结论**：
1. 极值是局部性质，最值是全局性质
2. 极值点的导数一定为0，但导数为0的点不一定是极值点
3. 极值点一定是驻点，驻点不一定是极值点
4. 在闭区间上，最值在端点或极值点处取得

**典型题型**：
1. 求函数的极值
2. 求函数的最值
3. 含参函数的极值最值
4. 极值最值的综合应用

## 模块5：零点与不等式

### 5.1 零点存在定理

**核心方法**：
- 零点存在定理：若f(x)在[a,b]上连续，且f(a)·f(b) < 0，则f(x)在(a,b)内有零点
- 应用步骤：
  1. 确定函数在区间上连续
  2. 检查端点函数值的乘积是否小于0
  3. 利用定理证明零点存在

**解题技巧**：
- 验证f(x)在[a,b]上连续
- 计算f(a), f(b)
- 判断f(a)·f(b) < 0
- 得出结论：f(x)在(a,b)内有零点

**二级结论**：
1. 零点存在定理的条件：连续、端点函数值异号
2. 零点存在定理的结论：零点存在，但不确定个数
3. 若f(x)在[a,b]上单调且f(a)·f(b) < 0，则f(x)在(a,b)内有且仅有一个零点

**典型题型**：
1. 零点存在定理的应用
2. 零点个数的判断
3. 含参零点问题

---

### 5.2 零点个数判断

**核心方法**：
- 零点个数：根据函数图像与x轴的交点个数判断
- 判断方法：
  - 数形结合法：画图
  - 单调性法：利用函数的单调性
  - 分类讨论法：含参问题

**解题技巧**：
- 画图辅助分析
- 利用单调性判断零点唯一性
- 分类讨论参数
- 求出所有零点

**二级结论**：
1. 若f(x)在区间上单调，则零点最多一个
2. 若f(x)在区间上连续且单调，则零点最多一个
3. 若f(x)在区间上不单调，则零点可能有多个

**典型题型**：
1. 零点个数的判断
2. 含参零点个数
3. 零点个数的综合应用

---

### 5.3 含参不等式恒成立

**核心方法**：
- 含参不等式恒成立：f(x) > 0（或<0）对所有x∈D成立
- 求解方法：
  - 方法一：分离参数法
  - 方法二：最值法：f(x) > 0 ⇒ min f(x) > 0
  - 方法三：数形结合法

**解题技巧**：
- 方法一：分离参数法
  1. 将参数分离到不等式的一边
  2. 求另一边关于x的函数的最值
  3. 判断参数的范围
- 方法二：最值法
  1. 求f(x)的最值
  2. 令最值 > 0（或<0）
  3. 求参数的范围
- 方法三：数形结合法
  1. 画图辅助分析
  2. 利用图像判断

**二级结论**：
1. f(x) > 0恒成立 ⇒ min f(x) > 0
2. f(x) < 0恒成立 ⇒ max f(x) < 0
3. 若f(x)在区间上单调，则最值在端点处取得

**典型题型**：
1. 含参不等式恒成立（分离参数法）
2. 含参不等式恒成立（最值法）
3. 含参不等式恒成立（数形结合法）
4. 含参不等式恒成立的综合应用

---

### 5.4 能成立问题

**核心方法**：
- 能成立问题：f(x) > 0（或<0）至少有一个x∈D成立
- 求解方法：
  - 方法一：分离参数法
  - 方法二：最值法：f(x) > 0 ⇒ max f(x) > 0
  - 方法三：数形结合法

**解题技巧**：
- 方法一：分离参数法
  1. 将参数分离到不等式的一边
  2. 求另一边关于x的函数的最大值
  3. 判断参数的范围
- 方法二：最值法
  1. 求f(x)的最大值
  2. 令最大值 > 0（或<0）

- 方法三：数形结合法
  1. 画图辅助分析
  2. 利用图像判断

**二级结论**：
1. f(x) > 0能成立 ⇒ max f(x) > 0

3. 若f(x)在区间上单调，则最大值在端点处取得

## 模块6：复合函数综合

### 6.1 复合函数单调性

**核心方法**：
- 复合函数单调性：同增异减
- 判断方法：
  1. 分解复合函数：f(g(x))
  2. 判断内层函数g(x)的单调性
  3. 判断外层函数f(u)的单调性
  4. 综合判断复合函数的单调性

**解题技巧**：
- 分解复合函数
- 分别判断内层和外层函数的单调性
- 利用"同增异减"的规律
- 注意定义域的限制

**二级结论**：
1. 复合函数f(g(x))的单调性与内层函数g(x)、外层函数f(u)的单调性同增异减
2. 若g(x)单调递增，f(u)单调递增，则f(g(x))单调递增
3. 若g(x)单调递增，f(u)单调递减，则f(g(x))单调递减
4. 若g(x)单调递减，f(u)单调递增，则f(g(x))单调递减
5. 若g(x)单调递减，f(u)单调递减，则f(g(x))单调递增

**典型题型**：
1. 复合函数单调性的判断

### 7.3 差值法

**核心方法**：
- 差值法：构造差值f(x₁)-f(x₂)
- 应用：比较大小、证明不等式

**解题技巧**：
- 构造差值
- 判断差值的符号
- 得出结论

**二级结论**：

2. 若f(x₁)-f(x₂) < 0，则f(x₁) < f(x₂)

4. 差值法可以用于比较大小和证明不等式

**典型题型**：
