# 第9章 立体几何与空间向量

## 模块1：立体图形的结构探究

### 1.1 棱柱与棱锥

**核心方法：**

- **棱柱**：有两个面互相平行，其余各面都是四边形，且每相邻两个四边形的公共边都互相平行
  - 直棱柱：侧棱与底面垂直
  - 正棱柱：底面为正多边形的直棱柱
  - 体积公式：$V = S_{\text{底}} \cdot h$
  - 侧面积：直棱柱 $S_{\text{侧}} = C \cdot h$（$C$为底面周长）

- **棱锥**：有一个面是多边形，其余各面都是有一个公共顶点的三角形
  - 正棱锥：底面是正多边形，顶点在底面的射影是底面中心
  - 体积公式：$V = \dfrac{1}{3}S_{\text{底}} \cdot h$
  - 正棱锥侧面积：$S_{\text{侧}} = \dfrac{1}{2}C \cdot h'$（$h'$为斜高）

**解题思路：**

1. 识别几何体的类型（柱、锥、台）
2. 找出关键元素（高、斜高、侧棱、底面边长等）
3. 利用截面（轴截面、对角面等）将空间问题转化为平面问题
4. 建立未知量之间的关系式求解

**典型题型：**

- 求棱柱/棱锥的体积、侧面积、全面积
- 已知体积和部分条件反求其他量
- 正棱锥中高、斜高、侧棱、底面边长的关系

**二级结论：**

- 正$n$棱锥中，设底面边长为$a$，高为$h$，则斜高 $h' = \sqrt{h^2 + \left(\dfrac{a}{2\tan\frac{\pi}{n}}\right)^2}$，侧棱 $l = \sqrt{h^2 + \left(\dfrac{a}{2\sin\frac{\pi}{n}}\right)^2}$
- 棱台上底面面积$S_1$，下底面面积$S_2$，高$h$，则体积 $V = \dfrac{h}{3}(S_1 + S_2 + \sqrt{S_1 S_2})$

### 1.2 球的内切与外接

**核心方法：**

- **外接球**：球心到各顶点距离相等（等于半径$R$）
  - 长方体外接球：$R = \dfrac{\sqrt{a^2 + b^2 + c^2}}{2}$
  - 正方体外接球：$R = \dfrac{\sqrt{3}a}{2}$

- **内切球**：球心到各面距离相等（等于半径$r$）
  - 正方体内切球：$r = \dfrac{a}{2}$
  - 等体积法：$V = \dfrac{1}{3}S_{\text{全}} \cdot r$

- **构造法**：找几何体的外接球，关键在于确定球心位置

**解题思路：**

1. **长方体模型**：将几何体补成长方体，球心即长方体中心
2. **侧面模型**：确定底面外接圆半径$r$和球心到外接圆平面的距离$d$，则$R^2 = r^2 + d^2$
3. **等体积法**求内切球半径：$r = \dfrac{3V}{S_{\text{全}}}$

**典型题型：**

- 求正棱锥/正棱柱的外接球半径
- 求几何体的内切球半径
- 与外接球内切球相关的最值问题

**二级结论：**

- **正四面体外接球**：棱长为$a$，$R = \dfrac{\sqrt{6}}{4}a$
- **正四面体内切球**：棱长为$a$，$r = \dfrac{\sqrt{6}}{12}a$，$R:r = 3:1$
- **正三棱锥（底面边长$a$，高$h$）外接球**：$R = \dfrac{a^2}{4h} + \dfrac{h}{2}$（利用$R^2 = r^2 + (R-h)^2$，$r = \dfrac{a}{\sqrt{3}}$为底面外接圆半径）
- **正四棱锥（底面边长$a$，高$h$）外接球**：$R = \dfrac{a^2 + 2h^2}{4h}$（底面外接圆半径$r = \dfrac{\sqrt{2}}{2}a$）
- **三条侧棱两两垂直的棱锥外接球**：设三条侧棱长为$a, b, c$，则$R = \dfrac{\sqrt{a^2 + b^2 + c^2}}{2}$
- **正三棱柱外接球**：底面边长$a$，高$h$，$R = \sqrt{\dfrac{a^2}{3} + \dfrac{h^2}{4}}$

### 1.3 截面问题

**核心方法：**

- **截面作法**：利用平面与几何体各面的交线确定截面形状
- **公理应用**：
  - 公理1：若直线上两点在平面内，则整条直线在平面内
  - 公理2：若两个不重合平面有一个公共点，则它们有且只有一条过该点的公共直线
- **常用方法**：延长线法（延长截面与面的交线找到新的交点）

**解题思路：**

1. 找出截面与几何体各面的公共点
2. 由公共点确定截面多边形的顶点
3. 判断截面形状并计算

**典型题型：**

- 作出截面并判断截面形状
- 求截面面积
- 截面周长的最值问题

**二级结论：**

- 过棱柱侧棱的截面是平行四边形（或矩形）
- 正棱锥的平行于底面的截面与底面相似，面积之比等于相似比的平方
- 球的截面是圆，截面圆半径 $r = \sqrt{R^2 - d^2}$（$d$为球心到截面的距离）

### 1.4 翻折问题

**核心方法：**

- **翻折前后不变量**：翻折轴上的点、翻折轴上的线段长度不变
- **翻折前后变量**：翻折轴两侧的点之间距离改变，角度改变
- **关键策略**：在翻折前的平面图形中标注所有已知量，翻折后只改变不在折痕上的量

**解题思路：**

1. 画出翻折前的平面图形，标注所有已知量
2. 分析翻折后哪些量不变（折痕上的），哪些量改变
3. 在翻折后建立空间关系求解
4. 优先利用不变量简化计算

**典型题型：**

- 翻折后求线线角、线面角、二面角
- 翻折后求线段长度
- 判断翻折后点线面的位置关系

**二级结论：**

- 翻折前后，折痕上的点到其他点的距离不变
- 若翻折后两点重合，则这两点关于折痕对称
- 翻折角度为$\theta$（二面角），翻折前相距为$d$的两点（分别在折痕两侧且到折痕距离分别为$a, b$），翻折后距离为$\sqrt{a^2 + b^2 - 2ab\cos\theta}$

## 模块2：位置关系的判定

### 2.1 平行关系：线面平行与面面平行

**核心方法：**

- **线面平行判定定理**：平面外一条直线与此平面内的一条直线平行，则该直线与此平面平行（$\Leftarrow$ 线线平行）
- **线面平行性质定理**：一条直线与一个平面平行，则过这条直线的任一平面与此平面的交线与该直线平行（$\Rightarrow$ 线线平行）
- **面面平行判定定理**：一个平面内的两条相交直线与另一个平面平行，则这两个平面平行（$\Leftarrow$ 线面平行×2→面面平行）
- **面面平行性质定理**：若两个平面平行，则其中一个平面内的直线与另一个平面平行；若两个平行平面同时与第三个平面相交，则交线平行

**解题思路：**

1. **证线面平行**：找面内一条线与已知线平行（三角形中位线、平行四边形对边、面面平行的交线等）
2. **证面面平行**：找一个面内的两条相交直线分别平行于另一个平面
3. **用线面平行性质**：过已知线作辅助面，得到交线与已知线平行
4. **用面面平行性质**：两平行平面被第三个面所截，交线平行

**典型题型：**

- 证明线面平行（找平行线）
- 证明面面平行（找两条相交线分别平行）
- 利用平行关系求比值

**二级结论：**

- 若$\alpha \parallel \beta$，$l \subset \alpha$，则$l \parallel \beta$
- 三个平面两两平行，夹在中间的线段成比例
- 线面平行$\iff$直线与平面无公共点$\iff$平面内存在直线与该直线平行
- 过平面外一点有且仅有一个平面与已知平面平行

### 2.2 垂直关系：线面垂直与面面垂直

**核心方法：**

- **线面垂直判定定理**：一条直线与一个平面内的两条相交直线都垂直，则该直线与此平面垂直
- **线面垂直性质定理**：垂直于同一个平面的两条直线平行
- **面面垂直判定定理**：一个平面过另一个平面的垂线，则这两个平面垂直（$\Leftarrow$ 线面垂直→面面垂直）
- **面面垂直性质定理**：两个平面垂直，则一个平面内垂直于交线的直线垂直于另一个平面（$\Rightarrow$ 面面垂直→线面垂直）

**解题思路：**

1. **证线面垂直**：找面内两条相交线都与已知线垂直
2. **证面面垂直**：找一面内的一条线垂直于另一面（常利用面面垂直性质定理）
3. **利用面面垂直性质**：在垂直面内作交线的垂线，得到线面垂直
4. **三垂线定理**：在平面内的一条直线，如果和这个平面的一条斜线的射影垂直，则它也和这条斜线垂直

**典型题型：**

- 证明线面垂直
- 证明面面垂直
- 利用垂直关系求角

**二级结论：**

- **三垂线定理**：平面内一条直线与平面的一条斜线垂直$\iff$该直线与斜线的射影垂直
- **逆定理**：平面内一条直线与斜线垂直$\iff$该直线与斜线在该平面内的射影垂直
- 过一点有且仅有一条直线与已知平面垂直
- 过一点有且仅有一个平面与已知直线垂直
- 若$l \perp \alpha$，$l \perp \beta$，则$\alpha \parallel \beta$
- 若$\alpha \perp \beta$，$\beta \perp \gamma$，且$\alpha \cap \gamma = l$，则$l \perp \beta$

## 模块3：空间向量及其应用

### 3.1 空间向量基本运算

**核心方法：**

- **空间向量加法**：$\vec{a} + \vec{b} = (x_1+x_2, y_1+y_2, z_1+z_2)$
- **空间向量数乘**：$\lambda\vec{a} = (\lambda x_1, \lambda y_1, \lambda z_1)$
- **数量积**：$\vec{a} \cdot \vec{b} = x_1x_2 + y_1y_2 + z_1z_2 = |\vec{a}||\vec{b}|\cos\langle\vec{a},\vec{b}\rangle$
- **向量模**：$|\vec{a}| = \sqrt{x_1^2 + y_1^2 + z_1^2}$
- **共线条件**：$\vec{a} \parallel \vec{b} \iff \vec{a} = \lambda\vec{b}$（$\vec{b} \neq \vec{0}$）
- **共面条件**：$\vec{p} = x\vec{a} + y\vec{b}$（$\vec{a}, \vec{b}$不共线）
- **空间向量基本定理**：若$\vec{a}, \vec{b}, \vec{c}$不共面，则空间任一向量$\vec{p} = x\vec{a} + y\vec{b} + z\vec{c}$（唯一）

**解题思路：**

1. 建立空间直角坐标系
2. 写出各点坐标
3. 用坐标运算求解

**典型题型：**

- 空间向量的坐标运算
- 四点共面的判断与证明
- 用空间向量基本定理表示向量

**二级结论：**

- 四点$A, B, C, D$共面$\iff \overrightarrow{AD} = x\overrightarrow{AB} + y\overrightarrow{AC}$且$x + y = 1$
- $\vec{a} \perp \vec{b} \iff \vec{a} \cdot \vec{b} = 0 \iff x_1x_2 + y_1y_2 + z_1z_2 = 0$
- $\cos\langle\vec{a},\vec{b}\rangle = \dfrac{\vec{a} \cdot \vec{b}}{|\vec{a}||\vec{b}|} = \dfrac{x_1x_2 + y_1y_2 + z_1z_2}{\sqrt{x_1^2+y_1^2+z_1^2}\sqrt{x_2^2+y_2^2+z_2^2}}$

### 3.2 证平行垂直与求角

**核心方法：**

- **证线线平行**：$\vec{a} \parallel \vec{b} \iff \vec{a} = \lambda\vec{b}$
- **证线线垂直**：$\vec{a} \perp \vec{b} \iff \vec{a} \cdot \vec{b} = 0$
- **证线面平行**：$\vec{s} \cdot \vec{n} = 0$（$\vec{s}$为方向向量，$\vec{n}$为法向量）
- **证线面垂直**：$\vec{s} \parallel \vec{n} \iff \vec{s} = \lambda\vec{n}$
- **证面面平行**：$\vec{n_1} \parallel \vec{n_2}$
- **证面面垂直**：$\vec{n_1} \perp \vec{n_2} \iff \vec{n_1} \cdot \vec{n_2} = 0$

**求角公式：**

- **线线角**：$\cos\theta = \dfrac{|\vec{a} \cdot \vec{b}|}{|\vec{a}||\vec{b}|}$（注意取绝对值，$\theta \in [0, \frac{\pi}{2}]$）
- **线面角**：$\sin\theta = \dfrac{|\vec{s} \cdot \vec{n}|}{|\vec{s}||\vec{n}|}$（$\theta$为线与面所成角，$\theta \in [0, \frac{\pi}{2}]$）
- **二面角**：$\cos\theta = \dfrac{\vec{n_1} \cdot \vec{n_2}}{|\vec{n_1}||\vec{n_2}|}$（注意判断锐角或钝角）

**解题思路：**

1. 建系→写坐标→求方向向量/法向量→代入公式
2. **法向量求法**：设$\vec{n} = (x, y, z)$，由$\vec{n} \cdot \overrightarrow{AB} = 0$和$\vec{n} \cdot \overrightarrow{AC} = 0$解出
3. **二面角判断**：观察法向量方向是否指向二面角内部/外部，决定取$\cos\theta$还是$-\cos\theta$

**典型题型：**

- 用空间向量证明平行、垂直关系
- 求异面直线所成角
- 求线面角
- 求二面角

**二级结论：**

- 异面直线所成角的范围：$(0, \frac{\pi}{2}]$
- 线面角的范围：$[0, \frac{\pi}{2}]$
- 二面角的范围：$[0, \pi]$
- 线面角与线线角的关系：线面角$\leq$线线角（斜线与平面内任何直线的角中，线面角最小）
- 求二面角时，若两法向量都指向二面角内部或都指向外部，取$|\cos\theta|$；若一内一外，取$\cos\theta$

### 3.3 求距离

**核心方法：**

- **点到平面距离**：$d = \dfrac{|\overrightarrow{PQ} \cdot \vec{n}|}{|\vec{n}|}$（$P$为面外点，$Q$为面上任一点，$\vec{n}$为法向量）
- **点到直线距离**：$d = \dfrac{|\overrightarrow{PQ} \times \vec{s}|}{|\vec{s}|}$（$P$为线外点，$Q$为线上任一点，$\vec{s}$为方向向量）
  - 坐标公式：$d = \sqrt{|\overrightarrow{PQ}|^2 - \left(\dfrac{\overrightarrow{PQ} \cdot \vec{s}}{|\vec{s}|}\right)^2}$
- **异面直线距离**：$d = \dfrac{|\overrightarrow{PQ} \cdot \vec{n}|}{|\vec{n}|}$（$\vec{n} = \vec{s_1} \times \vec{s_2}$，$P, Q$分别在两条异面直线上）
- **线面距离**：转化为点到面的距离（线上任一点到面的距离）
- **面面距离**：转化为点到面的距离

**解题思路：**

1. 确定所求距离的类型
2. 找到合适的公式（点面距最常用）
3. 建系→求法向量/方向向量→代入公式

**典型题型：**

- 求点到平面的距离
- 求异面直线的距离
- 求线面距离、面面距离

**二级结论：**

- 点面距离公式本质：投影到法向量方向的绝对值
- 异面直线距离存在的条件：两条直线的方向向量不平行（不共线）
- 两平行平面间的距离等于一个平面上任一点到另一个平面的距离

## 模块4：综合提升篇

### 4.1 翻折与展开

**核心方法：**

- **翻折问题**：关注折痕上的不变量，翻折后转化为空间问题
- **展开问题**：将多面体的面展开成平面图形，求最短路径
- **常用策略**：翻折前在平面图上标注所有量，区分变与不变

**解题思路：**

1. 翻折：平面→空间，抓住不变量
2. 展开：空间→平面，将侧面展开求最短路径
3. 综合时注意翻折后角度的变化

**典型题型：**

- 翻折后求角和距离
- 多面体表面最短路径
- 翻折与垂直平行关系结合

**二级结论：**

- 正方体表面最短路径：展开相邻面，直线距离最短
- 圆锥侧面展开为扇形，弧长等于底面周长
- 圆台侧面展开为扇环

### 4.2 动点轨迹

**核心方法：**

- **建系法**：设动点坐标$(x, y, z)$，利用条件建立方程
- **几何法**：利用空间几何性质判断轨迹形状
- **定义法**：判断轨迹满足什么几何定义（球面、圆、直线等）

**解题思路：**

1. 设动点坐标，利用条件建立关系式
2. 消参得到轨迹方程
3. 判断轨迹的几何意义

**典型题型：**

- 动点在面上的轨迹
- 动点满足距离条件的轨迹
- 动点满足角度条件的轨迹

**二级结论：**

- 到定点距离等于定长的轨迹是球面（在面内是圆）
- 到定直线距离等于定长的轨迹是圆柱面
- 到两定点距离相等的轨迹是中垂面
- 在正方体/正棱锥中，动点轨迹常为圆或线段

### 4.3 最值问题

**核心方法：**

- **函数法**：建立目标函数，利用函数性质求最值
- **不等式法**：利用基本不等式求最值
- **几何法**：利用空间几何性质（对称、投影等）求最值
- **坐标法**：建系后用代数方法求最值

**解题思路：**

1. 明确目标量（距离、面积、体积、角度等）
2. 选择合适的方法建立关系
3. 求最值

**典型题型：**

- 动点距离最值
- 截面面积最值
- 体积最值
- 角度最值

**二级结论：**

- 球面上两点间最短距离是大圆劣弧
- 外接球半径最小化问题常利用对称性
- 截面面积最值常在特殊位置取得（过对称轴等）

### 4.4 外接球内切球二级结论汇总

**核心公式与二级结论：**

- **长方体**（$a \times b \times c$）：$R = \dfrac{\sqrt{a^2+b^2+c^2}}{2}$，$r = \dfrac{2V}{S_{\text{全}}} = \dfrac{2abc}{2(ab+bc+ca)} = \dfrac{abc}{ab+bc+ca}$

- **正方体**（棱长$a$）：$R = \dfrac{\sqrt{3}a}{2}$，$r = \dfrac{a}{2}$

- **正四面体**（棱长$a$）：$R = \dfrac{\sqrt{6}}{4}a$，$r = \dfrac{\sqrt{6}}{12}a$

- **正三棱锥**（底面边长$a$，高$h$）：
  - 底面外接圆半径 $r = \dfrac{a}{\sqrt{3}} = \dfrac{\sqrt{3}}{3}a$
  - $R^2 = r^2 + (R - h)^2$，解得 $R = \dfrac{r^2 + h^2}{2h} = \dfrac{a^2}{4h} + \dfrac{h}{2}$

- **正四棱锥**（底面边长$a$，高$h$）：
  - 底面外接圆半径 $r = \dfrac{\sqrt{2}}{2}a$
  - $R = \dfrac{2a^2 + 4h^2}{8h} = \dfrac{a^2 + 2h^2}{4h}$

- **三条侧棱两两垂直的棱锥**（侧棱长$a, b, c$）：
  - 补成长方体，$R = \dfrac{\sqrt{a^2+b^2+c^2}}{2}$

- **正三棱柱**（底面边长$a$，高$h$）：
  - $R = \sqrt{\left(\dfrac{a}{\sqrt{3}}\right)^2 + \left(\dfrac{h}{2}\right)^2} = \sqrt{\dfrac{a^2}{3} + \dfrac{h^2}{4}}$

- **等体积法求内切球**：$r = \dfrac{3V}{S_{\text{全}}}$

- **万能公式**：$R^2 = r^2 + d^2$，其中$r$为底面外接圆半径，$d$为球心到外接圆所在平面的距离

**高考常见构型：**

| 几何体 | 外接球半径$R$ | 内切球半径$r$ |
|--------|-------------|-------------|
| 长方体$a\times b\times c$ | $\frac{\sqrt{a^2+b^2+c^2}}{2}$ | $\frac{abc}{ab+bc+ca}$ |
| 正方体$a$ | $\frac{\sqrt{3}a}{2}$ | $\frac{a}{2}$ |
| 正四面体$a$ | $\frac{\sqrt{6}a}{4}$ | $\frac{\sqrt{6}a}{12}$ |
| 正三棱锥$(a, h)$ | $\frac{a^2+3h^2}{4h}$之半 | 等体积法 |
| 侧棱两两垂直$(a,b,c)$ | $\frac{\sqrt{a^2+b^2+c^2}}{2}$ | 等体积法 |
