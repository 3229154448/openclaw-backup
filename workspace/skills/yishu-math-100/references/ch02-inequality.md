# 第2章 一元二次函数、方程与不等式

## 模块1：不等式与二次函数

### 2.1 一元二次不等式解法

**核心方法**：
- 一元二次不等式：ax²+bx+c>0（或≥0,<0或≤0）
- 解法：先求对应方程ax²+bx+c=0的根，再根据开口方向和根的情况确定解集
- 三个"二次"关系：二次函数、一元二次方程、一元二次不等式

**解题技巧**：
- 先解方程ax²+bx+c=0，求根x₁,x₂
- 判断开口方向：a>0向上，a<0向下
- 解集规律：
  - a>0且x₁<x₂：>0的解集为(-∞,x₁)∪(x₂,+∞)，<0的解集为(x₁,x₂)
  - a<0且x₁<x₂：>0的解集为(x₁,x₂)，<0的解集为(-∞,x₁)∪(x₂,+∞)
- 注意边界：>0不包括根，≥0包括根

**二级结论**：
1. 一元二次不等式ax²+bx+c>0的解集与a的符号和判别式Δ有关
2. 当Δ>0时，不等式ax²+bx+c>0的解集为x<x₁或x>x₂（a>0）
3. 当Δ=0时，不等式ax²+bx+c>0的解集为x≠-b/(2a)（a>0）
4. 当Δ<0时，不等式ax²+bx+c>0的解集为R（a>0）

**典型题型**：
1. 一元二次不等式的解法
2. 含参数的一元二次不等式
3. 一元二次不等式的应用

### 2.2 二次函数图像与性质

**核心方法**：
- 二次函数：y=ax²+bx+c（a≠0）
- 顶点式：y=a(x-h)²+k，顶点(h,k)
- 一般式：y=ax²+bx+c，顶点(-b/(2a), (4ac-b²)/(4a))
- 对称轴：x=-b/(2a)或x=h
- 开口方向：a>0向上，a<0向下

**解题技巧**：
- 顶点式便于求最值和对称轴
- 一般式便于求函数值
- 利用对称性求函数值
- 数形结合：画图辅助分析

**二级结论**：
1. 二次函数的对称轴为x=-b/(2a)
2. 二次函数的最值：a>0时，最小值f(-b/(2a))=4ac-b²/(4a)；a<0时，最大值f(-b/(2a))=4ac-b²/(4a)
3. 抛物线与x轴交点为方程ax²+bx+c=0的根
4. 抛物线与y轴交点为(0,c)

**典型题型**：
1. 二次函数的图像与性质
2. 二次函数的最值问题
3. 二次函数的应用

### 2.3 三个"二次"的关系

**核心方法**：
- 二次函数：y=ax²+bx+c
- 一元二次方程：ax²+bx+c=0
- 一元二次不等式：ax²+bx+c>0（或<0）

**解题技巧**：
- 三个"二次"相互转化
- 方程的根对应函数的零点
- 不等式的解集对应函数值大于/小于0的区间
- Δ>0：抛物线与x轴有两个交点
- Δ=0：抛物线与x轴有一个交点（切点）
- Δ<0：抛物线与x轴无交点

**二级结论**：
1. 二次函数y=ax²+bx+c的零点就是方程ax²+bx+c=0的根
2. 一元二次不等式ax²+bx+c>0的解集就是y>0的x的范围
3. 二次函数y=ax²+bx+c在区间上的最值与方程和不等式密切相关

**典型题型**：
1. 三个"二次"的相互转化
2. 函数零点与方程根的关系
3. 不等式解集与函数图像的关系

### 2.4 韦达定理应用

**核心方法**：
- 韦达定理：若x₁,x₂是方程ax²+bx+c=0的两根，则x₁+x₂=-b/a，x₁x₂=c/a
- 韦达定理的逆定理：若x₁,x₂满足x₁+x₂=-b/a，x₁x₂=c/a，则x₁,x₂是方程ax²+bx+c=0的两根

**解题技巧**：
- 已知两根求方程
- 求根式代数式的值
- 求对称式的值
- 解含参问题

**二级结论**：
1. x₁²+x₂² = (x₁+x₂)² - 2x₁x₂
2. 1/x₁ + 1/x₂ = (x₁+x₂)/(x₁x₂)
3. x₁³+x₂³ = (x₁+x₂)³ - 3x₁x₂(x₁+x₂)
4. (x₁-x₂)² = (x₁+x₂)² - 4x₁x₂

**典型题型**：
1. 韦达定理的应用
2. 对称式的求值
3. 含参问题的求解
4. 根与系数的关系

### 2.5 含参不等式的解法

**解题技巧**：
- 先讨论a的符号（开口方向）
- 再讨论Δ的符号（根的情况）
- 最后讨论根的范围
- 注意边界条件的讨论

**二级结论**：
1. a>0时，Δ>0的解集为(-∞,x₁)∪(x₂,+∞)
2. a<0时，Δ>0的解集为(x₁,x₂)
3. a>0时，Δ=0的解集为x≠-b/(2a)
4. a>0时，Δ<0的解集为R

**典型题型**：
1. 含参数的一元二次不等式

## 模块2：基本不等式

### 2.1 均值不等式及变形

**核心方法**：
- 算术平均数与几何平均数不等式（均值不等式）：
  - 若a,b>0，则(a+b)/2 ≥ √(ab)，当且仅当a=b时取等号
  - 即a+b ≥ 2√(ab)
- 推广形式：a+b+c ≥ 3∛(abc)
- 反向形式：a+b ≤ 2√((a²+b²)/2)

**解题技巧**：
- 直接应用均值不等式
- 注意条件：a,b>0，当且仅当a=b时取等号
- 常用变形：ab ≤ ((a+b)/2)²，(a+b)² ≤ 2(a²+b²)
- 灵活运用"1的代换"和"配凑法"

**二级结论**：
1. 均值不等式成立的条件：a,b>0
2. 等号成立的条件：a=b
3. 若a,b>0，则(a+b)² ≥ 4ab
4. 若a,b>0，则a²+b² ≥ 2ab

**典型题型**：
1. 均值不等式的直接应用
2. 均值不等式的变形
3. 均值不等式的推广

### 2.2 条件求最值

**核心方法**：
- 常数和定值问题：利用均值不等式求最值
- 和定积最大、积定和最小
- 注意条件：正数、等号成立

**解题技巧**：
- 确定求最值的对象
- 寻找定值和
- 配凑均值不等式的形式
- 检验等号成立的条件

**二级结论**：
1. 若a+b=定值，则ab的最大值为((a+b)/2)²
2. 若ab=定值，则a+b的最小值为2√(ab)
3. 若a²+b²=定值，则ab的最大值为(a²+b²)/2
4. 若ab=定值，则a²+b²的最小值为2ab

**典型题型**：
1. 和定积最大
2. 积定和最小
3. 条件最值问题

### 2.3 1的代换技巧

**核心方法**：
- 利用1的代换：a + 1/a ≥ 2（a>0）
- 多次应用：a + 1/a + b + 1/b ≥ 4
- 拆分与组合：将复杂表达式拆分为若干项，每项形如x + 1/x

**解题技巧**：
- 观察表达式，寻找可以1的代换的部分
- 将1表示为x/x或y/y的形式
- 应用均值不等式
- 注意等号成立的条件

**二级结论**：
1. 若a>0，则a + 1/a ≥ 2，当且仅当a=1时取等号
2. 若a>0，b>0，则a + 1/a + b + 1/b ≥ 4，当且仅当a=b=1时取等号
3. 若a>0，则a + 1/a ≥ 2，当且仅当a=1时取等号

**典型题型**：
1. 1的代换的应用
2. 分式求最值
3. 分数不等式

### 2.4 齐次化方法

**核心方法**：
- 齐次式：各项次数相同的表达式
- 非齐次式：各项次数不同的表达式
- 齐次化：将非齐次式转化为齐次式

**解题技巧**：
- 观察表达式的次数
- 通过乘除适当的因子使各项次数相同
- 利用齐次式的不等式性质
- 注意定义域的限制

**二级结论**：
1. 若a,b>0，则(a+b)/(ab) ≥ 2/(a+b)
2. 若a,b>0，则(a+b)/(ab) = 1/a + 1/b ≥ 4/(a+b)
3. 若a,b,c>0，则(a+b+c)/(ab+bc+ca) ≥ 3/(a+b+c)

**典型题型**：
1. 齐次化方法的应用
2. 分式不等式
3. 分式最值
