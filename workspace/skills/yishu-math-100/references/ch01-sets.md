# 第1章 集合与常用逻辑用语

## 模块1：集合

### 1.1 集合的概念与表示

**核心方法**：
- 集合：具有某种特定性质的对象的总体
- 表示法：列举法（{a,b,c}）、描述法（{x|P(x)}）
- 元素与集合关系：∈（属于）、∉（不属于）
- 集合与集合关系：⊆（包含于）、⊂（真包含于）、∩（交集）、∪（并集）、ℂ（补集）

**解题技巧**：
- 列举法适用于有限集合，描述法适用于无限集合
- 描述法格式：{x∈M | P(x)}，其中M是限定范围
- 注意区分"元素"与"集合"：a∈A vs A⊆B

**二级结论**：
1. 空集是任何集合的子集，空集是任何非空集合的真子集
2. 若A⊆B，则|A|≤|B|
3. n(A∩B) = n(A) + n(B) - n(A∪B)

**典型题型**：
1. 集合的表示法判断与转换
2. 元素与集合关系的判断
3. 集合间包含关系的证明

---

### 1.2 集合间的关系

**核心方法**：
- 子集：若A⊆B，则A中任意元素都属于B
- 真子集：A⊂B且A≠B
- 不包含关系：A⊈B
- 相等：A=B ⇔ A⊆B且B⊆A

**解题技巧**：
- 证明A⊆B：取x∈A，证明x∈B
- 证明A=B：双向包含法
- 注意：空集是任何集合的子集，但不是任何非空集合的真子集

**二级结论**：
1. 若A⊆B且B⊆C，则A⊆C
2. n(P(A)) = 2^n（A的幂集元素个数）
3. 若A⊆B且|A|=n，|B|=m，则m≥n且m-n为非负整数

**典型题型**：
1. 子集关系的判断与证明
2. 相等集合的判定
3. 集合包含关系的综合应用

---

### 1.3 集合的运算

**核心方法**：
- 交集A∩B = {x | x∈A且x∈B}
- 并集A∪B = {x | x∈A或x∈B}
- 补集ℂ_A B = {x | x∈U且x∉B}
- 容斥原理：n(A∪B) = n(A) + n(B) - n(A∩B)

**解题技巧**：
- 画Venn图辅助理解
- 先求交集，再求并集
- 注意补集的取值范围（全集U）
- 运算顺序：先括号，后补集，最后交并

**二级结论**：
1. A∩A = A，A∪A = A
2. A∩B = B∩A，A∪B = B∪A
3. A∩(B∩C) = (A∩B)∩C
4. A∪(B∪C) = (A∪B)∪C
5. A∩(B∪C) = (A∩B)∪(A∩C)
6. A∪(B∩C) = (A∪B)∩(A∪C)

**典型题型**：
1. 集合的交、并、补运算
2. Venn图的应用
3. 容斥原理的应用

---

### 1.4 Venn图应用

**核心方法**：
- 用Venn图直观表示集合关系
- 标记各部分元素个数
- 通过图形关系推导等式

**解题技巧**：
- 先确定全集U的范围
- 标记各集合的重叠部分
- 利用图形关系列方程求解
- 注意区分"在...中"和"不在...中"

**二级结论**：
1. 任意集合A，A∩A = A，A∪A = A
2. A∩B = B∩A，A∪B = B∪A
3. A∩B = ∅ ⇔ A∪B = A⊕B（对称差）

**典型题型**：
1. Venn图元素计数
2. 集合关系与Venn图结合
3. 容斥原理的实际应用

---

### 1.5 容斥原理

**核心方法**：
- 二元容斥：n(A∪B) = n(A) + n(B) - n(A∩B)
- 三元容斥：n(A∪B∪C) = n(A) + n(B) + n(C) - n(A∩B) - n(A∩C) - n(B∩C) + n(A∩B∩C)
- 推广：n(⋃_{i=1}^n A_i) = Σ(-1)^{k+1} Σ_{1≤i_1<...<i_k≤n} n(A_{i_1}∩...∩A_{i_k})

**解题技巧**：
- 画Venn图辅助分析
- 标记各部分元素个数
- 按区域逐个求解
- 注意排除重复计数

**二级结论**：
1. 若A∩B = ∅，则n(A∪B) = n(A) + n(B)
2. 若A⊆B，则n(A∪B) = n(B)，n(A∩B) = n(A)

**典型题型**：
1. 二元容斥原理应用
2. 三元容斥原理应用
3. 多集合容斥的综合应用

## 模块2：常用逻辑用语

### 2.1 充分条件与必要条件

**核心方法**：
- 充分条件：A⇒B（若A成立，则B一定成立）
- 必要条件：B⇒A（若B成立，则A一定成立）
- 充要条件：A⇔B
- 既不充分也不必要：A⇏B且B⇏A

**解题技巧**：
- 转化为命题关系：A⇒B（A是B的充分条件）
- 使用逆否命题：A⇒B ⇔ ¬B⇒¬A
- 分析定义域和值域
- 对于含参数问题，分类讨论

**二级结论**：
1. A是B的充分条件 ⇔ B是A的必要条件
2. A是B的必要条件 ⇔ B是A的充分条件
3. A⇔B ⇔ A是B的充要条件
4. A⇒B ⇔ ¬B⇒¬A（逆否命题等价）

**典型题型**：
1. 充分必要条件的判断
2. 含参数问题的分类讨论
3. 逆否命题的应用

---

### 2.2 命题与量词

**核心方法**：
- 命题：可以判断真假的陈述句
- 全称量词：∀x∈M，P(x)（对于所有x，P(x)成立）
- 存在量词：∃x∈M，P(x)（存在x，使P(x)成立）
- 命题的否定：¬(∀x∈M，P(x)) ⇔ ∃x∈M，¬P(x)
- 命题的否定：¬(∃x∈M，P(x)) ⇔ ∀x∈M，¬P(x)

**解题技巧**：
- 找到量词的主语
- 注意否定时量词要取反
- 注意否定时谓语也要取反
- 举例验证：对于具体数值检验

**二级结论**：
1. 命题的否定是原命题的逆否命题的否定
2. 复合命题的否定要逐步否定
3. 量词否定后，定义域不变

**典型题型**：
1. 量词命题的判断
2. 命题的否定
3. 量词命题的真假判断

---

### 2.3 否命题与逆否命题

**核心方法**：
- 原命题：若P，则Q（P⇒Q）
- 逆命题：若Q，则P（Q⇒P）
- 否命题：若¬P，则¬Q（¬P⇒¬Q）
- 逆否命题：若¬Q，则¬P（¬Q⇒¬P）
- 逆否命题与原命题同真同假

**解题技巧**：
- 区分逆命题、否命题、逆否命题
- 逆否命题是判断原命题真假的重要工具
- 命题的否定：否定结论和条件
- 注意"若P则Q"的否定是"P且¬Q"

**二级结论**：
1. 原命题与逆否命题同真同假
2. 逆命题与否命题同真同假
3. 充分条件与必要条件的等价关系

**典型题型**：
1. 逆命题、否命题、逆否命题的构造
2. 命题真假的判断
3. 逆否命题的应用

---

### 2.4 逻辑联结词

**核心方法**：
- 且（∧）：P∧Q，当P、Q都为真时为真
- 或（∨）：P∨Q，当P、Q至少一个为真时为真
- 非（¬）：¬P，P为真时为假
- "且"运算：A∩B（交集）
- "或"运算：A∪B（并集）

**解题技巧**：
- 理解逻辑联结词的含义
- 注意"且"和"或"的区别
- 利用真值表判断
- 与集合运算对应理解

**二级结论**：
1. P∧Q的真值：真∧真=真，其他为假
2. P∨Q的真值：假∨假=假，其他为真
3. ¬P的真值：P真则¬P假，P假则¬P真
4. P∧Q = Q∧P，P∨Q = Q∨P

**典型题型**：
1. 逻辑联结词的真值判断
2. 复合命题的真值判断
3. 逻辑联结词与集合运算的结合

---

### 2.5 充分条件与必要条件的综合应用

**核心方法**：
- 充分条件：A⇒B
- 必要条件：B⇒A
- 充要条件：A⇔B
- 既不充分也不必要：A⇏B且B⇏A

**解题技巧**：
- 转化为集合包含关系：A⊆B（A是B的充分条件）
- 利用逆否命题：A⇒B ⇔ ¬B⇒¬A
- 分类讨论：对含参问题分类讨论
- 数形结合：用函数图像辅助判断

**二级结论**：
1. A是B的充分条件 ⇔ B是A的必要条件
2. A是B的必要条件 ⇔ B是A的充分条件
3. A⇔B ⇔ A是B的充要条件
4. A⇒B ⇔ ¬B⇒¬A（逆否命题等价）

**典型题型**：
1. 充分必要条件的判断
2. 含参问题的分类讨论
3. 数形结合的应用
4. 逆否命题的应用

---

### 2.6 逻辑推理方法

**核心方法**：
- 归纳推理：从特殊到一般
- 演绎推理：从一般到特殊
- 类比推理：从特殊到特殊
- 分析推理：逻辑分析、排除法

**解题技巧**：
- 注意推理的可靠性
- 演绎推理最可靠
- 类比推理需要验证
- 分析推理需要逻辑严密

**二级结论**：
1. 归纳推理的结论不一定可靠
2. 演绎推理的结论一定可靠
3. 类比推理的结论需要验证

**典型题型**：
1. 归纳推理的应用
2. 演绎推理的应用
3. 类比推理的应用
4. 分析推理的应用
