# 第11章 计数原理

## 模块1：排列与组合

### 1.1 分类加法计数原理

**核心方法：**

- **分类加法计数原理**：完成一件事有$n$类办法，在第1类办法中有$m_1$种不同的方法，在第2类办法中有$m_2$种不同的方法，……，在第$n$类办法中有$m_n$种不同的方法，则完成这件事共有$N = m_1 + m_2 + \cdots + m_n$种不同的方法
- **关键**：各类办法之间是"或"的关系（每类办法都能独立完成这件事）

**解题思路：**

1. 明确要完成的事是什么
2. 分析可以分成几类办法
3. 每类办法有多少种方法
4. 各类方法数相加

**典型题型：**

- 分类计数求方法总数
- 与排列组合结合的分类问题

**二级结论：**

- 分类要"不重不漏"：各类之间互斥，所有类合起来完备
- 分类标准要明确统一

### 1.2 分步乘法计数原理

**核心方法：**

- **分步乘法计数原理**：完成一件事需要分成$n$个步骤，做第1步有$m_1$种不同的方法，做第2步有$m_2$种不同的方法，……，做第$n$步有$m_n$种不同的方法，则完成这件事共有$N = m_1 \times m_2 \times \cdots \times m_n$种不同的方法
- **关键**：各步骤之间是"且"的关系（每步都不能独立完成，必须依次完成所有步骤）

**解题思路：**

1. 明确要完成的事是什么
2. 分析需要分成几个步骤
3. 每步有多少种方法
4. 各步方法数相乘

**典型题型：**

- 分步计数求方法总数
- 与排列组合结合的分步问题

**二级结论：**

- 分步要"有序完整"：各步按顺序进行，不能遗漏步骤
- 分类的标志是"或"，分步的标志是"且"
- 复杂问题常"先分类后分步"或"先分步后分类"

### 1.3 排列数公式

**核心方法：**

- **排列**：从$n$个不同元素中取出$m$个元素，按照一定的顺序排成一列
- **排列数公式**：
  $$A_n^m = n(n-1)(n-2)\cdots(n-m+1) = \dfrac{n!}{(n-m)!}$$
- **全排列**：$A_n^n = n! = n(n-1)(n-2)\cdots 2 \cdot 1$
- **规定**：$0! = 1$

**解题思路：**

1. 判断是否为排列问题（元素是否有序）
2. 选择合适的公式计算
3. 注意特殊情况

**典型题型：**

- 排列数的计算
- 排列方程/不等式
- 简单排列应用题

**二级结论：**

- $A_n^m = n \cdot A_{n-1}^{m-1}$
- $A_n^m = A_n^k \cdot A_{n-k}^{m-k}$（$k \leq m$）
- $A_n^n = n!$，$A_n^1 = n$，$A_n^0 = 1$

### 1.4 组合数性质

**核心方法：**

- **组合**：从$n$个不同元素中取出$m$个元素组成一组（不考虑顺序）
- **组合数公式**：
  $$C_n^m = \dfrac{A_n^m}{m!} = \dfrac{n!}{m!(n-m)!}$$
- **组合数性质**：
  - $C_n^m = C_n^{n-m}$（对称性）
  - $C_n^m = C_{n-1}^{m-1} + C_{n-1}^m$（递推公式，即杨辉三角）
  - $C_n^0 + C_n^1 + \cdots + C_n^n = 2^n$
  - $C_n^0 + C_n^2 + C_n^4 + \cdots = C_n^1 + C_n^3 + C_n^5 + \cdots = 2^{n-1}$
  - $kC_n^k = nC_{n-1}^{k-1}$

**解题思路：**

1. 判断是否为组合问题（元素是否无序）
2. 利用组合数性质简化计算
3. 利用递推公式证明恒等式

**典型题型：**

- 组合数的计算与化简
- 组合恒等式的证明
- 组合数性质的应用

**二级结论：**

- $C_n^m = C_n^{n-m}$，当$m > \dfrac{n}{2}$时用$C_n^{n-m}$简化
- 杨辉三角：第$n$行第$k$个数为$C_{n-1}^{k-1}$
- $\sum_{k=0}^{n} C_n^k = 2^n$，$\sum_{k=0}^{n} kC_n^k = n \cdot 2^{n-1}$

### 1.5 捆绑法、插空法与隔板法

**核心方法：**

- **捆绑法**：某些元素必须相邻时，将它们"捆绑"成一个整体，与其他元素一起排列，再乘以捆绑元素内部的排列数
  - 步骤：①捆绑②整体排列③内部排列
  - 公式：$A_{n-k+1}^{n-k+1} \times A_k^k$（$k$个元素捆绑，共$n$个元素）

- **插空法**：某些元素必须不相邻时，先排其他元素，再将要求不相邻的元素插入空隙中
  - 步骤：①排其他元素②找空隙③选空插入
  - $n$个元素排好后有$n+1$个空隙（含两端），若$k$个元素不相邻，则有$A_{n+1}^k$种插入方式

- **隔板法**：将$n$个相同元素分给$m$个不同对象，每个对象至少1个
  - 公式：$C_{n-1}^{m-1}$（在$n-1$个间隔中选$m-1$个放隔板）
  - 变形：允许某些对象分到0个时，令$x_i' = x_i + 1$，公式变为$C_{n+m-1}^{m-1}$

**解题思路：**

1. **相邻问题**$\to$捆绑法
2. **不相邻问题**$\to$插空法
3. **相同元素分组**$\to$隔板法
4. 复杂问题可能需要多种方法组合

**典型题型：**

- 某些人必须站在一起的排列
- 某些人必须不相邻的排列
- 将相同物品分配给不同人

**二级结论：**

- 捆绑法中，有多个不同的"捆绑组"时，各组分别计算内部排列，再乘以整体排列
- 插空法中，若要求不相邻的元素也有顺序，用$A$；无顺序则用$C$
- 隔板法只适用于"相同元素分给不同对象"
- 隔板法变形技巧：每个至少0个$\to$先给每个1个，再用标准隔板法

### 1.6 分组分配问题

**核心方法：**

- **均匀分组**（每组人数相同）：先选人分组，再除以组数的阶乘（因为组的顺序不重要）
  - 例：将$6$人平均分成$3$组：$\dfrac{C_6^2 C_4^2 C_2^2}{A_3^3} = 15$种

- **非均匀分组**（各组人数不同）：直接用组合数相乘
  - 例：将$6$人分成$1$人、$2$人、$3$人三组：$C_6^1 C_5^2 C_3^3 = 60$种

- **部分均匀分组**（有些组人数相同，有些不同）：均匀部分除以阶乘
  - 例：将$7$人分成$2$人、$2$人、$3$人三组：$\dfrac{C_7^2 C_5^2 C_3^3}{A_2^2} = 105$种

- **分配问题**：分组后再分配给不同的对象
  - 均匀分组后分配：$\dfrac{C_6^2 C_4^2 C_2^2}{A_3^3} \times A_3^3 = C_6^2 C_4^2 C_2^2 = 90$种
  - 或者直接分步分配

**解题思路：**

1. 区分"分组"和"分配"：分组不区分组的编号，分配区分
2. 区分"均匀"和"非均匀"
3. 均匀分组要除以阶乘（消去组的顺序）
4. 分配问题在分组基础上乘以组数的排列

**典型题型：**

- 将人/物品分成若干组
- 分组后分配到不同位置
- 有条件限制的分组分配

**二级结论：**

- **分组vs分配**：分组不区分编号，分配区分编号
- **均匀分组必除阶乘**：$k$组人数相同，除以$A_k^k = k!$
- **分配$=$分组$\times$排列**：将$m$组分配到$m$个不同位置，乘$A_m^m$
- **有序分组**：若组有编号（如甲组、乙组），则不需要除阶乘
- 经典结论：$n$个不同元素分成$k$组，各组人数为$n_1, n_2, \ldots, n_k$，则分法数为$\dfrac{n!}{n_1! n_2! \cdots n_k! \cdot m_1! m_2! \cdots}$（其中$m_i$为人数相同的组的个数）

## 模块2：二项式定理

### 2.1 展开式通项公式

**核心方法：**

- **二项式定理**：
  $$(a+b)^n = \sum_{k=0}^{n} C_n^k a^{n-k} b^k$$
  $$= C_n^0 a^n + C_n^1 a^{n-1}b + C_n^2 a^{n-2}b^2 + \cdots + C_n^n b^n$$

- **通项公式**：$(a+b)^n$展开式的第$r+1$项为
  $$T_{r+1} = C_n^r a^{n-r} b^r$$（$r = 0, 1, 2, \ldots, n$）

- **注意**：
  - 第$r+1$项的二项式系数是$C_n^r$（不是$C_n^{r+1}$）
  - 项数$= n + 1$
  - 各项次数之和$= n$

**解题思路：**

1. 写出通项公式$T_{r+1} = C_n^r a^{n-r} b^r$
2. 根据条件（如含$x$的某次幂）确定$r$的值
3. 代入$r$求对应项

**典型题型：**

- 求展开式中某一项
- 求展开式中含$x^k$的项的系数
- 求展开式中的最大项

**二级结论：**

- $(a+b)^n$展开式中，$a$的指数从$n$递减到$0$，$b$的指数从$0$递增到$n$
- 通项$T_{r+1}$中，$a$的指数$= n - r$，$b$的指数$= r$，$a,b$指数之和$= n$
- 系数与二项式系数的区别：系数可能含$a, b$本身的系数，二项式系数仅指$C_n^r$

### 2.2 二项式系数性质

**核心方法：**

- **对称性**：$C_n^k = C_n^{n-k}$（与首末两端"等距离"的两个二项式系数相等）
- **增减性**：当$k < \dfrac{n+1}{2}$时，$C_n^k$递增；当$k > \dfrac{n+1}{2}$时，$C_n^k$递减
- **最大值**：
  - $n$为偶数时，中间项$C_n^{n/2}$最大
  - $n$为奇数时，中间两项$C_n^{(n-1)/2}$和$C_n^{(n+1)/2}$相等且最大
- **二项式系数之和**：$C_n^0 + C_n^1 + \cdots + C_n^n = 2^n$
- **奇数项与偶数项**：$C_n^0 + C_n^2 + \cdots = C_n^1 + C_n^3 + \cdots = 2^{n-1}$

**解题思路：**

1. 利用对称性化简
2. 利用增减性判断最大项位置
3. 利用求和公式计算

**典型题型：**

- 求二项式系数最大的项
- 二项式系数之和
- 证明二项式系数相关等式

**二级结论：**

- 二项式系数最大值：
  - $n$偶数：$C_n^{n/2}$
  - $n$奇数：$C_n^{(n-1)/2} = C_n^{(n+1)/2}$
- 展开式中系数最大的项不一定与二项式系数最大的项相同（当$a, b$含系数时）
- 求系数最大的项需要解不等式：$\dfrac{T_{r+1}}{T_r} \geq 1$且$\dfrac{T_{r+2}}{T_{r+1}} \leq 1$

### 2.3 赋值法求系数和

**核心方法：**

- **赋值法**：在恒等式$(a+b)^n = \sum_{k=0}^n C_n^k a^{n-k}b^k$中对$a, b$赋特殊值
  - 令$a=1, b=1$：$2^n = \sum_{k=0}^n C_n^k$（所有二项式系数之和）
  - 令$a=1, b=-1$：$0 = C_n^0 - C_n^1 + C_n^2 - \cdots$（奇数项$=$偶数项$= 2^{n-1}$）
  - 令$a=1, b=x$：$(1+x)^n = \sum_{k=0}^n C_n^k x^k$
  - 令$a=1, b=2$：$3^n = \sum_{k=0}^n C_n^k \cdot 2^k$（各项系数乘以$2^k$之和）

- **求展开式各项系数之和**：令所有变量为1
- **求奇数项/偶数项系数之和**：利用$a=1,b=1$和$a=1,b=-1$相加减

**解题思路：**

1. 根据所求系数和的类型选择赋值
2. 联立方程组求解
3. 注意区分"二项式系数之和"和"系数之和"

**典型题型：**

- 求展开式中各项系数之和
- 求奇数项/偶数项系数之和
- 求满足某条件的系数之和

**二级结论：**

- 二项式系数之和$= 2^n$（令$a=b=1$）
- 展开式各项系数之和$= f(1)$（将$x$替换为1）
- 奇数项系数和$= \dfrac{f(1) + f(-1)}{2}$
- 偶数项系数和$= \dfrac{f(1) - f(-1)}{2}$
- 其中$f(x)$为展开式整理后的多项式

### 2.4 多项式展开

**核心方法：**

- **三项式及多项式展开**：转化为二项式处理
  - 例：$(a+b+c)^n$可视为$((a+b)+c)^n$展开
- **求某一项的系数**：利用通项公式逐层展开
- **多项式乘积的系数**：逐项相乘后合并同类项

**解题思路：**

1. 将多项式转化为若干二项式的乘积
2. 对每个二项式使用通项公式
3. 组合各部分得到目标项

**典型题型：**

- 三项式展开中某项的系数
- 两个二项式乘积中某项的系数
- 求多项式展开的常数项

**二级结论：**

- $(a+b+c)^n$中$a^p b^q c^r$（$p+q+r=n$）的系数$= \dfrac{n!}{p!q!r!}$（多项式定理）
- 两个展开式乘积的系数：对各项系数进行卷积运算
- 常数项：令通项中变量的指数为0，解方程求$r$
