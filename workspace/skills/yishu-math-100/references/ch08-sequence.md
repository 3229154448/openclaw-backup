# 第8章 数列

## 模块1：等差、等比数列问题

### 1.1 等差数列

**核心方法：**
- **定义**：$a_{n+1} - a_n = d$（常数）$\iff \{a_n\}$为等差数列
- **通项公式**：$a_n = a_1 + (n-1)d$
- **前$n$项和公式**：$S_n = \frac{n(a_1 + a_n)}{2} = \frac{n}{2}[2a_1 + (n-1)d] = n a_1 + \frac{n(n-1)}{2}d$
- **中项公式**：$2a_n = a_{n-1} + a_{n+1}$
- **性质**：
  - $m+n=p+q \iff a_m + a_n = a_p + a_q$
  - $\{a_n\}$为等差数列 $\iff \{a_n\}$的前$n$项和$S_n$是关于$n$的二次函数

**解题思路：**
1. 利用等差数列的通项公式求通项
2. 利用等差数列的前$n$项和公式求和
3. 利用等差数列的性质化简和证明

**典型题型：**
- 等差数列的通项公式
- 等差数列的前$n$项和公式
- 等差数列的性质应用
- 等差数列的综合应用

**二级结论：**
- $a_n = a_m + (n-m)d$
- $S_n = \frac{d}{2}n^2 + (a_1 - \frac{d}{2})n$
- $a_n = A n + B$（$A = d, B = a_1 - d$）
- $S_n = An^2 + Bn$（$A = \frac{d}{2}, B = a_1 - \frac{d}{2}$）
- $S_{2n-1} = (2n-1)a_n$
- $S_n = n a_{\frac{n+1}{2}}$（$n$为奇数）

### 1.2 等比数列

**核心方法：**
- **定义**：$\frac{a_{n+1}}{a_n} = q$（常数）$\iff \{a_n\}$为等比数列
- **通项公式**：$a_n = a_1 q^{n-1}$（$q \neq 0$）
- **前$n$项和公式**：$S_n = \frac{a_1(1 - q^n)}{1 - q}$（$q \neq 1$），$S_n = n a_1$（$q = 1$）
- **中项公式**：$a_n^2 = a_{n-1} \cdot a_{n+1}$
- **性质**：
  - $m+n=p+q \iff a_m \cdot a_n = a_p \cdot a_q$
  - $\{a_n\}$为等比数列 $\iff \{\ln a_n\}$为等差数列（$a_n > 0$）

**解题思路：**
1. 利用等比数列的通项公式求通项
2. 利用等比数列的前$n$项和公式求和
3. 利用等比数列的性质化简和证明

**典型题型：**
- 等比数列的通项公式
- 等比数列的前$n$项和公式
- 等比数列的性质应用
- 等比数列的综合应用

**二级结论：**
- $a_n = a_m \cdot q^{n-m}$
- $S_n = \frac{a_1 - a_{n+1}}{1 - q}$（$q \neq 1$）
- $a_n = A \cdot q^n$（$A = \frac{a_1}{q}$）
- $S_n = A(1 - q^n)$（$A = \frac{a_1}{1 - q}$）
- $S_{2n-1} = (2n-1)a_n^2$（$q \neq 1$）
- $S_n = n \sqrt{a_n \cdot a_{n+1}}$（$q > 0$）

### 1.3 数列性质

**核心方法：**
- **等差数列性质**：
  - $m+n=p+q \iff a_m + a_n = a_p + a_q$
  - $\{a_n\}$为等差数列 $\iff \{a_n\}$的前$n$项和$S_n$是关于$n$的二次函数
  - $S_n = n a_{\frac{n+1}{2}}$（$n$为奇数）

- **等比数列性质**：
  - $m+n=p+q \iff a_m \cdot a_n = a_p \cdot a_q$
  - $\{a_n\}$为等比数列 $\iff \{\ln a_n\}$为等差数列（$a_n > 0$）

**解题思路：**
1. 利用等差数列的性质化简和证明
2. 利用等比数列的性质化简和证明
3. 利用等差数列和等比数列的相互转化

**典型题型：**
- 等差数列的性质应用
- 等比数列的性质应用
- 等差数列和等比数列的综合应用

**二级结论：**
- $a_n = a_m + (n-m)d$
- $a_n = a_m \cdot q^{n-m}$
- $S_{2n-1} = (2n-1)a_n$（等差数列）
- $S_{2n-1} = (2n-1)a_n^2$（等比数列，$q \neq 1$）
- $S_n = n a_{\frac{n+1}{2}}$（等差数列，$n$为奇数）
- $S_n = n \sqrt{a_n \cdot a_{n+1}}$（等比数列，$q > 0$）

### 1.4 中项公式

**核心方法：**
- **等差数列中项**：$2a_n = a_{n-1} + a_{n+1}$
- **等比数列中项**：$a_n^2 = a_{n-1} \cdot a_{n+1}$

**解题思路：**
1. 利用中项公式进行化简和证明
2. 利用中项公式求解未知数
3. 利用中项公式解决综合问题

**典型题型：**
- 等差数列的中项公式应用
- 等比数列的中项公式应用
- 中项公式的综合应用

**二级结论：**
- $2a_n = a_{n-1} + a_{n+1}$
- $a_n^2 = a_{n-1} \cdot a_{n+1}$
- $a_{n-1} + a_{n+1} = 2a_n$（等差数列）
- $a_{n-1} \cdot a_{n+1} = a_n^2$（等比数列）
- $a_n = \frac{a_{n-1} + a_{n+1}}{2}$（等差数列）
- $a_n = \sqrt{a_{n-1} \cdot a_{n+1}}$（等比数列，$a_n > 0$）

## 模块2：求通项与求和

### 2.1 累加法

**核心方法：**
- **定义**：$a_{n+1} - a_n = f(n) \implies a_n = a_1 + \sum_{k=1}^{n-1} f(k)$
- **公式**：$a_n = a_1 + \sum_{k=1}^{n-1} (a_{k+1} - a_k)$

**解题思路：**
1. 利用累加法求通项
2. 利用累加法化简表达式
3. 利用累加法解决综合问题

**典型题型：**
- 利用累加法求通项
- 累加法的应用
- 累加法的综合应用

**二级结论：**
- $a_n = a_1 + \sum_{k=1}^{n-1} (a_{k+1} - a_k)$
- $\sum_{k=1}^{n-1} (a_{k+1} - a_k) = a_n - a_1$
- $a_n = a_1 + \sum_{k=1}^{n-1} f(k)$（$a_{k+1} - a_k = f(k)$）

### 2.2 累乘法

**核心方法：**
- **定义**：$\frac{a_{n+1}}{a_n} = f(n) \implies a_n = a_1 \cdot \prod_{k=1}^{n-1} f(k)$
- **公式**：$a_n = a_1 \cdot \prod_{k=1}^{n-1} \frac{a_{k+1}}{a_k}$

**解题思路：**
1. 利用累乘法求通项
2. 利用累乘法化简表达式
3. 利用累乘法解决综合问题

**典型题型：**
- 利用累乘法求通项
- 累乘法的应用
- 累乘法的综合应用

**二级结论：**
- $a_n = a_1 \cdot \prod_{k=1}^{n-1} \frac{a_{k+1}}{a_k}$
- $\prod_{k=1}^{n-1} \frac{a_{k+1}}{a_k} = \frac{a_n}{a_1}$
- $a_n = a_1 \cdot \prod_{k=1}^{n-1} f(k)$（$\frac{a_{k+1}}{a_k} = f(k)$）

### 2.3 待定系数法

**核心方法：**
- **等差数列**：设$a_n = An + B$，利用$a_{n+1} - a_n = A$求$A$，利用$a_1$求$B$
- **等比数列**：设$a_n = A \cdot q^n + B$，利用$\frac{a_{n+1}}{a_n} = q$求$A, q, B$

**解题思路：**
1. 根据数列的特点选择合适的待定系数形式
2. 利用已知条件求解待定系数
3. 验证并写出通项公式

**典型题型：**
- 等差数列的待定系数法
- 等比数列的待定系数法
- 待定系数法的综合应用

**二级结论：**
- 等差数列：$a_n = An + B$（$A = d, B = a_1 - d$）
- 等比数列：$a_n = A \cdot q^n + B$（$A = \frac{a_1}{q}, B = 0$）
- $S_n = An^2 + Bn$（等差数列）
- $S_n = A(1 - q^n) + B$（等比数列，$q \neq 1$）

### 2.4 错位相减法

**核心方法：**
- **适用**：等差数列与等比数列对应项相乘
- **步骤**：
  1. 写出$S_n$和$qS_n$
  2. 相减消去等比部分
  3. 化简并求解

**解题思路：**
1. 识别错位相减法的适用条件
2. 正确写出$S_n$和$qS_n$
3. 相减消去等比部分
4. 化简并求解

**典型题型：**
- 错位相减法的应用
- 错位相减法的综合应用
- 错位相减法的变形应用

**二级结论：**
- $S_n = a_1 + a_2 + \cdots + a_n$
- $qS_n = a_1 q + a_2 q + \cdots + a_n q$
- $S_n - qS_n = a_1 - a_n q + a_2 - a_{n-1} q + \cdots + a_{n-1} - a_2 q + a_n - a_1 q$
- $S_n = \frac{a_1 - a_n q}{1 - q}$（$q \neq 1$）

### 2.5 裂项相消法

**核心方法：**
- **定义**：将一项拆成两项之差，求和时中间项抵消
- **常见形式**：
  - $\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$
  - $\frac{1}{(2n-1)(2n+1)} = \frac{1}{2}(\frac{1}{2n-1} - \frac{1}{2n+1})$
  - $\frac{1}{\sqrt{n} + \sqrt{n+1}} = \sqrt{n+1} - \sqrt{n}$

**解题思路：**
1. 识别裂项相消法的适用条件
2. 正确裂项
3. 求和并化简

**典型题型：**
- 裂项相消法的应用
- 裂项相消法的综合应用
- 裂项相消法的变形应用

**二级结论：**
- $\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$
- $\frac{1}{(2n-1)(2n+1)} = \frac{1}{2}(\frac{1}{2n-1} - \frac{1}{2n+1})$
- $\frac{1}{\sqrt{n} + \sqrt{n+1}} = \sqrt{n+1} - \sqrt{n}$
- $\frac{1}{n(n+1)(n+2)} = \frac{1}{2}(\frac{1}{n(n+1)} - \frac{1}{(n+1)(n+2)})$
- $\sum_{k=1}^{n} \frac{1}{k(k+1)} = 1 - \frac{1}{n+1} = \frac{n}{n+1}$

### 2.6 倒序相加法

**核心方法：**
- **定义**：将数列的项按相反顺序排列，然后相加
- **公式**：$S_n = \frac{n}{2}(a_1 + a_n)$

**解题思路：**
1. 识别倒序相加法的适用条件
2. 正确倒序
3. 相加并化简

**典型题型：**
- 倒序相加法的应用
- 倒序相加法的综合应用
- 倒序相加法的变形应用

**二级结论：**
- $S_n = \frac{n}{2}(a_1 + a_n)$
- $S_n = n a_{\frac{n+1}{2}}$（$n$为奇数，等差数列）
- $S_n = \frac{n}{2}(a_1 + a_n)$（等差数列）
- $S_n = n a_{\frac{n+1}{2}}$（等差数列，$n$为奇数）

### 2.7 分组求和法

**核心方法：**
- **定义**：将数列分成几个简单的数列分别求和
- **常见形式**：
  - $a_n = A_n + B_n$（$A_n$为等差数列，$B_n$为等比数列）
  - $a_n = A_n + B_n + C_n$（$A_n, B_n, C_n$分别为等差数列、等比数列、其他数列）

**解题思路：**
1. 识别分组求和法的适用条件
2. 将数列分成几个简单的数列
3. 分别求和并合并

**典型题型：**
- 分组求和法的应用
- 分组求和法的综合应用
- 分组求和法的变形应用

**二级结论：**
- $a_n = A_n + B_n$
- $S_n = S_n(A_n) + S_n(B_n)$
- $\sum_{k=1}^{n} (A_k + B_k) = \sum_{k=1}^{n} A_k + \sum_{k=1}^{n} B_k$

## 模块3：数列拔高题型

### 3.1 放缩法证明不等式

**核心方法：**
- **定义**：利用放缩法将数列的项放大或缩小，从而证明不等式
- **技巧**：
  - 放大：$a_n \leq b_n \implies \sum a_n \leq \sum b_n$
  - 缩小：$a_n \geq b_n \implies \sum a_n \geq \sum b_n$

**解题思路：**
1. 识别放缩法的适用条件
2. 选择合适的放缩方式
3. 证明不等式

**典型题型：**
- 放缩法的应用
- 放缩法的综合应用
- 放缩法的变形应用

**二级结论：**
- $a_n \leq b_n \implies \sum_{k=1}^{n} a_k \leq \sum_{k=1}^{n} b_k$
- $a_n \geq b_n \implies \sum_{k=1}^{n} a_k \geq \sum_{k=1}^{n} b_k$
- $\frac{1}{n+1} < \ln(1 + \frac{1}{n}) < \frac{1}{n}$
- $\frac{1}{n+1} < \ln(\frac{n+1}{n}) < \frac{1}{n}$

### 3.2 数列单调性

**核心方法：**
- **定义**：$a_{n+1} > a_n \implies \{a_n\}$单调递增；$a_{n+1} < a_n \implies \{a_n\}$单调递减
- **方法**：
  - 作差法：$a_{n+1} - a_n$
  - 作商法：$\frac{a_{n+1}}{a_n}$（$a_n > 0$）

**解题思路：**
1. 识别数列单调性的判断方法
2. 选择合适的方法
3. 判断数列的单调性

**典型题型：**
- 数列单调性的判断
- 数列单调性的综合应用
- 数列单调性的变形应用

**二级结论：**
- $a_{n+1} > a_n \implies \{a_n\}$单调递增
- $a_{n+1} < a_n \implies \{a_n\}$单调递减
- $a_{n+1} - a_n > 0 \implies \{a_n\}$单调递增
- $\frac{a_{n+1}}{a_n} > 1 \implies \{a_n\}$单调递增（$a_n > 0$）
- $\frac{a_{n+1}}{a_n} < 1 \implies \{a_n\}$单调递减（$a_n > 0$）

### 3.3 周期数列

**核心方法：**
- **定义**：$a_{n+k} = a_n \implies \{a_n\}$为周期数列（$k$为周期）
- **方法**：
  - 作差法：$a_{n+2} - a_{n+1} = a_{n+1} - a_n$（二阶差分）
  - 作商法：$\frac{a_{n+2}}{a_{n+1}} = \frac{a_{n+1}}{a_n}$（二阶比例）

**解题思路：**
1. 识别周期数列的判断方法
2. 选择合适的方法
3. 判断数列的周期性

**典型题型：**
- 周期数列的判断
- 周期数列的综合应用
- 周期数列的变形应用

**二级结论：**
- $a_{n+k} = a_n \implies \{a_n\}$为周期数列（$k$为周期）
- $a_{n+2} - a_{n+1} = a_{n+1} - a_n \implies \{a_n\}$为等差数列
- $\frac{a_{n+2}}{a_{n+1}} = \frac{a_{n+1}}{a_n} \implies \{a_n\}$为等比数列
- $a_{n+2} = a_n \implies \{a_n\}$为周期数列（周期为2）
- $a_{n+3} = a_n \implies \{a_n\}$为周期数列（周期为3）

### 3.4 递推数列

**核心方法：**
- **一阶递推**：$a_{n+1} = f(n, a_n)$
- **二阶递推**：$a_{n+2} = f(n, a_{n+1}, a_n)$
- **方法**：
  - 变换法：$a_{n+1} - a_n = f(n)$（累加法）
  - 除法法：$\frac{a_{n+1}}{a_n} = f(n)$（累乘法）
  - 累加法：$a_n = a_1 + \sum_{k=1}^{n-1} f(k)$
  - 累乘法：$a_n = a_1 \cdot \prod_{k=1}^{n-1} f(k)$

**解题思路：**
1. 识别递推数列的类型
2. 选择合适的解法
3. 求解通项公式

**典型题型：**
- 递推数列的求解
- 递推数列的综合应用
- 递推数列的变形应用

**二级结论：**
- $a_{n+1} = f(n, a_n) \implies a_n = a_1 + \sum_{k=1}^{n-1} f(k)$（累加法）
- $\frac{a_{n+1}}{a_n} = f(n) \implies a_n = a_1 \cdot \prod_{k=1}^{n-1} f(k)$（累乘法）
- $a_{n+1} - a_n = d \implies \{a_n\}$为等差数列
- $\frac{a_{n+1}}{a_n} = q \implies \{a_n\}$为等比数列
- $a_{n+2} = a_{n+1} + a_n \implies \{a_n\}$为斐波那契数列
- $a_{n+2} = a_{n+1} \cdot a_n \implies \{a_n\}$为等比数列
