## 模块1：同角三角函数关系与诱导公式

### 4.1 同角三角函数关系

**核心方法**：
- 基本关系式：
  - 平方关系：sin²x + cos²x = 1
  - 商数关系：tan x = sin x / cos x
  - 倒数关系：cot x = 1 / tan x，csc x = 1 / sin x，sec x = 1 / cos x
- 升幂降幂：
  - 1 + cos²x = 2cos²x + sin²x
  - 1 - cos²x = 2sin²x + cos²x

**解题技巧**：
- 熟记基本关系式
- 灵活运用平方关系
- 利用商数关系进行转化
- 注意角的范围对三角函数值符号的影响

**二级结论**：
1. sin²x + cos²x = 1（平方关系）
2. tan x = sin x / cos x（商数关系）
3. cot x = 1 / tan x（倒数关系）
4. sin²x = (1 - cos 2x)/2，cos²x = (1 + cos 2x)/2（降幂公式）
5. sin 2x = 2sin x cos x（二倍角公式）

**典型题型**：
1. 同角三角函数关系的应用
2. 三角函数的求值
3. 三角恒等式的证明
4. 三角函数的化简

### 4.2 诱导公式

**核心方法**：
- 诱导公式口诀：奇变偶不变，符号看象限
- 奇变偶不变：
  - kπ/2 + x（k为整数）
  - 若k为奇数，函数名改变：sin→cos，cos→sin，tan→cot
  - 若k为偶数，函数名不变
- 符号看象限：
  - 将x看作锐角，看原函数在kπ/2 + x所在象限的符号

**常用诱导公式**：
- sin(π/2 - x) = cos x
- cos(π/2 - x) = sin x
- tan(π/2 - x) = cot x
- sin(π/2 + x) = cos x
- cos(π/2 + x) = -sin x
- tan(π/2 + x) = -cot x
- sin(π - x) = sin x
- cos(π - x) = -cos x
- tan(π - x) = -tan x
- sin(π + x) = -sin x
- cos(π + x) = -cos x
- tan(π + x) = tan x
- sin(-x) = -sin x
- cos(-x) = cos x
- tan(-x) = -tan x

**解题技巧**：
- 记忆口诀：奇变偶不变，符号看象限
- 灵活运用诱导公式
- 注意角的范围
- 化简三角函数式

**二级结论**：
1. 诱导公式的核心：sin(π/2 + x) = cos x，cos(π/2 + x) = -sin x
2. 诱导公式可以用于化简三角函数式
3. 诱导公式可以用于求三角函数值

**典型题型**：
1. 诱导公式的应用
2. 三角函数的化简

4. 三角恒等式的证明

## 模块2：三角恒等变换

### 4.1 两角和差公式

**核心方法**：
- 两角和公式：
  - sin(x + y) = sin x cos y + cos x sin y
  - cos(x + y) = cos x cos y - sin x sin y
  - tan(x + y) = (tan x + tan y) / (1 - tan x tan y)
- 两角差公式：
  - sin(x - y) = sin x cos y - cos x sin y
  - cos(x - y) = cos x cos y + sin x sin y
  - tan(x - y) = (tan x - tan y) / (1 + tan x tan y)

**解题技巧**：
- 熟记两角和差公式
- 灵活运用公式
- 注意公式的变形
- 化简三角函数式

**二级结论**：
1. sin(x + y) = sin x cos y + cos x sin y
2. cos(x + y) = cos x cos y - sin x sin y
3. tan(x + y) = (tan x + tan y) / (1 - tan x tan y)
4. sin(x - y) = sin x cos y - cos x sin y
5. cos(x - y) = cos x cos y + sin x sin y
6. tan(x - y) = (tan x - tan y) / (1 + tan x tan y)

**典型题型**：

### 4.3 辅助角公式

**核心方法**：
- 辅助角公式：a sin x + b cos x = √(a² + b²) sin(x + φ)，其中tan φ = b/a
- 另一种形式：a sin x + b cos x = √(a² + b²) cos(x - φ)，其中tan φ = a/b

**解题技巧**：
- 将a sin x + b cos x转化为单一三角函数
- 求辅助角φ：tan φ = b/a
- 注意φ的范围
- 用于化简和求值

**二级结论**：
1. a sin x + b cos x = √(a² + b²) sin(x + φ)，其中tan φ = b/a
2. a sin x + b cos x = √(a² + b²) cos(x - φ)，其中tan φ = a/b
3. 辅助角公式可以用于化简三角函数式
4. 辅助角公式可以用于求三角函数的最大值和最小值

**典型题型**：
1. 辅助角公式的应用

4. 三角函数的最值

### 5.1 五点作图法

**核心方法**：
- 五点作图法：利用五个关键点画出y = A sin(ωx + φ) + k的图像
- 五个关键点：
  1. 峰值点：x = π/2ω - φ/ω，y = A + k
  2. 0点：x = 0 - φ/ω，y = k
  3. 谷值点：x = 3π/2ω - φ/ω，y = -A + k
  4. 0点：x = π - φ/ω，y = k
  5. 峰值点：x = 2π - φ/ω，y = A + k

**解题技巧**：
- 确定五个关键点的坐标

**二级结论**：
