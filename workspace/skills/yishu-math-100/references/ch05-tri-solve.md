# 第5章 解三角形

## 模块1：相关定理的基本运用

### 1.1 正弦定理

**核心方法：**
- **公式**：$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$（$R$为外接圆半径）
- **边角互化**：$a=2R\sin A$, $b=2R\sin B$, $c=2R\sin C$
- **正弦定理的推论**：
  - $\frac{\sin A}{a} = \frac{\sin B}{b} = \frac{\sin C}{c} = \frac{1}{2R}$
  - $\frac{a}{b} = \frac{\sin A}{\sin B}$

**解题思路：**
1. 已知两角一边，求其他边角
2. 已知两边及其中一边的对角，用正弦定理求角（注意增根）

**典型题型：**
- 已知三角形两角及一边，求其他边角
- 已知两边及其中一边的对角，解三角形
- 利用正弦定理求三角形的面积

**二级结论：**
- $\sin A : \sin B : \sin C = a : b : c$
- 在非直角三角形中，大边对大角，大角对大边
- $\sin^2 A + \sin^2 B + \sin^2 C = 2 + 2\cos A\cos B\cos C$

### 1.2 余弦定理

**核心方法：**
- **公式**：
  - $a^2 = b^2 + c^2 - 2bc\cos A$
  - $b^2 = a^2 + c^2 - 2ac\cos B$
  - $c^2 = a^2 + b^2 - 2ab\cos C$
- **余弦定理的推论**：
  - $\cos A = \frac{b^2 + c^2 - a^2}{2bc}$
  - $\cos B = \frac{a^2 + c^2 - b^2}{2ac}$
  - $\cos C = \frac{a^2 + b^2 - c^2}{2ab}$

**解题思路：**
1. 已知三边，求角
2. 已知两边及夹角，求第三边
3. 判断三角形的形状（锐角/直角/钝角）

**典型题型：**
- 已知三边求角
- 已知两边及夹角求第三边
- 判断三角形的形状和类型
- 利用余弦定理证明线段平方关系

**二级结论：**
- $a^2 + b^2 + c^2 = 2(s^2 - 4Rr - r^2)$（$s$为半周长，$R$为外接圆半径，$r$为内切圆半径）
- $a^2 + b^2 + c^2 = 2(b^2\cos^2 C + c^2\cos^2 A + a^2\cos^2 B)$

### 1.3 面积公式

**核心方法：**
- **基本公式**：$S = \frac{1}{2}bc\sin A = \frac{1}{2}ac\sin B = \frac{1}{2}ab\sin C$
- **海伦公式**：$S = \sqrt{s(s-a)(s-b)(s-c)}$（$s = \frac{a+b+c}{2}$）
- **外接圆半径公式**：$S = \frac{abc}{4R}$
- **内切圆半径公式**：$S = rs$（$r$为内切圆半径）

**解题思路：**
1. 已知两边及夹角求面积
2. 已知三边求面积
3. 利用面积法解决其他问题

**典型题型：**
- 已知两边及夹角求面积
- 已知三边求面积
- 利用面积法求高、中线、角平分线
- 面积比问题

**二级结论：**
- $S = \frac{1}{2}ab\sin C = \frac{1}{2}ac\sin B = \frac{1}{2}bc\sin A$
- $S = \frac{a^2\sin B\sin C}{2\sin A} = \frac{b^2\sin A\sin C}{2\sin B} = \frac{c^2\sin A\sin B}{2\sin C}$
- $S = \frac{1}{2}R^2(\sin 2A + \sin 2B + \sin 2C)$

### 1.4 射影定理

**核心方法：**
- **公式**：$a = b\cos C + c\cos B$
- **推广形式**：
  - $b = a\cos C + c\cos A$
  - $c = a\cos B + b\cos A$

**解题思路：**
1. 将边转化为角的余弦线段和
2. 处理三角形中的线段和问题

**典型题型：**
- 三角形中线、高、角平分线问题
- 三角形中的线段和式证明

**二级结论：**
- $a^2 = b^2 + c^2 - 2bc\cos A$（余弦定理的射影形式）
- $S = \frac{1}{2}(b\cos C + c\cos B)\sin A = \frac{1}{2}(a\cos B + b\cos A)\sin C$

## 模块2：代数问题篇

### 2.1 边角互化

**核心方法：**
- 利用正弦定理和余弦定理进行边角转换
- $\sin A = \frac{a}{2R}$, $\cos A = \frac{b^2 + c^2 - a^2}{2bc}$
- $\tan A = \frac{a\sin B}{b - a\cos B}$

**解题思路：**
1. 已知边长条件，转化为角的条件
2. 已知角的条件，转化为边长的条件
3. 结合其他定理综合求解

**典型题型：**
- 边角混合条件的三角形问题
- 利用边角互化证明三角恒等式

**二级结论：**
- $\sin A + \sin B + \sin C = 4\cos\frac{A}{2}\cos\frac{B}{2}\cos\frac{C}{2}$
- $\cos A + \cos B + \cos C = 1 + 4\sin\frac{A}{2}\sin\frac{B}{2}\sin\frac{C}{2}$

### 2.2 最值问题

**核心方法：**
- 利用正弦定理：$a = 2R\sin A$，将边的问题转化为角的三角函数问题
- 利用余弦定理：$a^2 = b^2 + c^2 - 2bc\cos A$，将边的问题转化为角的问题
- 利用面积公式：$S = \frac{1}{2}bc\sin A$

**解题思路：**
1. 确定变量范围（$0 < A, B, C < \pi$，$A+B+C=\pi$）
2. 将目标函数转化为三角函数
3. 利用三角函数的性质求最值

**典型题型：**
- 三角形边长的最大值/最小值
- 三角形面积的最大值/最小值
- 三角形周长、内切圆半径等的最值问题

**二级结论：**
- 在三角形中，$\sin A + \sin B + \sin C \leq \frac{3\sqrt{3}}{2}$（等号当且仅当$A=B=C=60^\circ$时取到）
- $\sin A\sin B\sin C \leq \frac{3\sqrt{3}}{8}$（等号当且仅当$A=B=C=60^\circ$时取到）
- $\cos A\cos B\cos C \leq \frac{1}{8}$（等号当且仅当$A=B=C=60^\circ$时取到，钝角时为负）

### 2.3 范围问题

**核心方法：**
- 利用正弦定理：$a = 2R\sin A$，将边的问题转化为角的三角函数问题
- 利用余弦定理：$a^2 = b^2 + c^2 - 2bc\cos A$，将边的问题转化为角的问题
- 利用重要不等式和三角恒等式

**解题思路：**
1. 确定变量范围（$0 < A, B, C < \pi$，$A+B+C=\pi$）
2. 将目标函数转化为三角函数
3. 利用三角函数的性质和不等式求范围

**典型题型：**
- 三角形边长的取值范围
- 三角形角度的取值范围
- 三角形中线、高、角平分线的范围

**二级结论：**
- 若$A > B$，则$a > b$
- $\sin A + \sin B + \sin C \in (\frac{3\sqrt{3}}{2}, 3]$
- $\cos A + \cos B + \cos C \in (1, \frac{3}{2}]$
- $\tan A + \tan B + \tan C = \tan A\tan B\tan C$

### 2.4 齐次式处理

**核心方法：**
- 利用正弦定理：$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$
- 将齐次式中的边转化为角的正弦函数
- 将齐次式中的角的正弦函数转化为边

**解题思路：**
1. 识别齐次式的形式
2. 选择合适的转化方式（边→角或角→边）
3. 化简并求解

**典型题型：**
- 齐次式的求值/证明
- 三角形中的齐次不等式
- 利用齐次式化简复杂的表达式

**二级结论：**
- $a^2\sin B + b^2\sin A = 2ab\sin A\sin B = 4R\sin A\sin B\sin C$
- $\frac{a^2\sin A + b^2\sin B + c^2\sin C}{\sin A + \sin B + \sin C} = \frac{abc}{4R} = S$（面积）

## 模块3：几何问题篇

### 3.1 中线问题

**核心方法：**
- **中线长度公式**：$m_a^2 = \frac{2b^2 + 2c^2 - a^2}{4}$
- **中线交点性质**：中线交点为重心
- **重心坐标**：重心到顶点的距离是中线长度的$\frac{2}{3}$

**解题思路：**
1. 利用中线公式求中线长度
2. 利用重心性质解决三角形中的点的问题
3. 利用中线将三角形分成两个小三角形

**典型题型：**
- 已知三边求中线长度
- 利用中线将复杂三角形问题分解
- 重心性质的应用

**二级结论：**
- $m_a^2 + m_b^2 + m_c^2 = \frac{3}{4}(a^2 + b^2 + c^2)$
- $m_a^2 = b^2 + c^2 - 2bc\cos\frac{A}{2}\cos\frac{A}{2}$
- $S_{\triangle ABC} = 3S_{\triangle ABD}$（$D$为$BC$中点）

### 3.2 角平分线问题

**核心方法：**
- **角平分线长度公式**：$t_a^2 = bc\left[1 - \frac{a^2}{(b+c)^2}\right] = \frac{2bc\cos\frac{A}{2}}{b+c}$
- **角平分线性质**：角平分线分对边成比例，$\frac{BD}{DC} = \frac{AB}{AC} = \frac{c}{b}$

**解题思路：**
1. 利用角平分线长度公式求角平分线长度
2. 利用角平分线性质处理比例问题
3. 利用角平分线将三角形分成两个小三角形

**典型题型：**
- 已知三边求角平分线长度
- 利用角平分线性质证明比例关系
- 角平分线在三角形中的应用

**二级结论：**
- $t_a = \frac{2bc\cos\frac{A}{2}}{b+c}$
- $t_a^2 = bc - BD \cdot DC$
- $t_a^2 + t_b^2 + t_c^2 = \frac{1}{2}(a^2 + b^2 + c^2) - \frac{1}{4}(a^2 + b^2 + c^2) = \frac{1}{4}(a^2 + b^2 + c^2)$

### 3.3 重心/垂心/外心

**核心方法：**
- **重心（$G$）**：三条中线的交点，$G$到顶点的距离是中线长度的$\frac{2}{3}$
- **垂心（$H$）**：三条高的交点
- **外心（$O$）**：三条垂直平分线的交点，到三个顶点的距离相等（$OA=OB=OC=R$）

**解题思路：**
1. 利用重心性质解决三角形中的点的问题
2. 利用垂心性质解决高的问题
3. 利用外心性质解决外接圆问题

**典型题型：**
- 重心、垂心、外心的性质应用
- 三角形中的四心问题
- 四心之间的距离和关系

**二级结论：**
- $OH^2 = 9R^2 - (a^2 + b^2 + c^2)$
- $OG^2 = R^2 - \frac{1}{9}(a^2 + b^2 + c^2)$
- $OG^2 + GH^2 = OH^2$
- $R^2 = \frac{1}{9}(a^2 + b^2 + c^2) + 3r^2$（$r$为内切圆半径）

### 3.4 多三角形组合

**核心方法：**
- 利用正弦定理、余弦定理处理多个三角形
- 利用面积公式计算多个三角形的面积和
- 利用相似三角形性质

**解题思路：**
1. 将复杂图形分解为多个简单三角形
2. 利用正弦定理和余弦定理建立关系
3. 利用面积公式计算面积
4. 综合求解

**典型题型：**
- 三角形与特殊四边形（平行四边形、矩形、菱形、正方形）
- 三角形与特殊多边形（正多边形）
- 三角形与圆（内切圆、外接圆、切线）

**二级结论：**
- $S_{\triangle ABC} = S_{\triangle ABD} + S_{\triangle ACD}$
- $S_{\triangle ABC} = S_{\triangle ABD} + S_{\triangle BCD} + S_{\triangle CAD}$（$D$为三角形内一点）
- $\frac{S_{\triangle ABD}}{S_{\triangle ACD}} = \frac{BD}{DC} = \frac{AB}{AC}$（$AD$为角平分线）
- $\frac{S_{\triangle ABD}}{S_{\triangle BCD}} = \frac{AB}{BC}$（$AD$为高）
