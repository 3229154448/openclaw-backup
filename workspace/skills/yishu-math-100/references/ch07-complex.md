# 第7章 复数

## 复数的概念与几何意义

### 1.1 复数的概念

**核心方法：**
- **定义**：形如$z = a + bi$（$a, b \in \mathbb{R}$，$i^2 = -1$）的数称为复数
- **实部**：$a = \text{Re}(z)$
- **虚部**：$b = \text{Im}(z)$
- **纯虚数**：$a = 0, b \neq 0$，即$z = bi$
- **零复数**：$a = b = 0$，即$z = 0$

**解题思路：**
1. 识别复数的实部和虚部
2. 判断复数的类型（实数、纯虚数、零）
3. 利用复数的概念进行化简和计算

**典型题型：**
- 复数的分类
- 复数的表示
- 复数的化简

**二级结论：**
- $a + bi = c + di \iff a = c$且$b = d$
- $z = 0 \iff a = 0$且$b = 0$
- $z$为实数 $\iff b = 0$
- $z$为纯虚数 $\iff a = 0$且$b \neq 0$

### 1.2 复数的几何意义

**核心方法：**
- **复平面**：横轴为实轴，纵轴为虚轴
- **点表示**：$z = a + bi$对应复平面上的点$P(a, b)$
- **向量表示**：$z = a + bi$对应向量$\vec{OP}$（$O$为原点）
- **模**：$|z| = \sqrt{a^2 + b^2}$（$z$到原点的距离）
- **辐角**：$\arg z = \theta$（$z$与正实轴的夹角）

**解题思路：**
1. 将复数转化为复平面上的点或向量
2. 利用复平面的几何意义解决复数问题
3. 利用模和辐角进行计算

**典型题型：**
- 复数与复平面的对应关系
- 复数的模的计算
- 复数的辐角的计算
- 复数的几何意义应用

**二级结论：**
- $|z| = \sqrt{a^2 + b^2}$
- $|z| = 0 \iff z = 0$
- $|z|^2 = z \cdot \overline{z}$（$\overline{z}$为$z$的共轭复数）
- $|z_1 \cdot z_2| = |z_1| \cdot |z_2|$
- $|\frac{z_1}{z_2}| = \frac{|z_1|}{|z_2|}$（$z_2 \neq 0$）
- $|z^n| = |z|^n$
- $|z_1 \pm z_2| \leq |z_1| + |z_2|$（三角不等式）

### 1.3 共轭复数

**核心方法：**
- **定义**：$z = a + bi$，则$\overline{z} = a - bi$为$z$的共轭复数
- **性质**：
  - $\overline{\overline{z}} = z$
  - $\overline{z_1 + z_2} = \overline{z_1} + \overline{z_2}$
  - $\overline{z_1 \cdot z_2} = \overline{z_1} \cdot \overline{z_2}$
  - $\overline{(\frac{z_1}{z_2})} = \frac{\overline{z_1}}{\overline{z_2}}$（$z_2 \neq 0$）
  - $z \cdot \overline{z} = |z|^2 = a^2 + b^2$
  - $z + \overline{z} = 2a = 2\text{Re}(z)$
  - $z - \overline{z} = 2bi = 2i\text{Im}(z)$
  - $z = \overline{z} \iff z \in \mathbb{R}$

**解题思路：**
1. 利用共轭复数的性质进行化简
2. 利用共轭复数求解方程
3. 利用共轭复数解决复数问题

**典型题型：**
- 共轭复数的性质应用
- 利用共轭复数化简复数
- 利用共轭复数求解方程

**二级结论：**
- $\overline{z_1 \pm z_2} = \overline{z_1} \pm \overline{z_2}$
- $\overline{z_1 \cdot z_2} = \overline{z_1} \cdot \overline{z_2}$
- $\overline{z_1 / z_2} = \overline{z_1} / \overline{z_2}$
- $z \cdot \overline{z} = |z|^2 = a^2 + b^2$
- $z + \overline{z} = 2a$
- $z - \overline{z} = 2bi$
- $z = \overline{z} \iff z \in \mathbb{R}$
- $z \in \mathbb{R} \iff \overline{z} = z$

### 1.4 复平面与向量

**核心方法：**
- **对应关系**：复数$z = a + bi$对应复平面上的点$P(a, b)$和向量$\vec{OP}$
- **模的几何意义**：$|z| = |\vec{OP}|$（$z$到原点的距离）
- **复数加减的几何意义**：$\vec{OP_1} + \vec{OP_2} = \vec{OP_3}$（平行四边形法则）
- **复数乘法的几何意义**：$z_1 \cdot z_2$的模为$|z_1| \cdot |z_2|$，辐角为$\arg z_1 + \arg z_2$

**解题思路：**
1. 将复数转化为复平面上的点或向量
2. 利用复平面的几何意义解决复数问题
3. 利用模和辐角进行计算

**典型题型：**
- 复数与复平面的对应关系
- 复数的模的计算
- 复数的辐角的计算
- 复数的几何意义应用

**二级结论：**
- $z_1 + z_2 = 0 \iff z_2 = -z_1$
- $|z_1 + z_2|^2 = |z_1|^2 + |z_2|^2 + 2\text{Re}(z_1 \cdot \overline{z_2})$
- $|z_1 - z_2|^2 = |z_1|^2 + |z_2|^2 - 2\text{Re}(z_1 \cdot \overline{z_2})$
- $|z_1 + z_2| = |z_1 - z_2| \iff z_1 \perp z_2$（$z_1 \cdot \overline{z_2} + \overline{z_1} \cdot z_2 = 0$）

## 复数的运算

### 2.1 复数的加减法

**核心方法：**
- **代数形式**：$(a + bi) \pm (c + di) = (a \pm c) + (b \pm d)i$
- **几何意义**：平行四边形法则或三角形法则
- **坐标运算**：$(x_1, y_1) \pm (x_2, y_2) = (x_1 \pm x_2, y_1 \pm y_2)$

**解题思路：**
1. 利用代数形式进行加减运算
2. 利用几何意义理解加减法的含义
3. 利用坐标运算进行计算

**典型题型：**
- 复数加减法的计算
- 复数加减法的几何意义
- 复数加减法的应用

**二级结论：**
- $(a + bi) + (c + di) = (a + c) + (b + d)i$
- $(a + bi) - (c + di) = (a - c) + (b - d)i$
- $z_1 + z_2 = 0 \iff z_2 = -z_1$
- $|z_1 + z_2| \leq |z_1| + |z_2|$（三角不等式）

### 2.2 复数的乘法

**核心方法：**
- **代数形式**：$(a + bi)(c + di) = (ac - bd) + (ad + bc)i$
- **几何意义**：$|z_1 \cdot z_2| = |z_1| \cdot |z_2|$，$\arg(z_1 \cdot z_2) = \arg z_1 + \arg z_2$
- **共轭复数性质**：$(a + bi)(a - bi) = a^2 + b^2 = |z|^2$

**解题思路：**
1. 利用代数形式进行乘法运算
2. 利用几何意义理解乘法的含义
3. 利用共轭复数进行化简

**典型题型：**
- 复数乘法的计算
- 复数乘法的几何意义
- 利用共轭复数化简

**二级结论：**
- $(a + bi)(c + di) = (ac - bd) + (ad + bc)i$
- $|z_1 \cdot z_2| = |z_1| \cdot |z_2|$
- $\arg(z_1 \cdot z_2) = \arg z_1 + \arg z_2$
- $(a + bi)(a - bi) = a^2 + b^2 = |z|^2$
- $(z_1 \cdot z_2) \cdot \overline{z_1 \cdot z_2} = |z_1 \cdot z_2|^2 = |z_1|^2 \cdot |z_2|^2$

### 2.3 复数的除法

**核心方法：**
- **代数形式**：$\frac{a + bi}{c + di} = \frac{(a + bi)(c - di)}{(c + di)(c - di)} = \frac{ac + bd}{c^2 + d^2} + \frac{bc - ad}{c^2 + d^2}i$
- **分母实化**：分子分母同乘分母的共轭复数
- **几何意义**：$\frac{z_1}{z_2}$的模为$\frac{|z_1|}{|z_2|}$，辐角为$\arg z_1 - \arg z_2$

**解题思路：**
1. 利用分母实化进行除法运算
2. 利用几何意义理解除法的含义
3. 利用共轭复数进行化简

**典型题型：**
- 复数除法的计算
- 复数除法的几何意义
- 利用分母实化化简

**二级结论：**
- $\frac{a + bi}{c + di} = \frac{ac + bd}{c^2 + d^2} + \frac{bc - ad}{c^2 + d^2}i$
- $|\frac{z_1}{z_2}| = \frac{|z_1|}{|z_2|}$（$z_2 \neq 0$）
- $\arg(\frac{z_1}{z_2}) = \arg z_1 - \arg z_2$
- $\frac{1}{z} = \frac{\overline{z}}{|z|^2}$（$z \neq 0$）

### 2.4 i的幂次周期性

**核心方法：**
- **i的定义**：$i^2 = -1$
- **周期性**：$i^{4n} = 1, i^{4n+1} = i, i^{4n+2} = -1, i^{4n+3} = -i$（$n \in \mathbb{Z}$）
- **通式**：$i^n = i^{n \pmod 4}$

**解题思路：**
1. 利用$i$的幂次周期性化简
2. 利用通式进行计算
3. 利用周期性解决复数问题

**典型题型：**
- $i$的幂次的计算
- 复数的幂次的计算
- 利用周期性化简

**二级结论：**
- $i^0 = 1, i^1 = i, i^2 = -1, i^3 = -i$
- $i^{4n} = 1, i^{4n+1} = i, i^{4n+2} = -1, i^{4n+3} = -i$
- $i^{n+4} = i^n$
- $i^n + i^{n+1} + i^{n+2} + i^{n+3} = 0$
- $i^n \cdot i^{n+1} = i^{2n+1} = i \cdot i^{2n} = i \cdot (-1)^n$

## 复数常用技巧

### 3.1 分母实化

**核心方法：**
- **定义**：将分母中的虚数转化为实数
- **方法**：分子分母同乘分母的共轭复数
- **公式**：$\frac{a + bi}{c + di} = \frac{(a + bi)(c - di)}{c^2 + d^2} = \frac{ac + bd}{c^2 + d^2} + \frac{bc - ad}{c^2 + d^2}i$

**解题思路：**
1. 识别分母中的虚数
2. 分子分母同乘分母的共轭复数
3. 化简并得到结果

**典型题型：**
- 分母实化的计算
- 复数除法的分母实化
- 复数化简的分母实化

**二级结论：**
- $\frac{a + bi}{c + di} = \frac{ac + bd}{c^2 + d^2} + \frac{bc - ad}{c^2 + d^2}i$
- $\frac{1}{c + di} = \frac{c - di}{c^2 + d^2}$
- $\frac{a + bi}{c - di} = \frac{(a + bi)(c + di)}{c^2 + d^2} = \frac{ac - bd}{c^2 + d^2} + \frac{bc + ad}{c^2 + d^2}i$

### 3.2 设复数技巧

**核心方法：**
- **代数形式**：$z = a + bi$
- **三角形式**：$z = r(\cos\theta + i\sin\theta)$
- **几何形式**：$z = x + yi$对应点$(x, y)$
- **共轭复数**：$\overline{z} = a - bi$

**解题思路：**
1. 根据题目条件选择合适的复数形式
2. 利用复数的性质进行化简
3. 利用复数的几何意义解决几何问题

**典型题型：**
- 设复数求解方程
- 设复数解决几何问题
- 设复数化简表达式

**二级结论：**
- $z = a + bi$，则$\overline{z} = a - bi$
- $z \cdot \overline{z} = |z|^2 = a^2 + b^2$
- $z + \overline{z} = 2a = 2\text{Re}(z)$
- $z - \overline{z} = 2bi = 2i\text{Im}(z)$
- $z = \overline{z} \iff z \in \mathbb{R}$

### 3.3 模的运算性质

**核心方法：**
- **定义**：$|z| = \sqrt{a^2 + b^2}$
- **性质**：
  - $|z| = 0 \iff z = 0$
  - $|z| \geq 0$
  - $|z_1 \cdot z_2| = |z_1| \cdot |z_2|$
  - $|\frac{z_1}{z_2}| = \frac{|z_1|}{|z_2|}$（$z_2 \neq 0$）
  - $|z^n| = |z|^n$
  - $|z_1 \pm z_2| \leq |z_1| + |z_2|$（三角不等式）
  - $||z_1| - |z_2|| \leq |z_1 \pm z_2| \leq |z_1| + |z_2|$

**解题思路：**
1. 利用模的性质进行计算
2. 利用模的性质化简表达式
3. 利用模的性质解决几何问题

**典型题型：**
- 复数模的计算
- 利用模的性质化简
- 利用模的性质解决几何问题

**二级结论：**
- $|z| = \sqrt{a^2 + b^2}$
- $|z| = 0 \iff z = 0$
- $|z_1 \cdot z_2| = |z_1| \cdot |z_2|$
- $|\frac{z_1}{z_2}| = \frac{|z_1|}{|z_2|}$（$z_2 \neq 0$）
- $|z^n| = |z|^n$
- $|z_1 + z_2| \leq |z_1| + |z_2|$
- $|z_1 - z_2| \geq ||z_1| - |z_2||$
- $|z_1 + z_2|^2 = |z_1|^2 + |z_2|^2 + 2\text{Re}(z_1 \cdot \overline{z_2})$
- $|z_1 - z_2|^2 = |z_1|^2 + |z_2|^2 - 2\text{Re}(z_1 \cdot \overline{z_2})$
