# 第10章 有机化学

## 模块一 有机物分类与命名

### 第1讲 有机物的分类

**按碳骨架分类**
- 链状化合物（脂肪族）
- 环状化合物：脂环族、芳香族（含苯环）

**按官能团分类**
| 官能团 | 名称 | 类别 | 代表物 |
|--------|------|------|--------|
| C=C | 碳碳双键 | 烯烃 | CH₂=CH₂ |
| C≡C | 碳碳三键 | 炔烃 | CH≡CH |
| -X(F,Cl,Br,I) | 卤素原子 | 卤代烃 | CH₃Cl |
| -OH | 羟基 | 醇/酚 | CH₃OH / C₆H₅OH |
| -CHO | 醛基 | 醛 | CH₃CHO |
| -CO- | 羰基 | 酮 | CH₃COCH₃ |
| -COOH | 羧基 | 羧酸 | CH₃COOH |
| -COO- | 酯基 | 酯 | CH₃COOCH₃ |
| -NH₂ | 氨基 | 胺 | CH₃NH₂ |
| -CONH- | 酰胺键 | 酰胺 | CH₃CONH₂ |
| -NO₂ | 硝基 | 硝基化合物 | C₆H₅NO₂ |

### 第2讲 有机物系统命名法

**烷烃命名步骤**
1. 选最长碳链为主链，确定母体名称
2. 从距支链最近的一端编号
3. 将取代基的位次、数目、名称写在母体前面
4. 不同取代基按简单→复杂排列

**烯烃/炔烃命名**
- 选含双键/三键的最长碳链为主链
- 从距双键/三键最近的一端编号
- 标明双键/三键的位置

**含官能团有机物命名**
- 选含官能团的最长碳链为主链
- 编号使官能团位次最小
- 官能团位次写在母体名称前面

**二级结论**
- 系统命名中取代基位次用阿拉伯数字，多个取代基用逗号隔开
- 位次与名称之间用短横线"-"
- 相同取代基合并，用二、三、四等汉字标出

### 第3讲 同系物与同分异构体

**同系物**
- 结构相似（同种官能团），分子组成相差一个或若干个CH₂原子团
- 同系物必为同一类物质，通式相同

**同分异构体**
- 分子式相同，结构不同的化合物
- 类型：
  1. **碳链异构**：碳骨架不同（如正丁烷与异丁烷）
  2. **位置异构**：官能团位置不同（如1-丙醇与2-丙醇）
  3. **官能团异构（类别异构）**：官能团不同（如乙醇与甲醚）
  4. **顺反异构**：双键两侧基团排列不同
  5. **对映异构**：手性碳原子存在

**常见官能团异构**
| 分子式 | 异构类型 |
|--------|---------|
| C₂H₆O | 乙醇(CH₃CH₂OH)、甲醚(CH₃OCH₃) |
| C₃H₆O | 丙醛(CH₃CH₂CHO)、丙酮(CH₃COCH₃)、环氧丙烷、烯醇 |
| C₃H₆O₂ | 丙酸(CH₃CH₂COOH)、乙酸甲酯(CH₃COOCH₃)、甲酸乙酯(HCOOC₂H₅)、羟基丙酮 |

**二级结论**
- 同分异构体 → 分子式相同；同系物 → 通式相同但分子式不同
- 同分异构体之间物理性质不同，化学性质可能相似（同类别）或不同（不同类别）
- 书写同分异构体遵循：主链由长到短、支链由整到散、位置由心到边

## 模块二 官能团与性质

### 第4讲 烷烃的性质

**主要反应——取代反应**
- 与X₂（光照条件）：CH₄ + Cl₂ →(光照) CH₃Cl + HCl
- 卤素反应活性：F₂ > Cl₂ > Br₂ > I₂
- 不同H被取代的活性：3°H > 2°H > 1°H

**二级结论**
- 烷烃不能使酸性KMnO₄褪色（不能被氧化）
- 烷烃不能使溴水褪色（不能加成）
- 烷烃的特征反应是取代反应

### 第5讲 烯烃与炔烃的性质

**加成反应**
- 与Br₂：CH₂=CH₂ + Br₂ → CH₂BrCH₂Br（使溴水褪色）
- 与H₂：CH₂=CH₂ + H₂ →(催化剂) CH₃CH₃
- 与HCl：CH₂=CH₂ + HCl → CH₃CH₂Cl
- 与H₂O：CH₂=CH₂ + H₂O →(催化剂) CH₃CH₂OH

**加聚反应**
- nCH₂=CH₂ → [-CH₂-CH₂-]ₙ

**氧化反应**
- 使酸性KMnO₄溶液褪色
- 烯烃的氧化（臭氧分解）

**二级结论**
- 鉴别烷烃和烯烃：用溴水或酸性KMnO₄溶液
- 马氏规则：不对称烯烃加成时，H加在H多的碳上（如CH₃CH=CH₂ + HCl → CH₃CHClCH₃）
- 共轭二烯烃可发生1,2-加成和1,4-加成

### 第6讲 苯及芳香烃的性质

**苯的特殊性**
- 苯环含离域大π键，具有特殊的稳定性
- 苯不易发生加成反应，易发生取代反应

**主要反应**
- 取代反应：
  - 卤代（Fe作催化剂）：C₆H₆ + Br₂ →(FeBr₃) C₆H₅Br + HBr
  - 硝化（浓H₂SO₄作催化剂和吸水剂）：C₆H₆ + HNO₃ →(浓H₂SO₄,50-60°C) C₆H₅NO₂ + H₂O
  - 磺化：C₆H₆ + H₂SO₄(浓) → C₆H₅SO₃H + H₂O
- 加成反应：C₆H₆ + 3H₂ →(催化剂) C₆H₁₂（环己烷）

**苯的同系物**
- 苯环+烷基（如甲苯C₆H₅CH₃）
- 苯的同系物能使酸性KMnO₄褪色（侧链被氧化）
- 甲苯硝化生成TNT（三硝基甲苯）

**二级结论**
- 苯不能使酸性KMnO₄褪色（无侧链烷基）
- 甲苯能使酸性KMnO₄褪色（侧链被氧化为-COOH）
- 苯的卤代和硝化属于取代反应
- 苯的同系物通式：CₙH₂ₙ₋₆(n≥6)

### 第7讲 醇与酚的性质

**醇（-OH连在脂肪烃基上）**
- 与Na反应：2CH₃CH₂OH + 2Na → 2CH₃CH₂ONa + H₂↑
- 催化氧化：2CH₃CH₂OH + O₂ →(Cu/Ag,△) 2CH₃CHO + 2H₂O
- 消去反应：CH₃CH₂OH →(浓H₂SO₄,170°C) CH₂=CH₂↑ + H₂O
- 酯化反应：CH₃CH₂OH + CH₃COOH ⇌(浓H₂SO₄,△) CH₃COOCH₂CH₃ + H₂O

**酚（-OH连在苯环上）**
- 弱酸性：C₆H₅OH + NaOH → C₆H₅ONa + H₂O（醇不能与NaOH反应）
- 显色反应：苯酚遇FeCl₃溶液显紫色
- 取代反应：苯酚与浓溴水反应生成2,4,6-三溴苯酚白色沉淀
  - C₆H₅OH + 3Br₂ → C₆H₂Br₃OH↓ + 3HBr

**二级结论**
- 醇的催化氧化条件：-OH所连碳原子上有2个H → 氧化为醛；1个H → 氧化为酮；无H → 不能催化氧化
- 消去反应条件：-OH所连碳原子的邻位碳上有H才能消去
- 苯酚酸性比H₂CO₃弱：向苯酚钠溶液中通CO₂生成苯酚（不管CO₂量多少）
  - C₆H₅ONa + CO₂ + H₂O → C₆H₅OH + NaHCO₃（只生成NaHCO₃，不生成Na₂CO₃）

### 第8讲 醛与羧酸的性质

**醛（-CHO）**
- 银镜反应：CH₃CHO + 2[Ag(NH₃)₂]OH →(水浴加热) CH₃COONH₄ + 2Ag↓ + 3NH₃ + H₂O
- 与新制Cu(OH)₂反应：CH₃CHO + 2Cu(OH)₂ →(△) CH₃COOH + Cu₂O↓ + 2H₂O
- 催化氧化：2CH₃CHO + O₂ →(催化剂) 2CH₃COOH
- 与H₂加成（还原）：CH₃CHO + H₂ →(催化剂) CH₃CH₂OH

**羧酸（-COOH）**
- 酸性：CH₃COOH ⇌ CH₃COO⁻ + H⁺（弱酸）
- 酯化反应：CH₃COOH + CH₃CH₂OH ⇌(浓H₂SO₄,△) CH₃COOCH₂CH₃ + H₂O

**二级结论**
- 检验醛基：银镜反应或新制Cu(OH)₂悬浊液
- 醛既有氧化性（可被还原为醇）又有还原性（可被氧化为酸）
- 酯化反应机理：酸脱羟基醇脱氢（同位素示踪法验证）
- 酯化反应是可逆反应，浓H₂SO₄作催化剂和吸水剂

### 第9讲 酯的性质

**水解反应**
- 酸性水解（可逆）：CH₃COOCH₂CH₃ + H₂O ⇌(稀H₂SO₄) CH₃COOH + CH₃CH₂OH
- 碱性水解（不可逆，皂化反应）：CH₃COOCH₂CH₃ + NaOH → CH₃COONa + CH₃CH₂OH

**二级结论**
- 酯在碱性条件下水解比酸性条件下更彻底
- 油脂是高级脂肪酸甘油酯，碱性水解叫皂化反应（制肥皂原理）

## 模块三 有机合成与推断

### 第10讲 官能团转化路线

**常见转化关系**
- 醇 → 醛 → 酸 → 酯
  - RCH₂OH →(催化氧化) RCHO →(催化氧化) RCOOH →(酯化) RCOOR'
- 烯烃 → 醇 → 醛/酮
  - RCH=CH₂ →(加成H₂O) RCH(OH)CH₃
- 卤代烃 → 醇/烯烃
  - RCH₂CH₂X →(NaOH水溶液,△) RCH₂CH₂OH
  - RCH₂CH₂X →(NaOH醇溶液,△) RCH=CH₂

**二级结论**
- 卤代烃在NaOH水溶液中发生取代（水解）生成醇
- 卤代烃在NaOH醇溶液中发生消去生成烯烃
- 醇→醛：催化氧化（去氢）；醛→醇：催化加氢（加氢）
- 引入-OH的方法：烯烃加水、醛/酮加H₂、卤代烃水解、酯水解

### 第11讲 有机合成路线设计

**合成原则**
1. 原料价廉、易得
2. 路线简短、产率高
3. 条件温和、操作安全
4. 绿色化学原则

**常见合成策略**
- 正推法：从原料出发，逐步转化到目标产物
- 逆推法：从目标产物出发，逆推到简单原料（更常用）

### 第12讲 有机推断题解法

**推断线索**
1. **分子式线索**：不饱和度 = (2n_C + 2 - n_H) / 2
   - Ω=1：含1个双键或1个环
   - Ω=2：含2个双键或1个三键或1个双键+1个环
   - Ω=4：可能含苯环
2. **反应条件线索**：
   - NaOH醇溶液/△ → 消去反应
   - NaOH水溶液/△ → 水解反应
   - 浓H₂SO₄/△ → 酯化或消去
   - H₂/催化剂 → 加成（还原）
   - O₂/Cu或Ag/△ → 醇的催化氧化
   - [Ag(NH₃)₂]OH/水浴 → 银镜反应（含-CHO）
   - Cu(OH)₂悬浊液/△ → 含-CHO
3. **特殊性质线索**：
   - 遇FeCl₃显紫色 → 酚
   - 使溴水褪色 → 含C=C或-CHO
   - 使酸性KMnO₄褪色 → 含C=C或-CHO或苯的同系物侧链

## 模块四 高分子化合物

### 第13讲 加聚反应与缩聚反应

**加聚反应**
- 含C=C的单体加成聚合
- nCH₂=CH₂ → [-CH₂-CH₂-]ₙ
- 单体判断：在链节中画"断键"，恢复C=C

**缩聚反应**
- 含两种官能团的单体通过缩合聚合
- 如：nHOOC-COOH + nHO-CH₂-CH₂-OH → [-OC-COO-CH₂-CH₂-O-]ₙ + 2nH₂O
- 常见缩聚：聚酯、聚酰胺（尼龙）、酚醛树脂

**二级结论**
- 加聚产物链节与单体组成相同
- 缩聚产物链节比单体少了小分子（H₂O等）的部分
- 由高分子判断单体：
  - 加聚：在链节中两碳间"切"开恢复双键
  - 缩聚：在链节中酯键(-COO-)或酰胺键(-CONH-)处"切"开，分别加-OH和-H
