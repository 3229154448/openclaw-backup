# 第3章 离子反应与离子方程式

## 模块一 电解质与非电解质

### 第1讲 电解质与非电解质的判断

**核心概念**
- **电解质**：在水溶液中或熔融状态下能导电的化合物
- **非电解质**：在水溶液中和熔融状态下均不能导电的化合物
- **强电解质**：在水溶液中能完全电离的电解质
- **弱电解质**：在水溶液中只能部分电离的电解质

**分类记忆**
| 类别 | 举例 |
|------|------|
| 强电解质 | 强酸（HCl、H₂SO₄、HNO₃）、强碱（NaOH、KOH、Ca(OH)₂）、大多数盐（NaCl、BaSO₄） |
| 弱电解质 | 弱酸（CH₃COOH、H₂CO₃、H₂SO₃、HF）、弱碱（NH₃·H₂O、Fe(OH)₃）、H₂O |
| 非电解质 | 蔗糖、酒精、葡萄糖、SO₂（溶于水导电是因为与水反应生成H₂SO₃） |

**二级结论**
- 电解质/非电解质都是**化合物**，单质和混合物既不是电解质也不是非电解质
- 能导电的不一定是电解质（如Cu、石墨是单质；NaCl溶液是混合物）
- BaSO₄难溶但为强电解质（溶解的部分完全电离）
- NH₃是纯非电解质，溶于水后生成NH₃·H₂O（弱电解质）
- SO₂、SO₃、CO₂溶于水导电是因为与水反应生成了酸，它们本身是非电解质

## 模块二 离子方程式

### 第2讲 离子方程式书写规则

**拆分规则**
1. **易溶且易电离的物质**写成离子形式：强酸、强碱、可溶性盐
2. **难电离、难溶、气体、单质、氧化物**保留化学式
3. **删去**方程式两边相同的离子（旁观离子）
4. 检查原子守恒和电荷守恒

**特殊拆分规则**
| 物质 | 处理方式 |
|------|----------|
| 强酸（HCl、H₂SO₄、HNO₃） | 拆开 |
| 强碱（NaOH、KOH、Ba(OH)₂） | 拆开 |
| 可溶性盐（NaCl、KNO₃等） | 拆开 |
| 弱酸（CH₃COOH、H₂CO₃等） | 不拆 |
| 弱碱（NH₃·H₂O等） | 不拆 |
| 难溶物（BaSO₄、AgCl等） | 不拆 |
| 气体（CO₂、SO₂等） | 不拆 |
| 单质、氧化物 | 不拆 |
| 浓H₂SO₄ | 不拆（浓硫酸中H⁺难以电离） |
| H₂O | 不拆 |

### 第3讲 离子方程式正误判断

**判断要点（按序检查）**
1. **是否符合客观事实**：如铁与稀盐酸生成Fe²⁺而非Fe³⁺
2. **是否拆分正确**：弱电解质、难溶物、气体不能拆
3. **是否守恒**：原子守恒 + 电荷守恒
4. **是否漏写反应**：如过量CO₂通入NaOH溶液应生成NaHCO₃而非Na₂CO₃
5. **量多少问题**：少量物质按化学计量数参与反应

**常见错误**
- Fe + 2H⁺ → Fe²⁺ + H₂↑（正确），而非 Fe + 3H⁺ → Fe³⁺ + 3/2H₂↑
- Cu 与稀HNO₃：3Cu + 2NO₃⁻ + 8H⁺ → 3Cu²⁺ + 2NO↑ + 4H₂O

### 第4讲 量多少问题（少量/过量问题）

**方法——以少量为准**
1. 少量物质的所有离子全部参与反应
2. 根据少量物质的化学式确定比例关系
3. 过量物质提供反应所需的离子，剩余部分保留

**经典题型**
- **少量CO₂通入NaOH溶液**：CO₂ + 2OH⁻ → CO₃²⁻ + H₂O
- **过量CO₂通入NaOH溶液**：CO₂ + OH⁻ → HCO₃⁻
- **少量NaOH滴入AlCl₃溶液**：3OH⁻ + Al³⁺ → Al(OH)₃↓
- **过量NaOH滴入AlCl₃溶液**：4OH⁻ + Al³⁺ → AlO₂⁻ + 2H₂O
- **少量NaOH滴入(NH₄)₂SO₄溶液**：OH⁻ + NH₄⁺ → NH₃·H₂O
- **过量NaOH滴入(NH₄)₂SO₄溶液**：OH⁻ + NH₄⁺ → NH₃·H₂O（同上，因为1:1反应）

**二级结论**
- 酸式盐与碱的反应：以碱为少量时，碱全部参与中和
- 向Na₂CO₃溶液中逐滴加稀盐酸：先 CO₃²⁻ + H⁺ → HCO₃⁻，后 HCO₃⁻ + H⁺ → CO₂↑ + H₂O

## 模块三 离子共存

### 第5讲 离子共存判断

**判断原则：在给定条件下不能发生反应的离子能共存**

**四种类型**

**1. 复分解型——不能共存**
- 生成沉淀：Ag⁺ + Cl⁻、Ba²⁺ + SO₄²⁻、Ca²⁺ + CO₃²⁻等
- 生成气体：H⁺ + CO₃²⁻、H⁺ + SO₃²⁻、H⁺ + S²⁻等
- 生成弱电解质：H⁺ + OH⁻、H⁺ + CH₃COO⁻、NH₄⁺ + OH⁻等

**2. 氧化还原型——不能共存**
- Fe³⁺与I⁻、S²⁻、SO₃²⁻
- MnO₄⁻(H⁺)与Cl⁻、Fe²⁺、SO₃²⁻、I⁻
- NO₃⁻(H⁺)与Fe²⁺、I⁻、S²⁻、SO₃²⁻
- ClO⁻与I⁻、S²⁻、Fe²⁺、SO₃²⁻

**3. 络合型——不能共存**
- Fe³⁺ + SCN⁻ → [Fe(SCN)]²⁺
- Ag⁺ + NH₃·H₂O → [Ag(NH₃)₂]⁺

**4. 双水解型——不能共存**
- Al³⁺与AlO₂⁻、CO₃²⁻、HCO₃⁻、S²⁻、HS⁻
- Fe³⁺与CO₃²⁻、HCO₃⁻、AlO₂⁻
- NH₄⁺与AlO₂⁻、SiO₃²⁻

**隐含条件分析**
- **无色溶液**：排除有色离子：Fe²⁺（浅绿）、Fe³⁺（黄色）、Cu²⁺（蓝色）、MnO₄⁻（紫色）
- **酸性溶液（pH=1或加石蕊变红）**：排除与H⁺反应的离子：OH⁻、CO₃²⁻、HCO₃⁻、SO₃²⁻、S²⁻等
- **碱性溶液（加酚酞变红）**：排除与OH⁻反应的离子：NH₄⁺、Al³⁺、Fe³⁺、Fe²⁺、Cu²⁺、Mg²⁺、HCO₃⁻等
- **水电离出的c(H⁺)=10⁻¹³ mol/L**：可能为酸溶液或碱溶液，两种都要考虑

**隐蔽反应**
- NO₃⁻在酸性条件下有强氧化性
- ClO⁻无论酸性碱性均有氧化性
- MnO₄⁻无论酸性碱性均有氧化性
