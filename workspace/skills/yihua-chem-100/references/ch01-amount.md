# 第1章 物质的量与化学计算

## 模块一 物质的量

### 第1讲 阿伏加德罗常数

**核心概念**
- 物质的量（n）：衡量微观粒子集合体的物理量，单位为摩尔（mol）
- 阿伏加德罗常数（N_A）：1 mol 任何粒子的粒子数，N_A ≈ 6.02 × 10²³ mol⁻¹
- 粒子数（N）与物质的量的关系：**N = n × N_A**

**解题方法——阿伏加德罗常数正误判断**
1. 明确"1 mol"对应的粒子种类（分子、原子、离子、电子等）
2. 检查物质的状态（固/液/气）：**气体需标明标准状况（0°C, 101 kPa）**
3. 检查特殊物质的结构：SiO₂（不含分子）、P₄（含6条共价键）、CH₄（含4条C-H键）
4. 检查可逆反应：合成氨、SO₃分解等，转化率不为100%

**二级结论**
- 标准状况下，1 mol 任何气体体积均约为22.4 L
- 非标准状况下，1 mol 气体体积不一定是22.4 L
- 标准状况下，SO₃为固体，水为液体，CCl₄为液体——不能当作气体用22.4 L
- 1 mol Fe 与足量盐酸反应转移2 mol e⁻，与足量氯气反应转移3 mol e⁻

**典型题型**
- 判断一定量物质中含有粒子数是否正确
- 给出具体条件（温度、压强），求气体体积与物质的量的关系

### 第2讲 摩尔质量与气体摩尔体积

**核心概念**
- 摩尔质量（M）：单位物质的量的物质所具有的质量，单位 g/mol
- M 与相对分子质量数值相等，单位不同
- 气体摩尔体积（V_m）：标准状况下 V_m ≈ 22.4 L/mol

**核心公式**
- **m = n × M**
- **V = n × V_m**（仅适用于标准状况下的气体）

**解题方法——气体体积计算**
1. 确认是否为标准状况
2. 确认物质是否为气态
3. 利用公式 V = n × V_m 或 n = V / V_m 计算

**二级结论**
- 同温同压下，同体积的任何气体含有相同数目的分子（阿伏加德罗定律）
- 同温同压下，气体密度之比 = 摩尔质量之比 = 相对分子质量之比
- 同温同体积下，气体压强之比 = 物质的量之比

### 第3讲 物质的量浓度

**核心概念**
- 物质的量浓度（c_B）：单位体积溶液中所含溶质B的物质的量
- **c_B = n_B / V**（V为溶液体积，非溶剂体积）

**核心公式**
- **c = n / V = m / (M × V)**
- 质量分数与物质的量浓度换算：**c = (1000ρ × w) / M**
  - ρ：溶液密度（g/mL），w：质量分数

**解题方法——溶液稀释与混合**
1. 稀释定律：**c₁V₁ = c₂V₂**（稀释前后溶质物质的量不变）
2. 混合后溶液体积：除非给出密度，否则不能简单相加
3. 注意"将V mL水加入溶液"≠"溶液变为(V+原体积)mL"

**二级结论**
- 稀释弱酸（如醋酸），c(H⁺)减小但电离度α增大
- 稀释时，溶液pH向7靠近（酸向7增大，碱向7减小）
- 同浓度同体积的盐酸和醋酸，中和碱的能力相同

**典型题型**
- 配制一定物质的量浓度的溶液（步骤与误差分析）
- 溶液稀释、混合后浓度的计算

## 模块二 化学计算技巧

### 第4讲 守恒法

**核心方法**
- **质量守恒**：反应前后各元素质量不变
- **原子守恒**：反应前后各元素原子个数不变
- **电荷守恒**：溶液中所有阳离子所带正电荷总数 = 所有阴离子所带负电荷总数
- **电子守恒**：氧化还原反应中，还原剂失电子总数 = 氧化剂得电子总数

**解题步骤**
1. 识别可用哪种守恒
2. 列出守恒等式
3. 求解未知量

**典型应用**
- 氧化还原反应中求某物质化合价变化 → 用电子守恒
- 电解质溶液中离子浓度大小比较 → 用电荷守恒+物料守恒+质子守恒

### 第5讲 差量法

**核心方法**
利用反应前后某种量的差值（质量差、体积差、物质的量差）与化学计量数成正比的关系。

**公式**
$$\frac{n_A}{\Delta n} = \frac{a}{\Delta a}$$

**解题步骤**
1. 写出化学方程式，标出有关量及差值
2. 列比例式求解

**二级结论**
- 固体质量增加：如 CO + CuO → Cu + CO₂，每1 mol CO参与反应，固体质量减少16 g
- 气体体积变化：如 2Na + 2H₂O → 2NaOH + H₂↑，反应前后气体体积变化可差量计算

### 第6讲 极值法

**核心方法**
将混合物视为全部为某一成分，求出极端值，实际值必介于两个极端值之间。

**适用场景**
- 混合物成分分析
- 反应物是否过量的判断
- 化合价取值范围的确定

**解题步骤**
1. 假设全部为成分A，求极端值1
2. 假设全部为成分B，求极端值2
3. 实际值在两极端值之间，由此判断成分或比例

### 第7讲 平均值法

**核心方法**
混合物的平均值（平均摩尔质量、平均化合价等）必介于各成分对应值之间。

**公式**
$$\bar{M} = M_A \cdot x_A + M_B \cdot x_B$$

**二级结论**
- 混合气体的平均相对分子质量必介于最小组分和最大组分之间
- 若平均值等于某一成分的值，则该混合物不可能只含此成分以外物质

### 第8讲 十字交叉法

**核心方法**
用于两组分混合体系中，已知平均值和各组分值，求组分比例。

**操作步骤**
1. 写出两组分的值 A、B（A > B）
2. 写出平均值 $\bar{X}$
3. 交叉相减：A 对应的比为 $\bar{X} - B$，B 对应的比为 $A - \bar{X}$
4. 比值即为两组分的物质的量之比（或质量比，取决于使用的物理量）

**示例**
已知平均相对分子质量为30的混合气体由CH₄(16)和O₂(32)组成：
- CH₄: 16 → 32-30 = 2
- O₂: 32 → 30-16 = 14
- n(CH₄) : n(O₂) = 2 : 14 = 1 : 7

**注意**
- 十字交叉法所得比值的含义取决于所用的基准量：用摩尔质量则为物质的量比，用密度则为体积比

### 第9讲 关系式法

**核心方法**
对于多步连续反应，通过化学方程式找出起始反应物与最终产物之间的物质的量关系，一步计算。

**解题步骤**
1. 写出各步反应的化学方程式
2. 找出中介物质，消去中介物质
3. 建立起始物与目标产物的关系式
4. 按关系式一步计算

**典型应用**
- 工业制硫酸：FeS₂ → 2SO₂ → 2SO₃ → 2H₂SO₄，关系式：FeS₂ ~ 2H₂SO₄
- 工业制硝酸：NH₃ → NO → NO₂ → HNO₃，关系式：NH₃ ~ HNO₃

**二级结论**
- 多步反应的转化率可累积：总转化率 = 各步转化率之积
- 关系式法中，各步转化率可直接乘到计量数上
