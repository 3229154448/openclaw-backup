# 第9章 电化学

## 模块一 原电池

### 第1讲 原电池原理与电极判断

**原电池本质**
化学能 → 电能（自发的氧化还原反应）

**构成条件**
1. 两个活泼性不同的电极
2. 电解质溶液
3. 形成闭合回路
4. 自发的氧化还原反应

**电极判断**
| 判断依据 | 负极 | 正极 |
|---------|------|------|
| 电子流向 | 流出电子 | 流入电子 |
| 电流流向 | 流入电流 | 流出电流 |
| 反应类型 | 氧化反应 | 还原反应 |
| 活泼性 | 较活泼金属 | 较不活泼金属或非金属 |
| 离子迁移 | 阳离子移向 | 阳离子移向 |

**二级结论**
- 电子：外电路从负极流向正极
- 电流：外电路从正极流向负极
- 阳离子移向正极，阴离子移向负极
- 负极失电子发生氧化反应，正极得电子发生还原反应

### 第2讲 电极反应式书写

**书写步骤**
1. 确定正负极和反应物
2. 写出负极反应式（氧化反应，失电子）
3. 写出正极反应式（还原反应，得电子）
4. 正负极反应式相加 = 总反应方程式

**注意事项**
- 酸性环境：可用H⁺和H₂O配平
- 碱性环境：可用OH⁻和H₂O配平（不能出现H⁺）
- 有熔融盐或固体电解质时，需用特定离子配平

**常见电池的电极反应式**

**锌铜原电池（稀H₂SO₄）**
- 负极：Zn - 2e⁻ → Zn²⁺
- 正极：2H⁺ + 2e⁻ → H₂↑

**氢氧燃料电池（酸性）**
- 负极：H₂ - 2e⁻ → 2H⁺
- 正极：O₂ + 4H⁺ + 4e⁻ → 2H₂O

**氢氧燃料电池（碱性）**
- 负极：H₂ + 2OH⁻ - 2e⁻ → 2H₂O
- 正极：O₂ + 2H₂O + 4e⁻ → 4OH⁻

**二级结论**
- 燃料电池负极通入燃料，正极通入氧化剂（通常为O₂）
- 水溶液中正极反应式 = 总反应 - 负极反应式
- 书写步骤：先写总反应 → 再写较简单一极的电极反应式 → 用总反应减去该极反应得另一极

### 第3讲 盐桥与离子迁移

**盐桥的作用**
1. 构成闭合回路
2. 保持电荷平衡（避免两极溶液带电）

**盐桥中的离子迁移**
- 阳离子移向正极（正极因得电子产生过剩正电荷需中和）
- 阴离子移向负极（负极因失电子产生过剩正电荷需中和）

**二级结论**
- 无盐桥时，两极溶液分别带电，电流很快中断
- 盐桥中通常盛有KCl或KNO₃饱和溶液（离子迁移速率相近）

## 模块二 化学电源

### 第4讲 常见化学电源

**一次电池（不可充电）**
- 锌锰干电池：Zn（负极）、MnO₂（正极）、NH₄Cl（电解质）
  - 负极：Zn - 2e⁻ → Zn²⁺
  - 正极：2MnO₂ + 2NH₄⁺ + 2e⁻ → Mn₂O₃ + 2NH₃ + H₂O
- 碱性锌锰电池：KOH电解质

**二次电池（可充电）**
- 铅蓄电池：Pb（负极）、PbO₂（正极）、H₂SO₄（电解质）
  - 放电负极：Pb + SO₄²⁻ - 2e⁻ → PbSO₄
  - 放电正极：PbO₂ + 4H⁺ + SO₄²⁻ + 2e⁻ → PbSO₄ + 2H₂O
  - 充电时阴极：PbSO₄ + 2e⁻ → Pb + SO₄²⁻（原负极作阴极）
  - 充电时阳极：PbSO₄ + 2H₂O - 2e⁻ → PbO₂ + 4H⁺ + SO₄²⁻（原正极作阳极）

- 锂离子电池：
  - 负极：C₆（嵌锂碳），Li⁺嵌入/脱嵌
  - 正极：LiCoO₂或LiFePO₄
  - 放电时：Li从负极脱嵌→正极嵌入（Li⁺通过电解质迁移）

**燃料电池**
- 氢氧燃料电池：H₂（负极）、O₂（正极）
- 甲烷燃料电池（酸性）：CH₄ + 2H₂O - 8e⁻ → CO₂ + 8H⁺（负极）
- 甲醇燃料电池（碱性）：CH₃OH + 8OH⁻ - 6e⁻ → CO₃²⁻ + 6H₂O（负极）

**二级结论**
- 充电时：原电池正极 → 电解池阳极，原电池负极 → 电解池阴极
- 充电时电极反应 = 放电时电极反应的逆反应
- 二次电池放电时作原电池，充电时作电解池

## 模块三 电解池

### 第5讲 电解原理

**电解池本质**
电能 → 化学能（非自发的氧化还原反应）

**电极判断**
| 判断依据 | 阳极 | 阴极 |
|---------|------|------|
| 与电源连接 | 与正极相连 | 与负极相连 |
| 电子流向 | 电子流出 | 电子流入 |
| 反应类型 | 氧化反应 | 还原反应 |
| 离子迁移 | 阴离子移向 | 阳离子移向 |

### 第6讲 放电顺序

**阳极放电顺序（氧化）**
- 活性电极（Fe、Cu、Ag等）> 非活性电极（Pt、Au、石墨）
- 活性电极先溶解：Cu - 2e⁻ → Cu²⁺
- 非活性电极时，阴离子放电：S²⁻ > I⁻ > Br⁻ > Cl⁻ > OH⁻ > 含氧酸根

**阴极放电顺序（还原）**
- Ag⁺ > Fe³⁺ > Cu²⁺ > H⁺(酸) > Pb²⁺ > Sn²⁺ > Fe²⁺ > Zn²⁺ > H⁺(水) > Al³⁺ > Mg²⁺ > Na⁺

**二级结论**
- 活性电极作阳极时，电极本身被氧化（阳极溶解）
- 电镀时：镀层金属作阳极，镀件作阴极，电镀液含镀层金属离子
- 电解水时：阴极产生H₂，阳极产生O₂

### 第7讲 电解规律

**用惰性电极电解常见溶液**

| 电解质类型 | 电解实质 | 阴极产物 | 阳极产物 | pH变化 | 恢复原浓度 |
|-----------|---------|---------|---------|--------|-----------|
| 强碱（NaOH） | 电解水 | H₂ | O₂ | 增大 | 加水 |
| 含氧酸（H₂SO₄） | 电解水 | H₂ | O₂ | 减小 | 加水 |
| 活泼金属含氧酸盐（Na₂SO₄） | 电解水 | H₂ | O₂ | 不变 | 加水 |
| 活泼金属无氧酸盐（NaCl） | 电解溶质+水 | H₂ | Cl₂ | 增大 | 加HCl |
| 不活泼金属含氧酸盐（CuSO₄） | 电解溶质+水 | Cu | O₂ | 减小 | 加CuO |
| 不活泼金属无氧酸盐（CuCl₂） | 电解溶质 | Cu | Cl₂ | 不变 | 加CuCl₂ |
| 卤化氢（HCl） | 电解溶质 | H₂ | Cl₂ | 增大 | 加HCl |

**二级结论**
- 电解含水电解质溶液的类型判断：看阴阳离子放电顺序
- 精炼铜：粗铜作阳极（Cu及杂质溶解），精铜作阴极（Cu²⁺析出）
- 电镀：阳极溶解、阴极析出，电镀液浓度基本不变

## 模块四 金属腐蚀与防护

### 第8讲 金属腐蚀

**化学腐蚀**
- 金属与接触的物质直接发生化学反应
- 如：铁与Cl₂直接反应

**电化学腐蚀（更常见）**
- **吸氧腐蚀**（中性或弱酸性环境）：
  - 负极：Fe - 2e⁻ → Fe²⁺
  - 正极：O₂ + 2H₂O + 4e⁻ → 4OH⁻
  - 总反应：2Fe + O₂ + 2H₂O → 2Fe(OH)₂
  - Fe(OH)₂进一步氧化：4Fe(OH)₂ + O₂ + 2H₂O → 4Fe(OH)₃
  - Fe(OH)₃分解：2Fe(OH)₃ → Fe₂O₃·xH₂O + (3-x)H₂O（铁锈）

- **析氢腐蚀**（强酸性环境）：
  - 负极：Fe - 2e⁻ → Fe²⁺
  - 正极：2H⁺ + 2e⁻ → H₂↑

**二级结论**
- 钢铁腐蚀以吸氧腐蚀为主
- 氧气浓度大的地方作正极（腐蚀较慢），氧气浓度小的地方作负极（腐蚀较快）
- 海水中的钢铁腐蚀速率 > 潮湿空气中的腐蚀速率

### 第9讲 金属防护

**防护方法**
1. **牺牲阳极保护法**（原电池原理）
   - 用更活泼的金属（Zn）与被保护金属（Fe）连接
   - Zn作负极被腐蚀，Fe作正极被保护

2. **外加电流保护法**（电解池原理）
   - 被保护金属作阴极（与电源负极相连）
   - 辅助阳极（惰性电极）与电源正极相连

3. **其他方法**
   - 涂保护层（涂油漆、涂油脂、搪瓷）
   - 镀层保护（镀锌、镀铬）
   - 制成合金（不锈钢）

**二级结论**
- 牺牲阳极保护法：保护金属作正极
- 外加电流保护法：保护金属作阴极
- 镀锌铁：Zn先被腐蚀（牺牲阳极），Fe被保护
- 镀锡铁：Fe先被腐蚀（Sn更不活泼作正极），Fe腐蚀更快
