# 第4章 元素周期律与元素推断

## 模块一 元素周期表结构

### 第1讲 元素周期表的结构与元素位置推断

**周期表结构**
- **7个周期**：1~3为短周期（含2、8、8种元素），4~7为长周期（含18、18、32、32种元素），第7周期为不完全周期
- **18列16族**：7个主族（ⅠA~ⅦA）、7个副族（ⅠB~ⅦB）、Ⅷ族（8~10列）、0族（18列）
- **分区**：s区（ⅠA、ⅡA）、p区（ⅢA~ⅦA、0族）、d区（ⅢB~ⅦB、Ⅷ族）、ds区（ⅠB、ⅡB）、f区（镧系、锕系）

**位置推断方法**
1. 由原子序数推断位置：
   - 确定周期数：与0族元素原子序数比较
   - 确定族数：根据该周期0族元素原子序数之差判断

**零族元素原子序数**：2、10、18、36、54、86、118

**二级结论**
- 第n周期第ⅠA族元素原子序数 = 第(n-1)周期0族元素原子序数 + 1
- 同周期主族元素从左到右：原子半径减小、金属性减弱、非金属性增强
- 同主族元素从上到下：原子半径增大、金属性增强、非金属性减弱

**对角线规则**
- Li 与 Mg、Be 与 Al、B 与 Si：对角线位置的元素性质相似
- 应用：LiOH难溶、Be(OH)₂两性、Li在空气中燃烧生成Li₂O（非Li₂O₂）

## 模块二 元素周期律

### 第2讲 原子半径递变规律

**递变规律**
- 同周期从左到右：原子半径逐渐减小（核电荷数增大，吸引增强）
- 同主族从上到下：原子半径逐渐增大（电子层数增多）
- 核外电子排布相同时：核电荷数越大，半径越小
  - 如 O²⁻ > F⁻ > Na⁺ > Mg²⁺ > Al³⁺（核外电子排布相同，但核电荷数递增）

**二级结论**
- 阴离子半径 > 对应原子半径（如 Cl⁻ > Cl）
- 阳离子半径 < 对应原子半径（如 Na⁺ < Na）
- 同元素不同价态：高价态半径 < 低价态半径（如 Fe³⁺ < Fe²⁺）

### 第3讲 电离能与电负性

**电离能（I）**
- 第一电离能：气态原子失去一个电子所需的最小能量
- 同周期从左到右：总体增大，但ⅡA > ⅢA（ns²全满，稳定），ⅤA > ⅥA（np³半满，稳定）
- 同主族从上到下：逐渐减小

**二级结论**
- Be的第一电离能大于B（2s²全满稳定）
- N的第一电离能大于O（2p³半满稳定）
- Mg的第一电离能大于Al（3s²全满稳定）
- P的第一电离能大于S（3p³半满稳定）
- 电离能突增判断：连续电离能数据中，某次电离能突然增大，说明该电子处于更内层

**电负性（χ）**
- 衡量原子在化学键中吸引电子能力的大小
- 同周期从左到右：增大；同主族从上到下：减小
- 电负性差 > 1.7 一般为离子键，< 1.7 一般为共价键（粗略判断）

### 第4讲 金属性与非金属性递变

**金属性判断依据**
1. 单质与水或酸反应的难易程度
2. 最高价氧化物对应水化物碱性强弱
3. 置换反应（活泼金属置换不活泼金属）
4. 阳离子氧化性强弱（氧化性越弱，对应金属金属性越强）

**非金属性判断依据**
1. 单质与氢气化合的难易程度
2. 气态氢化物的稳定性
3. 最高价氧化物对应水化物酸性强弱
4. 置换反应（活泼非金属置换不活泼非金属）

**二级结论**
- 金属性 Na > Mg > Al；K > Na > Li
- 非金属性 Cl > S > P > Si；F > Cl > Br > I
- Ca(OH)₂碱性强于Mg(OH)₂ → 金属性Ca > Mg
- HClO₄酸性强于H₂SO₄ → 非金属性Cl > S

## 模块三 元素推断

### 第5讲 原子序数关系推断

**方法**
- 同周期相邻元素原子序数之差为1（特殊：过渡元素与主族元素可能差更多）
- 同主族上下相邻元素原子序数之差：第2~3周期差8，第3~4周期差8或18（视位置而定），第4~5周期差18，第5~6周期差32

**二级结论**
- 若A、B同周期且A在B左侧，原子序数差 = B的族序数 - A的族序数
- 若A、B同主族且A在B上方，原子序数差 = A所在周期元素种数之和
- "上邻下邻"关系可快速定位元素

### 第6讲 "位—构—性"综合推断

**推断思路**
1. **由位推构**：位置 → 电子排布 → 原子结构
2. **由构推位**：电子排布 → 周期数（电子层数）+ 族数（最外层电子数）
3. **由位推性**：位置 → 金属性/非金属性 → 单质及化合物性质
4. **由性推位**：性质特征 → 位置判断

**常见推断线索**
- "短周期元素X的最外层电子数是内层电子数的2倍"→ X为C（2,4）
- "短周期元素Y的原子半径最大"→ Y为Na
- "元素Z的最高正价与最低负价绝对值之比为3:1"→ Z为S或Cl（S: +6/-2=3:1; Cl不是3:1）
- "元素W的气态氢化物与最高价氧化物对应水化物反应生成盐"→ W为N（NH₃ + HNO₃）

**典型题型**
- 给出前20号元素的推断
- 给出元素化合价特征、物质性质特征的推断
- 周期表局部推断（给出几个元素间的关系）

### 第7讲 元素周期表的应用

**应用方向**
1. 寻找新材料：半导体元素在金属与非金属分界线附近（Si、Ge、Ga等）
2. 寻找催化剂：过渡元素区域（Fe、Ni、Pt、Pd等）
3. 寻找农药元素：右上方非金属区域（F、Cl、S、P等）
4. 寻找耐高温合金：副族元素
