# 第7章 化学反应速率与化学平衡

## 模块一 化学反应速率

### 第1讲 化学反应速率的计算

**核心公式**
$$v = \frac{\Delta c}{\Delta t}$$
- v：反应速率（mol/(L·s) 或 mol/(L·min)）
- Δc：浓度变化量
- Δt：时间变化量

**不同物质表示的速率关系**
$$v_A : v_B : v_C : v_D = a : b : c : d$$
（a、b、c、d为化学方程式中的化学计量数）

**二级结论**
- 固体和纯液体的浓度视为常数，不能用其表示反应速率
- 不同物质表示的速率之比 = 化学计量数之比
- 反应速率取绝对值，无负值

### 第2讲 影响反应速率的因素

**影响因素**
| 因素 | 影响 | 解释 |
|------|------|------|
| 浓度 | 增大浓度，速率增大 | 活化分子浓度增大，有效碰撞频率增大 |
| 压强 | 增大压强（缩容），气相反应速率增大 | 气体浓度增大 |
| 温度 | 升高温度，速率增大 | 活化分子百分数增大（主要因素） |
| 催化剂 | 加入催化剂，速率增大 | 降低活化能，增大活化分子百分数 |

**有效碰撞理论**
- **活化分子**：能量足够高、能发生有效碰撞的分子
- **有效碰撞**：活化分子在适当取向下发生的碰撞
- **活化能**：活化分子具有的最低能量与反应物平均能量之差

**二级结论**
- 温度每升高10°C，反应速率一般增大2~4倍
- 催化剂对正逆反应速率同等程度增大
- 增大压强仅对有气体参与的反应有影响
- 充入惰性气体：恒容充入不影响速率（各组分浓度不变）；恒压充入减小速率（体积增大，浓度减小）

## 模块二 化学平衡

### 第3讲 化学平衡状态判断

**化学平衡特征**
- **逆**：可逆反应
- **等**：正反应速率 = 逆反应速率
- **动**：动态平衡（反应未停止）
- **定**：各组分浓度/含量保持不变
- **变**：条件改变，平衡移动

**等价标志判断**
1. **速率标志**：正反应速率 = 逆反应速率（同一物质的消耗速率 = 生成速率）
2. **浓度标志**：各组分浓度不再变化
3. **压强标志**：恒温恒容时，若Δn(g)≠0，则总压不变
4. **密度标志**：恒温恒压时，混合气体密度不变
5. **平均相对分子质量标志**：恒温恒压/恒容时，若Δn(g)≠0，平均相对分子质量不变

**二级结论**
- 对于Δn(g)=0的反应（如H₂+I₂⇌2HI），总压不变不能作为平衡标志
- 各组分浓度不变 ≠ 各组分浓度相等
- 转化率不变可作平衡标志

### 第4讲 等效平衡

**定义**
在相同条件下，同一可逆反应达到平衡后，各对应组分的百分含量（浓度、物质的量分数等）相同的平衡状态。

**类型与判断**

**1. 恒温恒容（T、V不变）**
- 若 Δn(g)≠0：极值换算后，对应物质的起始量完全相同 → 等效
- 若 Δn(g)=0：极值换算后，对应物质的起始量成比例 → 等效

**2. 恒温恒压（T、p不变）**
- 极值换算后，对应物质的起始量成比例 → 等效

**解题方法**
1. 将所有物质全部转化为同一侧（如反应物侧），称为"极值换算"
2. 比较极值换算后的各物质比例
3. 根据上述规则判断是否等效

## 模块三 化学平衡移动

### 第5讲 勒夏特列原理

**原理内容**
如果改变影响平衡的一个条件（浓度、压强、温度等），平衡就向能够减弱这种改变的方向移动。

**二级结论**
- "减弱"而非"消除"：移动后新平衡态的改变量小于原改变量
- 如增大某反应物浓度，平衡正移，但该反应物的新浓度仍大于原浓度

### 第6讲 浓度、压强、温度对平衡的影响

**浓度影响**
- 增大反应物浓度 → 正移
- 减小生成物浓度 → 正移
- 减小反应物浓度 → 逆移
- 增大生成物浓度 → 逆移

**压强影响**（仅对有气体参与且Δn(g)≠0的反应）
- 增大压强（缩容）→ 向气体物质的量减小的方向移动
- 减小压强（扩容）→ 向气体物质的量增大的方向移动
- Δn(g)=0 的反应：压强改变不影响平衡移动（但影响到达平衡的速率）

**温度影响**
- 升高温度 → 向吸热反应方向移动
- 降低温度 → 向放热反应方向移动

**催化剂**
- 催化剂同等程度增大正逆反应速率
- **不影响**化学平衡移动
- 缩短到达平衡的时间

**二级结论**
- 恒容充入惰性气体：总压增大，但各组分分压不变 → 平衡不移动
- 恒压充入惰性气体：体积增大，各组分分压减小 → 等效于减压 → 平衡向气体体积增大方向移动
- 增大一种反应物浓度，另一种反应物的转化率增大，但该反应物自身的转化率降低

### 第7讲 平衡常数与转化率

**化学平衡常数（K）**
- 表达式：对于反应 aA + bB ⇌ cC + dD
$$K = \frac{c^c(C) \cdot c^d(D)}{c^a(A) \cdot c^b(B)}$$
- 固体和纯液体不写入K的表达式
- K只与温度有关，与浓度、压强无关
- 升高温度：正向吸热 → K增大；正向放热 → K减小

**浓度商（Q）与平衡判断**
- Q < K → 反应正向进行
- Q = K → 处于平衡状态
- Q > K → 反应逆向进行

**转化率计算**
$$\alpha = \frac{\text{已反应的量}}{\text{起始总量}} \times 100\%$$

**二级结论**
- 同一反应，同一温度下，K值唯一
- 平衡常数K越大，正向反应程度越大
- 多重平衡：总反应的K = 各步反应K的乘积

## 模块四 化学平衡图像题

### 第8讲 化学平衡图像分析

**常见图像类型**

**1. 浓度-时间图**
- 看各物质浓度变化趋势
- 平衡时浓度不再变化

**2. 速率-时间图**
- 正逆反应速率在某时刻相等 → 达到平衡
- 条件改变时速率突变：催化剂（正逆同增）、温度（正逆异增）、浓度（一方增大）

**3. 转化率-温度图（等压线）**
- 先增后减型：升高温度先加快反应速率（未平衡时），达平衡后正向放热则转化率降低
- 单调型：正向吸热则转化率随温度升高而增大

**4. 转化率-压强图（等温线）**
- 增大压强，正向气体体积减小 → 转化率增大
- 增大压强，正向气体体积增大 → 转化率减小

**解题方法**
1. 看清横纵坐标的含义
2. 看曲线变化趋势
3. 结合平衡移动原理解释
4. 注意拐点、交点、极值点的含义
