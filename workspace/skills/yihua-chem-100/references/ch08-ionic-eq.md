# 第8章 水溶液中的离子平衡

## 模块一 弱电解质的电离平衡

### 第1讲 弱电解质的电离

**电离平衡特征**
- 弱电解质电离是可逆过程
- 达到电离平衡时：电离速率 = 结合速率
- 条件改变，电离平衡移动

**电离度（α）**
$$\alpha = \frac{\text{已电离的电解质分子数}}{\text{电解质分子总数}} \times 100\%$$

**电离常数（K_a / K_b）**
- 弱酸 HA：$K_a = \frac{c(H^+) \cdot c(A^-)}{c(HA)}$
- 弱碱 BOH：$K_b = \frac{c(B^+) \cdot c(OH^-)}{c(BOH)}$
- K_a / K_b 只与温度有关，与浓度无关
- 温度升高，K_a / K_b 增大（电离吸热）

**二级结论**
- K_a 越大，弱酸酸性越强
- 多元弱酸分步电离：K_a₁ >> K_a₂ >> K_a₃，酸性主要由第一步决定
- H₂CO₃：K_a₁ = 4.4×10⁻⁷，K_a₂ = 4.7×10⁻¹¹
- H₂SO₃：K_a₁ = 1.3×10⁻²，K_a₂ = 6.3×10⁻⁸

### 第2讲 稀释定律

**稀释对弱电解质的影响**
- 稀释时电离度α增大（平衡向电离方向移动）
- 稀释时各粒子浓度均减小（α增大的幅度 < 体积增大的幅度）
- c(H⁺)、c(A⁻)减小，c(HA)减小
- 加水稀释过程中，c(H⁺)/c(HA)增大

**二级结论**
- 加水稀释弱酸，pH增大（向7靠近）
- 加水稀释弱碱，pH减小（向7靠近）
- 稀释10倍，pH变化 < 1（弱电解质），稀释10倍，pH变化 = 1（强电解质）

## 模块二 水的电离与溶液pH

### 第3讲 水的电离与水的离子积

**水的离子积常数**
$$K_W = c(H^+) \cdot c(OH^-)$$
- 25°C时：K_W = 1.0 × 10⁻¹⁴
- 温度升高，K_W 增大（水电离吸热）
- 100°C时：K_W ≈ 1.0 × 10⁻¹²

**二级结论**
- 任何水溶液中：K_W = c(H⁺) · c(OH⁻) 恒成立
- 酸溶液中c(OH⁻)来自水的电离，碱溶液中c(H⁺)来自水的电离
- 促进水电离：加入能水解的盐（如Na₂CO₃、NH₄Cl）
- 抑制水电离：加入酸或碱

### 第4讲 pH计算

**公式**
$$pH = -\lg c(H^+)$$

**25°C时的关系**
| 溶液 | c(H⁺) | c(OH⁻) | pH |
|------|--------|--------|-----|
| 中性 | 10⁻⁷ | 10⁻⁷ | 7 |
| 酸性 | >10⁻⁷ | <10⁻⁷ | <7 |
| 碱性 | <10⁻⁷ | >10⁻⁷ | >7 |

**酸碱稀释规律**
- 强酸稀释10ⁿ倍：pH增大n（不超过7）
- 强碱稀释10ⁿ倍：pH减小n（不低于7）
- 弱酸稀释10ⁿ倍：pH增大 < n
- 弱碱稀释10ⁿ倍：pH减小 < n
- 无限稀释：pH均趋近于7

**酸碱混合pH计算**
- 强酸+强酸：c(H⁺) = (n₁c₁ + n₂c₂) / (V₁ + V₂)
- 强碱+强碱：先算c(OH⁻)，再求c(H⁺) = K_W / c(OH⁻)
- 强酸+强碱：先判断过量，再求混合后c(H⁺)或c(OH⁻)

**二级结论**
- 强酸 pH=a，稀释10ⁿ倍后 pH = a+n（a+n < 7）
- 强碱 pH=b，稀释10ⁿ倍后 pH = b-n（b-n > 7）
- 25°C，pH + pOH = 14

## 模块三 盐类水解

### 第5讲 盐类水解的规律

**水解本质**
盐中的离子与水电离出的H⁺或OH⁻结合生成弱电解质，破坏水的电离平衡。

**水解规律**
| 盐的类型 | 是否水解 | 溶液酸碱性 | 举例 |
|---------|---------|-----------|------|
| 强酸强碱盐 | 不水解 | 中性 | NaCl、KNO₃ |
| 强酸弱碱盐 | 阳离子水解 | 酸性 | NH₄Cl、FeCl₃、Al₂(SO₄)₃ |
| 弱酸强碱盐 | 阴离子水解 | 碱性 | Na₂CO₃、CH₃COONa、NaHCO₃ |
| 弱酸弱碱盐 | 双水解 | 视情况而定 | NH₄Ac（中性）、(NH₄)₂CO₃（碱性） |

**水解方程式书写**
- 单方水解：写可逆号"⇌"，不标"↑""↓"
  - 如 Fe³⁺ + 3H₂O ⇌ Fe(OH)₃ + 3H⁺
  - 如 CO₃²⁻ + H₂O ⇌ HCO₃⁻ + OH⁻
- 双水解（彻底进行）：写等号"="，标"↑""↓"
  - 如 Al³⁺ + 3HCO₃⁻ = Al(OH)₃↓ + 3CO₂↑

**二级结论**
- 越弱越水解：酸越弱，其对应的盐水解程度越大
- 盐的浓度越小，水解程度越大（稀释促进水解）
- 升高温度促进水解
- NaHCO₃溶液呈碱性：HCO₃⁻的水解程度 > 电离程度
- NaHSO₄溶液呈酸性：HSO₄⁻完全电离（强酸的酸式盐）

### 第6讲 溶液中离子浓度大小比较

**三大守恒**
1. **电荷守恒**：所有阳离子正电荷总数 = 所有阴离子负电荷总数
   - 如 Na₂CO₃溶液：c(Na⁺) + c(H⁺) = 2c(CO₃²⁻) + c(HCO₃⁻) + c(OH⁻)
2. **物料守恒（原子守恒）**：某种元素的总浓度等于其各种存在形式的浓度之和
   - 如 Na₂CO₃溶液：c(Na⁺) = 2[c(CO₃²⁻) + c(HCO₃⁻) + c(H₂CO₃)]
3. **质子守恒**：得质子产物与失质子产物的浓度关系
   - 可由电荷守恒 + 物料守恒推出

**解题步骤**
1. 判断溶液中的所有离子和分子
2. 比较主要离子浓度大小
3. 用三大守恒验证等式关系

**二级结论**
- Na₂CO₃溶液中：c(Na⁺) > c(CO₃²⁻) > c(OH⁻) > c(HCO₃⁻) > c(H⁺)
- CH₃COONa溶液中：c(Na⁺) > c(CH₃COO⁻) > c(OH⁻) > c(H⁺)
- NH₄Cl溶液中：c(Cl⁻) > c(NH₄⁺) > c(H⁺) > c(OH⁻)
- NaHCO₃溶液中：c(Na⁺) > c(HCO₃⁻) > c(OH⁻) > c(CO₃²⁻) > c(H₂CO₃) > c(H⁺)

## 模块四 酸碱中和滴定

### 第7讲 酸碱中和滴定

**滴定曲线**
- 强酸滴定强碱：起点pH高，终点突变明显
- 强碱滴定强酸：起点pH低，终点突变明显
- 滴定终点：pH = 7（强酸强碱）

**指示剂选择**
| 滴定类型 | 突跃范围 | 指示剂选择 |
|---------|---------|-----------|
| 强酸滴强碱 | 4.3~9.7 | 酚酞或甲基橙 |
| 强酸滴弱碱 | 4.3~6.3 | 甲基橙 |
| 强碱滴弱酸 | 7.7~9.7 | 酚酞 |

**误差分析**
- 仰视读数 → 读数偏大 → c(待测)偏大
- 俯视读数 → 读数偏小 → c(待测)偏小
- 未润洗滴定管 → 标准液被稀释 → 消耗体积偏大 → c(待测)偏大
- 滴定前气泡，滴定后无气泡 → 消耗体积偏大 → c(待测)偏大
- 锥形瓶用待测液润洗 → 待测液偏多 → c(待测)偏大

**二级结论**
- 滴定终点判断：半滴操作，悬而不落
- 多次滴定取平均值，偏差过大的舍去
- 雉形瓶不能润洗，但必须干燥或用待测液润洗后倒掉

## 模块五 沉淀溶解平衡

### 第8讲 沉淀溶解平衡

**溶度积常数（K_sp）**
对于沉淀 A_mB_n ⇌ mA^n⁺ + nB^m⁻
$$K_{sp} = c^m(A^{n+}) \cdot c^n(B^{m-})$$

**溶度积规则**
- Q_c > K_sp → 有沉淀生成
- Q_c = K_sp → 饱和溶液，平衡状态
- Q_c < K_sp → 不饱和溶液，无沉淀

**沉淀转化**
- 溶解度大的沉淀易转化为溶解度小的沉淀
- K_sp小的更难溶

**二级结论**
- 同类型沉淀（如AB型）：K_sp越小，溶解度越小
- 不同类型沉淀不能直接用K_sp比较溶解度，需换算
- 加入同离子使沉淀更完全（同离子效应）
- 沉淀完全的标准：c(被沉淀离子) < 10⁻⁵ mol/L
