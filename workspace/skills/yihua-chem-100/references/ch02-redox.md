# 第2章 氧化还原反应

## 模块一 氧化还原基本概念

### 第1讲 氧化还原反应的判断与基本概念

**核心概念**
- **氧化反应**：物质失去电子（化合价升高）的反应
- **还原反应**：物质得到电子（化合价降低）的反应
- 氧化还原反应的本质：**电子的转移**（得失或偏移）
- 氧化还原反应的特征：**化合价的升降**

**四组核心概念**
| 概念 | 含义 | 变化 |
|------|------|------|
| 氧化剂 | 得到电子的物质 | 化合价降低 → 发生还原反应 → 对应还原产物 |
| 还原剂 | 失去电子的物质 | 化合价升高 → 发生氧化反应 → 对应氧化产物 |
| 氧化产物 | 还原剂被氧化后的产物 | 化合价较原物质升高 |
| 还原产物 | 氧化剂被还原后的产物 | 化合价较原物质降低 |

**记忆口诀**：升失氧还、降得还氧
- 化合价升高 → 失电子 → 氧化反应 → 作还原剂 → 被氧化 → 生成氧化产物
- 化合价降低 → 得电子 → 还原反应 → 作氧化剂 → 被还原 → 生成还原产物

**二级结论**
- 氧化剂和还原剂可以是同一种物质（歧化反应）：如 Cl₂ + 2NaOH → NaCl + NaClO + H₂O
- 氧化产物和还原产物可以是同一种物质（反歧化反应）：如 NaClO + 2HCl → NaCl + Cl₂↑ + H₂O 中 Cl₂ 既是氧化产物又是还原产物
- 氧化性：氧化剂 > 氧化产物；还原性：还原剂 > 还原产物

### 第2讲 电子转移的表示方法

**双线桥法**
- 从反应物中化合价变化的元素出发，指向产物中对应元素
- 上方标"失去/得到 × ne⁻"
- 必须标注电子转移总数

**单线桥法**
- 从还原剂（失电子的元素）出发，指向氧化剂（得电子的元素）
- 只标注电子转移总数，不标得失
- 适用于简单氧化还原反应

**解题要点**
1. 先标化合价，确定哪些元素化合价变化
2. 计算化合价升降总数，确保升降相等
3. 画线桥标注

### 第3讲 常见氧化剂与还原剂

**常见氧化剂**
| 类型 | 举例 |
|------|------|
| 活泼非金属单质 | Cl₂、O₂、O₃、Br₂ |
| 高价态化合物 | KMnO₄、K₂Cr₂O₇、HNO₃、浓H₂SO₄ |
| 高价态离子 | Fe³⁺、Cu²⁺、Ag⁺ |
| 过氧化物 | H₂O₂、Na₂O₂ |
| 含氧酸 | 浓HNO₃ > 稀HNO₃；浓H₂SO₄ |

**常见还原剂**
| 类型 | 举例 |
|------|------|
| 活泼金属单质 | Na、Mg、Al、Fe、Zn |
| 低价态化合物 | CO、SO₂、SO₃²⁻、I⁻、Fe²⁺ |
| 非金属单质 | H₂、C |
| 特殊 | 浓盐酸（Cl⁻作还原剂） |

**二级结论**
- 氧化性：F₂ > Cl₂ > Br₂ > I₂ > S
- 还原性：S²⁻ > I⁻ > Fe²⁺ > Br⁻ > Cl⁻ > F⁻
- 同种元素不同价态：最高价态只有氧化性，最低价态只有还原性，中间价态既有氧化性又有还原性

## 模块二 氧化还原规律

### 第4讲 氧化还原反应的规律

**规律一：强弱律**
- 强氧化剂 + 强还原剂 → 弱氧化产物 + 弱还原产物
- 即：氧化性 氧化剂 > 氧化产物；还原性 还原剂 > 还原产物
- **应用**：由已知反应比较物质氧化性/还原性强弱

**规律二：先后律（竞争反应）**
- 当一种氧化剂遇到多种还原剂时，**还原性强的先反应**
- 当一种还原剂遇到多种氧化剂时，**氧化性强的先反应**
- **应用**：判断反应顺序、书写离子方程式

**规律三：价态律（归中律与歧化律）**
- 同种元素不同价态之间反应时，价态变化"只靠拢、不交叉"
- 归中反应：高价态 + 低价态 → 中间价态（不交叉）
- 歧化反应：中间价态 → 高价态 + 低价态

**规律四：转化律**
- 逐级转化规律：元素被氧化/还原时往往逐级进行
- 如：Fe → Fe²⁺ → Fe³⁺（先到+2再到+3）

**典型题型**
- 给出几个反应方程式，比较氧化性/还原性强弱
- 判断反应先后顺序
- 归中反应产物的判断

## 模块三 氧化还原方程式配平

### 第5讲 化合价升降法配平

**配平步骤**
1. 标出化合价发生变化的元素的化合价
2. 计算化合价升降总数，找出最小公倍数使升降相等
3. 用化学计量数配平氧化剂、还原剂、氧化产物、还原产物
4. 用观察法配平其他物质
5. 检查原子守恒和电荷守恒

**口诀**：标价→求升降价→找最小公倍数→配氧化还原部分→配其他→检验

### 第6讲 缺项配平

**方法**
氧化还原方程式中缺项一般为H₂O、H⁺、OH⁻等。

**解题步骤**
1. 先用化合价升降法配平氧化剂、还原剂及其对应产物
2. 检查方程式左右两边的H、O原子数和电荷数
3. 根据反应环境（酸性/碱性/中性）补缺：
   - **酸性环境**：用 H⁺ 和 H₂O 补齐
   - **碱性环境**：用 OH⁻ 和 H₂O 补齐
   - **中性环境**：反应物中补 H₂O，生成物中可能出 H⁺ 或 OH⁻

**二级结论**
- 电极反应式配平中，酸性环境不能出现OH⁻，碱性环境不能出现H⁺
- 配平顺序：先配C、H、O以外的原子→配H、O原子→配电荷

### 第7讲 信息型氧化还原方程式书写

**方法**
根据题目提供的信息（如"某物质被氧化为…""某物质作氧化剂"等），逐步推导完整的化学方程式。

**解题步骤**
1. 根据信息确定氧化剂、还原剂、氧化产物、还原产物
2. 根据电子守恒确定化学计量数
3. 根据原子守恒补齐其他物质
4. 检查电荷守恒

**二级结论**
- 氧化还原反应中，电子转移总数 = 化合价升高总数 = 化合价降低总数
- 多元素化合价变化时，分别计算各元素变化，总升降要相等
- KMnO₄在酸性条件还原为Mn²⁺，中性/弱碱性还原为MnO₂，强碱性还原为K₂MnO₄
