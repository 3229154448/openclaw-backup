# 第12章 物质结构与性质

## 模块一 原子结构

### 第1讲 核外电子排布规则

**构造原理（Aufbau原理）**
- 电子按能级从低到高依次排布
- 能级顺序：1s → 2s → 2p → 3s → 3p → 4s → 3d → 4p → 5s → 4d → 5p → 6s → 4f → 5d → 6p → 7s → 5f → 6d → 7p

**泡利不相容原理**
- 每个原子轨道最多容纳2个电子，且自旋方向相反

**洪特规则**
- 电子在等价轨道上排布时，优先单独占据一个轨道，且自旋方向相同
- 等价轨道全满、半满、全空状态更稳定

**二级结论**
- 24Cr：[Ar]3d⁵4s¹（而非3d⁴4s²，因为3d⁵半满更稳定）
- 29Cu：[Ar]3d¹⁰4s¹（而非3d⁹4s²，因为3d¹⁰全满更稳定）
- 42Mo：[Kr]4d⁵5s¹（半满稳定）
- 47Ag：[Kr]4d¹⁰5s¹（全满稳定）
- 第四周期过渡元素中，Cr和Cu是两个特例

### 第2讲 能级与能层

**能层（主量子数n）**
- 第1能层(K)：1s（最多2个电子）
- 第2能层(L)：2s、2p（最多8个电子）
- 第3能层(M)：3s、3p、3d（最多18个电子）
- 第4能层(N)：4s、4p、4d、4f（最多32个电子）

**能级类型与轨道数**
- s能级：1个轨道，最多2个电子
- p能级：3个轨道，最多6个电子
- d能级：5个轨道，最多10个电子
- f能级：7个轨道，最多14个电子

**二级结论**
- 每一能层最多容纳电子数 = 2n²
- 最外层电子数不超过8（K层不超过2）
- 次外层电子数不超过18
- 倒数第三层电子数不超过32

### 第3讲 第一电离能的异常

**同周期异常**
- Be > B：Be为2s²全满，B为2s²2p¹，全满稳定不易失电子
- N > O：N为2s²2p³半满，O为2s²2p⁴，半满稳定不易失电子
- Mg > Al：Mg为3s²全满，Al为3s²3p¹
- P > S：P为3s²3p³半满，S为3s²3p⁴

**连续电离能的突增**
- 电离能数据中某次突增，说明该电子从更内层能级失去
- 可判断元素的价电子数和族数
- 如：Na的I₁=496，I₂=4562（突增），说明Na有1个价电子，为ⅠA族

## 模块二 分子结构与性质

### 第4讲 杂化轨道理论（深化）

**杂化方式与分子构型对应**
- sp杂化：2个sp杂化轨道，直线形，键角180°
  - 示例：BeCl₂、CO₂、HC≡CH、HCN
- sp²杂化：3个sp²杂化轨道，平面三角形，键角120°
  - 示例：BF₃、C₂H₄、SO₃、苯环上的C
- sp³杂化：4个sp³杂化轨道，正四面体形，键角109°28'
  - 示例：CH₄、CCl₄、NH₃（1个孤电子对）、H₂O（2个孤电子对）

**等性杂化与不等性杂化**
- 等性杂化：杂化轨道中孤电子对数=0（如CH₄中C的sp³）
- 不等性杂化：杂化轨道中有孤电子对（如NH₃中N的sp³、H₂O中O的sp³）
- 孤电子对占据的空间 > 成键电子对，会挤压其他键角

**二级结论**
- 判断杂化方式：看中心原子的σ键数+孤电子对数
  - 2 → sp，3 → sp²，4 → sp³
- 含双键的碳为sp²杂化（双键中1个σ键+1个π键）
- 含三键的碳为sp杂化（三键中1个σ键+2个π键）

### 第5讲 分子极性与手性

**分子极性判断**
- 对称性分析：若分子中所有极性键的矢量合为零，则为非极性分子
- 非极性分子示例：CO₂、BF₃、CH₄、CCl₄、C₂H₂、C₂H₄
- 极性分子示例：H₂O、NH₃、CH₃Cl、CH₃OH、HF

**手性分子**
- 含手性碳原子（连接4个不同基团的碳原子）的分子可能有手性
- 手性异构体：旋光性不同（左旋/右旋）
- 常见手性分子：乳酸(CH₃CHOHCOOH)

**二级结论**
- 手性碳原子判断：该碳连有4个不同的原子或基团
- 手性异构体的物理性质（除旋光性外）和化学性质相同
- 制药中需注意手性异构体（一种有效，另一种可能有害）

## 模块三 晶体结构与性质

### 第6讲 晶体类型与性质

**四种基本晶体类型**

**离子晶体**
- 构成粒子：阴离子和阳离子
- 粒子间作用：离子键
- 熔沸点：较高
- 导电性：熔融态或溶液中导电
- 示例：NaCl、KBr、CaO、MgO

**原子晶体**
- 构成粒子：原子
- 粒子间作用：共价键
- 熔沸点：很高
- 硬度：很大
- 导电性：不导电（无自由电子）
- 示例：金刚石、SiO₂、SiC、BN

**分子晶体**
- 构成粒子：分子
- 粒子间作用：分子间作用力（含氢键）
- 熔沸点：较低
- 硬度：较小
- 示例：冰、干冰、I₂、S₈

**金属晶体**
- 构成粒子：金属阳离子和自由电子
- 粒子间作用：金属键
- 导电性：良好（自由电子）
- 示例：Na、Fe、Cu、Au

**二级结论**
- 熔沸点比较：原子晶体 > 离子晶体 > 分子晶体（一般情况）
- 同类型离子晶体：离子半径越小、电荷越大，离子键越强，熔沸点越高
  - 如：MgO > NaCl（Mg²⁺、O²⁻半径小、电荷大）
- 同类型分子晶体：相对分子质量越大，范德华力越强，熔沸点越高
  - 如：I₂ > Br₂ > Cl₂ > F₂
- 含氢键的分子晶体熔沸点反常高：如 H₂O > H₂S

### 第7讲 晶体计算

**晶胞参数**
- 晶胞是晶体中最小的重复单位
- 常见晶胞类型：简单立方、体心立方、面心立方

**晶胞中粒子数的计算**
- 顶点粒子：1/8（被8个晶胞共享）
- 棱上粒子：1/4（被4个晶胞共享）
- 面上粒子：1/2（被2个晶胞共享）
- 体心粒子：1（属于1个晶胞）

**常见晶胞**
- NaCl晶胞：4个Na⁺ + 4个Cl⁻（面心立方结构）
- CsCl晶胞：1个Cs⁺ + 1个Cl⁻（体心立方结构）
- 金刚石晶胞：8个C原子
- 面心立方金属：4个原子
- 体心立方金属：2个原子

**密度计算公式**
ρ = nM / (V × N_A)
- n：晶胞中粒子数
- M：摩尔质量（g/mol）
- V：晶胞体积（cm³）
- N_A：阿伏加德罗常数

**二级结论**
- 配位数：每个离子周围最近且等距的异号离子数
- NaCl中Na⁺的配位数为6，CsCl中Cs⁺的配位数为8
- 空间利用率：面心立方74%、体心立方68%、简单立方52%
- 金属晶体的空间利用率：面心立方 = 六方密堆积 > 体心立方

### 第8讲 晶体熔沸点比较方法

**不同类型晶体**
- 原子晶体 > 离子晶体 > 分子晶体（一般情况）
- 金属晶体变化范围大

**同类型晶体比较**
1. **离子晶体**：离子半径越小、电荷越大，离子键越强，熔沸点越高
   - MgO > CaO > NaCl > KCl
2. **原子晶体**：共价键键能越大，熔沸点越高
   - 金刚石 > SiC > Si
3. **分子晶体**：
   - 含氢键 > 不含氢键
   - 相对分子质量越大，范德华力越强
   - 结构相似时：HI > HBr > HCl（HF含氢键反常高）
4. **金属晶体**：离子半径越小、电荷越大，金属键越强，熔沸点越高
   - Na < Mg < Al

## 模块四 配合物

### 第9讲 配位化合物

**基本概念**
- **中心离子（原子）**：提供空轨道的金属离子（如Fe³⁺、Cu²⁺、Ag⁺）
- **配体**：提供孤电子对的分子或离子（如NH₃、H₂O、Cl⁻、CN⁻、SCN⁻）
- **配位数**：直接与中心离子相连的配位原子数
- **配位键**：配体→中心离子（箭头指向中心离子）

**常见配合物**
- [Cu(NH₃)₄]²⁺：铜氨配离子，深蓝色
- [Ag(NH₃)₂]⁺：银氨配离子
- [Fe(SCN)₃]：硫氰化铁，血红色
- [Fe(CN)₆]³⁻/⁴⁻：铁氰化物
- [Co(NH₃)₆]³⁺

**配合物的空间构型**
- 配位数2：直线形（如[Ag(NH₃)₂]⁺）
- 配位数4：四面体形或平面正方形（如[Zn(NH₃)₄]²⁺四面体，[Cu(NH₃)₄]²⁺平面正方形）
- 配位数6：正八面体形（如[Fe(CN)₆]³⁻）

**二级结论**
- 配合物的形成常改变溶解度：AgCl沉淀可溶于氨水，形成[Ag(NH₃)₂]⁺
- 配合物的形成常改变颜色：Cu²⁺(aq)浅蓝色，[Cu(NH₃)₄]²⁺深蓝色
- 配离子的稳定性：配位键越强，配合物越稳定
- 配合物中，中心离子的氧化性/还原性会改变
