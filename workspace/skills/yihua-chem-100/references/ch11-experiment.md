# 第11章 化学实验

## 模块一 常用仪器与基本操作

### 第1讲 常用仪器及使用方法

**计量仪器**
- 量筒：精度0.1 mL，粗略量取液体，0刻度在下方，不能加热
- 容量瓶：精度0.01 mL（标线），配制一定浓度溶液，标明温度和容积，只有一个刻度线
- 滴定管：精度0.01 mL，精确量取液体，0刻度在上方，酸式滴定管盛酸，碱式盛碱
- 托盘天平：精度0.1 g，称量固体，左物右码
- 温度计：注意量程，水银球位置因实验而异

**加热仪器**
- 直接加热：试管、蒸发皿、坩埚
- 垫石棉网加热：烧杯、烧瓶、锥形瓶

**二级结论**
- 量筒读数：仰视偏大，俯视偏小
- 滴定管读数：仰视偏大，俯视偏小
- 容量瓶不能在室温以外使用，不能长期存放溶液
- 温度计水银球位置：蒸馏→支管口；制乙烯→反应液中；测溶解度→水浴中

### 第2讲 化学实验基本操作

**药品取用**
- 固体：药匙（粉末）或镊子（块状），试管横放→慢慢竖起
- 液体：量筒量取或胶头滴管滴加
- 取用原则：不接触药品、不闻气味、不品尝味道

**加热操作**
- 液体加热：试管中不超过1/3，试管口不对人
- 固体加热：试管口略向下倾斜
- 蒸馏：温度计水银球在支管口处，加碎瓷片防暴沸

**溶解与稀释**
- 浓硫酸稀释：**酸入水**，沿器壁慢慢注入，不断搅拌
- NaOH溶解：大量放热，在烧杯中进行

**指示剂使用**
- 酚酞：pH 8.2~10.0（无色→红色）
- 甲基橙：pH 3.1~4.4（红色→橙色→黄色）
- 石蕊：pH 5.0~8.0（红色→紫色→蓝色）
- 一般用量：2~3滴

**二级结论**
- 蒸发结晶：蒸发皿中出现较多固体时停止加热，利用余热蒸干
- 过滤：一贴二低三靠
- 蒸馏：冷凝水从下口进、上口出（逆流冷却）

## 模块二 气体制备与收集

### 第3讲 常见气体的制备

**气体制备装置选择**

- O₂：2KMnO₄ →(△) K₂MnO₄ + MnO₂ + O₂↑（固固加热型），排水法/向上排空气法
- O₂：2H₂O₂ →(MnO₂) 2H₂O + O₂↑（固液常温型）
- CO₂：CaCO₃ + 2HCl → CaCl₂ + CO₂↑ + H₂O（固液常温型），向上排空气法
- H₂：Zn + H₂SO₄(稀) → ZnSO₄ + H₂↑（固液常温型），排水法/向下排空气法
- Cl₂：MnO₂ + 4HCl(浓) →(△) MnCl₂ + Cl₂↑ + 2H₂O（固液加热型），向上排空气法
- NH₃：2NH₄Cl + Ca(OH)₂ →(△) CaCl₂ + 2NH₃↑ + 2H₂O（固固加热型），向下排空气法
- NO：3Cu + 8HNO₃(稀) → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O，排水法
- NO₂：Cu + 4HNO₃(浓) → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O，向上排空气法
- SO₂：Na₂SO₃ + H₂SO₄(浓) → Na₂SO₄ + SO₂↑ + H₂O（固液常温型），向上排空气法
- C₂H₄：CH₃CH₂OH →(浓H₂SO₄,170°C) CH₂=CH₂↑ + H₂O（液液加热型），排水法

**二级结论**
- 制CO₂不能用H₂SO₄（与CaCO₃反应生成微溶CaSO₄覆盖表面，阻止反应）
- 制Cl₂用浓盐酸，稀盐酸不行
- 制NH₃不能用NH₄HCO₃（分解产生CO₂混入）
- 制C₂H₄需加碎瓷片防暴沸，温度计水银球插入液面以下
- 启普发生器适用于：块状固体+液体、不加热、生成的气体不溶于反应液

### 第4讲 气体的净化、干燥与尾气处理

**气体净化原则**
- 先净化后干燥
- 加热净化装置在前，冷却干燥在后
- 净化剂选择：不与目标气体反应，能除去杂质

**常用干燥剂**
- 浓H₂SO₄：可干燥 H₂、O₂、Cl₂、CO₂、SO₂、HCl；不可干燥 NH₃、HBr、HI、H₂S
- 碱石灰/NaOH固体：可干燥 H₂、O₂、NH₃、N₂；不可干燥 CO₂、SO₂、HCl、Cl₂、H₂S
- 无水CaCl₂：可干燥 H₂、O₂、HCl、Cl₂、CO₂、N₂；不可干燥 NH₃（形成配合物）
- P₂O₅：可干燥 H₂、O₂、CO₂、SO₂、N₂；不可干燥 NH₃、HBr、HI

**尾气处理**
- 有毒气体（Cl₂、SO₂、NO₂、H₂S等）：NaOH溶液吸收
- 可燃气体（CO等）：点燃或气球收集
- NH₃：用H₂SO₄溶液吸收

**二级结论**
- 酸性气体不能用碱性干燥剂，碱性气体不能用酸性干燥剂
- Cl₂不能用碱石灰干燥（会发生反应）
- NO不能用排空气法收集（会被O₂氧化为NO₂）

## 模块三 物质的检验与分离提纯

### 第5讲 常见离子的检验

**阳离子检验**
- H⁺：紫色石蕊试液→变红
- Na⁺：焰色试验→黄色火焰
- K⁺：焰色试验（透过蓝色钴玻璃）→紫色火焰
- NH₄⁺：加NaOH加热，用湿润红色石蕊试纸→试纸变蓝
- Fe³⁺：加KSCN溶液→血红色
- Fe²⁺：加KSCN无现象，再加氯水→变血红色
- Al³⁺：加NaOH至过量→先白色沉淀后溶解
- Cu²⁺：加NaOH溶液→蓝色沉淀
- Ag⁺：加稀HCl→白色沉淀（不溶于稀HNO₃）

**阴离子检验**
- Cl⁻：加AgNO₃和稀HNO₃→白色沉淀
- SO₄²⁻：加BaCl₂和稀HNO₃→白色沉淀
- CO₃²⁻：加稀盐酸→无色无味气体（使澄清石灰水变浑浊）
- SO₃²⁻：加稀盐酸→有刺激性气味气体（使品红褪色）
- OH⁻：酚酞→变红
- Br⁻：加AgNO₃和稀HNO₃→淡黄色沉淀
- I⁻：加AgNO₃和稀HNO₃→黄色沉淀

**二级结论**
- 检验SO₄²⁻时需先加盐酸酸化，排除CO₃²⁻、SO₃²⁻的干扰
- 检验Cl⁻时需先加HNO₃酸化，排除CO₃²⁻的干扰
- 焰色试验中K需透过蓝色钴玻璃观察（排除Na的干扰）

### 第6讲 物质的分离与提纯方法

**物理方法**
- 过滤：分离不溶固体和液体
- 蒸发结晶：从溶液中获取溶质
- 蒸馏：分离沸点不同的液体混合物
- 分液：分离互不相溶的液体
- 萃取：利用溶质在不同溶剂中溶解度不同进行分离
- 升华：分离可升华物质（如碘）

**化学方法**
- 洗气法：除气体杂质（如除CO₂中的HCl用饱和NaHCO₃溶液）
- 沉淀法：加试剂使杂质沉淀后过滤
- 氧化还原法：如除FeCl₃中的FeCl₂加氯水
- 加热法：如除Na₂CO₃中的NaHCO₃加热分解

**二级结论**
- 萃取剂选择条件：与原溶剂互不相溶、溶质在萃取剂中溶解度远大于原溶剂、不与溶质反应
- 萃取后在分液漏斗中分层，上层从上口倒出，下层从下口放出
- 除杂原则：不增（不引入新杂质）、不减（不减少目标物质）、易分离、易复原

## 模块四 实验方案设计与评价

### 第7讲 实验设计原则

**基本原则**
1. 科学性：原理正确
2. 安全性：操作安全，防污染
3. 可行性：条件可实现
4. 简约性：步骤简短，试剂易得

**控制变量法**
- 只改变一个变量，其余变量保持不变
- 常用于探究影响因素的实验

**对照实验**
- 设置实验组和对照组
- 对照组排除无关因素的干扰

### 第8讲 定量实验与误差分析

**常见定量实验**
- 配制一定物质的量浓度的溶液
- 酸碱中和滴定
- 中和热测定
- 硫酸铜晶体中结晶水含量的测定

**误差分析思路**
1. 确定计算公式
2. 分析哪个物理量偏大/偏小
3. 判断对最终结果的影响

**配制溶液的误差分析**
- 称量时左码右物（物码放反）：所称质量偏小→浓度偏小
- 转移时洒出：溶质减少→浓度偏小
- 未冷却到室温就定容：溶液体积偏小→浓度偏大
- 定容时仰视读数：加水偏多→浓度偏小
- 定容时俯视读数：加水偏少→浓度偏大

**二级结论**
- 容量瓶使用前需检查是否漏水
- 洗涤液需转入容量瓶（不能丢掉）
- 定容时先加水到刻度线下1~2 cm，再改用胶头滴管滴加
