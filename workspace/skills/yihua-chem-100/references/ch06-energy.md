# 第6章 化学反应与能量变化

## 模块一 反应热与焓变

### 第1讲 反应热的概念

**核心概念**
- **反应热（ΔH）**：恒温恒压下，化学反应中吸收或放出的热量
- 单位：kJ/mol
- **放热反应**：ΔH < 0（反应体系向外界释放能量）
- **吸热反应**：ΔH > 0（反应体系从外界吸收能量）

**从微观角度理解**
- 断键吸热 + 成键放热
- ΔH = 反应物键能总和 - 生成物键能总和
- 若成键释放能量 > 断键吸收能量 → 放热反应
- 若成键释放能量 < 断键吸收能量 → 吸热反应

**常见放热反应**
- 燃烧反应、中和反应、金属与酸反应、铝热反应
- 大多数化合反应

**常见吸热反应**
- Ba(OH)₂·8H₂O + 2NH₄Cl
- 大多数分解反应
- C + CO₂ → 2CO

**二级结论**
- 需要加热才能发生的反应不一定是吸热反应（如Fe与S反应需加热但为放热反应）
- 不需要加热就能发生的反应不一定是放热反应（如Ba(OH)₂·8H₂O + NH₄Cl）
- 放热反应在常温下不一定能自发进行（需要活化能）

### 第2讲 生成热、燃烧热与中和热

**标准生成热（ΔH_f⁰）**
- 定义：在标准状态下，由最稳定单质生成1 mol纯化合物时的反应热
- 最稳定单质的生成热为零：如O₂(g)、H₂(g)、C(石墨)、Fe(s)
- 注意：C(金刚石)的生成热不为零（石墨是最稳定碳单质）

**燃烧热**
- 定义：在101 kPa下，1 mol纯物质完全燃烧生成指定产物时的反应热
- 关键词：**1 mol 可燃物** + **完全燃烧**
- 指定产物：C → CO₂(g)、H → H₂O(l)、S → SO₂(g)、N → N₂(g)

**中和热**
- 定义：在稀溶液中，强酸与强碱发生中和反应生成1 mol H₂O时的反应热
- 强酸+强碱：ΔH ≈ -57.3 kJ/mol
- 弱酸+强碱：|ΔH| < 57.3 kJ/mol（弱酸电离吸热）
- 强酸+弱碱：|ΔH| < 57.3 kJ/mol（弱碱电离吸热）

**二级结论**
- 燃烧热是以1 mol可燃物为基准的，化学方程式系数可以为分数
- 中和热是以1 mol水为基准的
- 弱酸弱碱参与中和时，电离过程吸热，放出的热量小于57.3 kJ/mol

### 第3讲 键能计算反应热

**公式**
$$\Delta H = \sum E_{\text{反应物键能}} - \sum E_{\text{生成物键能}}$$

**解题步骤**
1. 找出反应物和生成物中所有断裂和形成的化学键
2. 查表或由题目获取各键的键能
3. 代入公式计算

**二级结论**
- 键能越大，化学键越牢固，分子越稳定
- N≡N键能最大 → N₂化学性质非常稳定
- 断开1 mol H-H键、Cl-Cl键、O=O键所需能量分别为436、243、498 kJ

## 模块二 热化学方程式

### 第4讲 热化学方程式书写

**书写要点**
1. 标明各物质的聚集状态：g（气）、l（液）、s（固）、aq（溶液）
2. 标明ΔH及其正负号和单位（kJ/mol）
3. 化学计量数可为分数，与ΔH对应
4. 同一反应，化学计量数不同，ΔH不同（成比例关系）
5. 正反应与逆反应的ΔH数值相等，符号相反

**常见错误**
- 未标明聚集状态
- ΔH的正负号写错（放热为负、吸热为正）
- 化学计量数与ΔH不对应
- 将ΔH的单位写为kJ（应为kJ/mol）

**二级结论**
- 热化学方程式中的化学计量数只表示物质的量，不表示分子个数
- ΔH与化学计量数成正比：若方程式乘以n，则ΔH也乘以n

## 模块三 盖斯定律

### 第5讲 盖斯定律及其应用

**盖斯定律**
- 化学反应的反应热只与反应体系的始态和终态有关，与反应途径无关
- 即：一个化学反应无论一步完成还是分几步完成，其反应热相同

**应用方法——方程式叠加法**
1. 找出目标反应（需要求ΔH的反应）
2. 用已知反应的线性组合表示目标反应
3. 对应的ΔH进行相同的线性组合

**解题步骤**
1. 写出目标方程式
2. 在已知方程式中找到目标方程式中各物质
3. 调整已知方程式方向和系数，使各物质与目标方程式匹配
4. 叠加方程式，消去中间物质
5. 对应ΔH进行相同运算

**二级结论**
- 方程式翻转 → ΔH变号
- 方程式乘以n → ΔH乘以n
- 方程式相加 → ΔH相加
- 盖斯定律的本质是状态函数的性质（焓H是状态函数）

**典型题型**
- 给出几个已知反应的ΔH，求目标反应的ΔH
- 工业生产中多步反应的总反应热计算
