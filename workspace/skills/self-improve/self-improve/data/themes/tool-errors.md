# 主题：工具调用错误

## dir_list 工具问题
- **问题**：node 参数在所有可用选项（auto、host、sandbox、gateway）中都报错
- **影响**：无法使用 dir_list 工具进行目录操作
- **解决方案**：使用 exec 命令替代
- **来源**：会话 891db4b7-1776-4a4c-879c-72a90112e387

## 速率限制处理
- **问题**：429 错误，账户达到速率限制
- **影响**：无法继续调用 API
- **解决方案**：等待并重试
- **来源**：会话 09759429-8194-4c3f-9a1f-dfcacdc77d97

## LLM idle timeout
- **问题**：模型 120 秒无响应
- **影响**：任务中断
- **解决方案**：切换模型
- **来源**：会话 425935e0-b5a9-461f-99cd-428682707ed0
